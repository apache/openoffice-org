Important notes for Apache OpenOffice

Please read the notes before downloading and using any Apache OpenOffice build.


  • Links to other download websites

    Some of the links may refer to the native language download pages. These pages are maintained by the native language community.

  • Installation instructions

    Installation instructions will help you with the first steps. Or see the Installation Guide and the Setup Guides are also available.

  • How to report problems?

    Users of any build are encouraged to join the Apache OpenOffice project and report any errors, improvements or feature requests. Especially the Quality Assurance team) needs help with the the review and investigation of unconfirmed issues.

  • MD5 checksums

    MD5 checksums are used to verify that a locally downloaded file is exactly the same than on the server. It's a hash value that represents the integrity of data. Please read more how to use it.
