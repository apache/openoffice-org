Inter Office Suite Collaboration

This site will contain up to date information on the ongoing quest for standardisation amongst the various open office suites. It is hosted at OpenOffice.org and is meant to be the home for all office developers interested in this effort, particularly GnomeOffice, KOffice and OpenOffice.org. We hope to find many possible areas where we can collaborate, and hopefully share code.

The first issues we want to explore are:

  • Structured file format - a solution to the problem of XML Packaging
  • XML File formats - is it possible to have over-lapping DTDs ?
