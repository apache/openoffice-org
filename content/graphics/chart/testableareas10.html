Testable areas in Milestone 10 ( Integration Candidate 1 Developer Snapshot of the chart reimplementation  )

We need to make sure that we don't introduce severe regression when replacing the old chart. Here you can help a lot with testing and  submitting issues.

In Milestone 10 you now can test all areas besides help content.

If you find a bug that does not occur in the official OpenOffice.org version please use the following query to check whether someone else has already filed an issue:
Known Bugs in the new chart
If the bug is unknown so far please submit an issue starting the summary with 'new chart:'.

Note: When testing save and load you need to activate the chart by double click to see what really was loaded. Otherwise you only will see the replacement image that was created during save.

Thanks a lot for your help!
