Info for the Developer Snapshot of the chart reimplementation from 1st September 2006 (Milestone 8)

The chart reimplementation is still ongoing. In the following you find a list of recently fixed issues and an overview of working and non working areas to give you an idea of what you can expect from this developer snapshot.

Installation Hints:

You will get no system integration with this version at all.

  • Windows: Just exrtact the zip file to YOUDIRECTORY and start YOUDIRECTORY\openoffice_chart2_milestone8\program\soffice.exe

  • Linux: Just exrtact the tar.gz file to YOUDIRECTORY and start YOUDIRECTORY/openoffice_chart2_milestone8/program/soffice

  • Solaris: Just exrtact the tgz file to YOUDIRECTORY and start YOUDIRECTORY/openoffice_chart2_milestone8/opt/openoffice.org2.0/program/soffice

Fixed:

  • Chart Data Table (available with own data)

  • Axis labels and plot area size do respect font size and diagram size now (27354)

  • Autoscaling of Axis (Min, Max, automatic major step count dependent on Fontsize etc.)

  • Automatic Layout of Axis labels (27355)

    • known issue: The automatic 'break' feature is not implemented yet, I would like to know wether someone is missing this or all are more satisfied with the current implementation

  • Font sizes of axis are not loaded correctly (65997)

  • Sorting of x values in xy charts (46521)

  • Statusbar

  • Tooltips ( 27352 )

  • High Contrast mode ( 27356 )

    • known bug: Big charttype icons in wizard (69120)

  • Font and Symbol sizes of data labels do not resize with diagram (note: you need to enable the toolbar button 'Scale Text') ( 29600 )

  • Export Meta Stream in XML for Charts to be able to detect written version

  • Cancel button in chart wizard does not work in writer (Issue 66176)

  • Loaded net charts now display the categories

  • Loading of donut charts differentiates between correct new files and wrong old files now (58457)

  • Excel Import (Stockcharts,Errorbars,Trendlines)
