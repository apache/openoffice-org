Thank you for using OpenOffice.org - now Apache OpenOffice

Get the latest version

In June 2011 OpenOffice.org has been renamed to Apache OpenOffice and is now developed as a top-level project within the Apache Software Foundation (ASF).

If the latest version of Apache OpenOffice is not yet available in your language, this means that we need volunteers to help us with the translation. If you are interested, we have a page how you can help to translate Apache OpenOffice. Your volunteer contributions will help us bring free software to the entire world.
