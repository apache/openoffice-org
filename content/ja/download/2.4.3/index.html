ダウンロード

ダウンロードを開始するには、ご利用のOSの名前をクリックしてください。
Windows (115MB)
JREなし
Windows 98, 2000, XP, Vista, 2003
Linux (150MB)
RPM, JREなし 		Linux (148MB)
DEB, JREなし

P2P(BitTorrent)でダウンロードする場合はOpenOffice.org P2P Downloads(英語)を参照してください。

日本語ランゲージパックのダウンロード

上記日本語版をインストールした場合は必要ありません。英語版をインストールしたうえに、日本語ランゲージパックをインストールすれば、メニュー、ヘルプなどが日本語になります。
  • Windows用日本語ランゲージパック (19M)
  • Linux用(RPM)日本語ランゲージパック (16M)
  • Linux用(DEB)日本語ランゲージパック (16M)

ダウンロードが成功したか確認したい

ダウンロードが成功したか確認したいを参考にして、md5sumの確認をしてください。2.4.3のmd5sumはこちらです。
