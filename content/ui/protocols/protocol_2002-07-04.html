2 Round table

2.1 Selection Handling in Dialog Edit Fields

GT: Most edit fields in dialogs do not preselect their contents on opening. The selection behaviour should be changed to:

- Select all on opening the dialog and on getting the focus from another control in the same dialog (TAB key, mouse click).

- Keep selection/cursor position on reactivating the dialog.

2.2 Border Format Control in High Contrast

CJ: The border format control shows nothing in HC Black. Reason: Border color and control background are black and borders do not support auto color. Possible solution is to replace the black color with the current text color (only in HC Black mode).

2.3 New Picture Format for Toolbox Icons

MMP: Sometime (not so near future) we should use PNG instead of BMP to support transparency for toolbox items. Stella Schulze will contribute some PNG images for testing. CJ and MMP will talk with Stephan Schaefer, Kai Ahrens, Malte Timmermann and Phillip Lohmann about the general technical possibility.

2.4 Function Bar & PDF Export

MMP: For the PDF export a new button will be inserted into the function bar. Because it is already overloaded, we will initially hide (not remove) the Stop button used to cancel background actions (i.e. loading an HTML document).
