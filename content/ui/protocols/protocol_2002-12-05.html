2 Round Table

Treelistbox keyboard accessibility

CJ raised the issue that our treelistbox is only keyboard accessible with the cursor keys. This is not user friendly for treelistboxes with larger contents. Using some kind of autocompletion would help. We discussed this a bit and decided that the user expirience team should make a specification but its not important enough for OOo1.1 Beta
