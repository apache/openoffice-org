2 Comments on Action Items

2.1 Toggle Buttons in Toolbars

FS: It is easily possible to toggle images in toolbars and menues. The problem is the replacement of text. FS will discuss this with Mathias Bauer (MBA).

2.2 Database Beamer Problems

CJ: Because of different problems on different systems, the beamer will not support undocked mode anymore. So it is only possible to use it docked to a specific document.

2.3 Important: User Interface Specifications

CJ: Everyone who writes a specification should not forget to include names for new slots and mnemonic characters for all new menu/dialog items.

2.4 Submenu Selection

CJ: While the mouse is over a submenu, its parent menu entry is not selected. This should be changed to increase consistency to other applications. CJ will contact SSA.

2.5 Bold Fonts in High Contrast White

2) In HC White mode the UI should use the bold version of the current font. CJ will contact SSA.

3 Round Table

3.1 Transparency of Standard Boolbox Images

CJ: The conversion of the non-HC images to the new transparency color Magenta is still in progress. The new transparency color is needed to be able to differ between gray and transparent.

Cropping Images

CJ: The crop tabpage in the image format dialog is not very user-friendly. Scaling settings are repeated from the first tabpage of the dialog. Also per default the dialog would scale the image while cropping. A solution is to offer only a checkbox (resize while cropping, default: no) and to show the scaling information read-only.
