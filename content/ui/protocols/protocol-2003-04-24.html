2 Roundtable

2.1 Possible changes of docked windows having a 'pin' symbol

MMP brought up the issue of docked windows (like the database browser) that have a pin that determines whether the window 'flies' over the document window or moves it.

It has been aggreed that this feature doesn't make much sense anymore. It was useful as long as an auto hide function was available (StarOffice 5.2).
