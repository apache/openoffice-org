2 Removing unused Bitmaps

We discussed shortly that we need to remove unused Bitmaps sometime.
CJ said that the old black&white toolbox images can be removed.
FS suggested that we automate this process by modifying the bitmap tool and the resource compiler to generate a list of used bitmaps.
PB said we have to edit the global office imagelist by hand to get rid of old unused explorer images.
According to CJ, this should happen as soon as we have some time for it.
