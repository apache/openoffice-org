2 Round table

It has been decided to assign the implementation of the accessibility support for the BrowseBox to DR.

FS brought up the topic of the right mouse button handling. Currently the context menu applies to the selection available prior to the right mouse button click. At least one exception exists in the file dialog where the right click also changes the selection. CJ will check the handling of context menus in different window managers (after his vacation until March, 15th).

CJ pointed to the Renewed document preview proposal (can be found at http://ui.openoffice.org/servlets/ProjectDocumentList). It is about keyboard handling, use of icons, and the mark of the selection. FS brought up the question of zooming into the preview in writer which is currently unable to show parts of a page zoomed in.

CJ also pointed to the Custom colors proposal (can be found at http://ui.openoffice.org/servlets/ProjectDocumentList). OS stated that it is not yet decided who will implement the dialog and the configuration entries for the relating task.
