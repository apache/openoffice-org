1.1 Save Cursor Position in Writer Document

CJ has written bugs for all unintentional cursor movements in Writer documents (i.e. after saving).

1.2 Reorganize document section in ui.openoffice.org

OS has moved all proposals/specifications from Documentation section to Source Code section, which is reachable via CVS (http://ui.openoffice.org/source/browse/ui/www/proposals).

1.3 Submit task regarding the problem in seleting text in Edits by mouse

This bug was already fixed without any action by MMP.

1.4 TabBar issues

CJ has stated that we need icons in TabBars. TabBars are used i.e. in Calc to show the sheet names or in Draw/Impress to show the page names. The icons will visualize special states of the according page or sheet. Currently this is realized with a special text color, but this is unusable in High Contrast modes.

DR will collect all occurences of colored TabBars in the Spreadsheet application.

1.5 Menu issues

CJ will write bugs for wrong painting of menues. Sometimes menu entries with icons are higher than entries without icons. Additionally the window title color is used to draw selected menu entries, and not the selection color.

MMP will talk with Hennes Rohling if it is possible to receive a menu background color from system settings.
