2 Roundtable

2.1 Selection inside of the Dialog Editor (Basic IDE)

CJ brought up a problem of the dialog editor used in the Basic IDE. When selecting all elements of the dialog the selection is only highlighting on the outside. There's no visual feedback for the also selected objects inside of the dialog.

The same applies to selection of multiple drawing objects in all office applications. Additionally it's only possible to move the selected objects by grabbing the selection frame with the mouse.

Further investigation of this issue is needed.

2.2 Changes in the option dialog in OOo 2.0

It is planned to improve the separation of the modules. One of the actions to achieve this goal is to change the elements of the displayed in the options dialog.

The dialog should only contain general dialog pages and those of the current module. E.g. when selecting tools/options with a text document open only the pages of the text document settings and those of general settings are shown. Those of Calc, Draw, Impress ... are not shown.

Additionally it is planned to remove the settings that only belong to the current document from the options dialog and create a new dialog that changes the document settings but not those that affect the next created documents.
