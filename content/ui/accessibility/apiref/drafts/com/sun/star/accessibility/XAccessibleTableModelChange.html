interface XAccessibleTableModelChange in module com::sun::sun::star::accessibility::

(Global Index)

Syntax

interface XAccessibleTableModelChange : com::sun::star::uno::XInterface ;

Description

Implement this interface to let an event give access to a change of a table model.

The methods of the XAccessibleTableModelChange interface give access to the type and cell range of the change. See AccessibleTableModelChangeType for details of the change type. The range of the affected rows, columns, and/or cells can be obtained by calling the other four methods.

Method Summary

getType

Syntax

short getType ();

Description

Returns the type of event.

The model change either inserted or deleted one or more rows and/or columns or modified the content of a number of cells. See AccessibleTableModelChangeType for details of the type of the model change.

The type of the event as defined in AccessibleTableModelChangeType .
