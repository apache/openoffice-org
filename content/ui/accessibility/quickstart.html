OpenOffice Accessibiliy

OpenOffice has support for various Accessibility aspects.

Features

Keyboard navigation, scheming, AT support and much more. Details can be found here: https://ui.openoffice.org/accessibility/whitepaper.html.

AT support

If you want to find out if your AT works with OpenOffice, or how to configure OpenOffice to be used with an AT tool, you can have a look into https://wiki.openoffice.org/wiki/Accessibility.

Mailing list

There is a mailing list which can be used for user questions and comments, as well as for developer related things. Please subscribe to
dev@openoffice.apache.org.

The OpenOffice Accessibility Project

More details about all Accessibility related things around OpenOffice can be found in the OpenOffice Accessibility Project and in the OpenOffice Wiki:

https://ui.openoffice.org/accessibility/
