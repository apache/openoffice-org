Links fra OpenOffice.org Jam Session på Linuxforum 2006

På denne side finder du links til de sider, som Louis Saurez-Potts nævnte under OpenOffice.org Jam Session på Linuxforum 2006.

  • OpenOffice.org wiki - mange ressourcer til udviklere.
  • OOoMacros.org - dokumentation og eksempler på makroer i OpenOffice.org.
  • Documentation Project - officiel side om OpenOffice.org-dokumentation
  • Programming - officiel side om OpenOffice.org-programmering
  • Contributing to OpenOffice.org - officiel side om, hvordan du kan bidrage til OpenOffice.org
  • Active Projects and Leads - liste over officielle OpenOffice.org-projekter. "Accepted projects" er det samme som de tidligere "Whitelist projects".
  • Quality Assurance - officiel side om kvalitetssikring i OpenOffice.org-projektet herunder fejlmelding og -rettelser.
  • OpenOffice.org Specification Project - detaljerne om specifikationerne findes dog på wikisiden.
  • XML Project - officiel side om dokumentformatet i OpenOffice.org, som er XML-baseret.
  • Extensions : From production to end-users - officiel side om udvidelser til OpenOffice.org.
  • OpenOffice.org codelines - udviklingsplan for OpenOffice.org (Roadmap). Selve Roadmappen findes som PDF på adressen http://development.openoffice.org/releases/ooo_roadmap.pdf og er lige blevet opdateret (marts 2006).
  • Issue Tracker - officiel sagshåndteringssystem til OpenOffice.org, hvor alle forslag om forbedringer og fejlrettelse samt fejlmeddelelser bliver registeret.
  • ongoing - Tim Brays blog.
  • Velkommen til DA.OpenOffice.org - den danske OpenOffice.org-projektgruppe.

Opdateret 13. marts 2006
