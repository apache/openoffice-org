Key to File Formats

  • HTML files are viewable in your web browser.
  • SXW files can be opened and edited in OpenOffice.org Writer.
  • ODT files can be opened and edited in OpenOffice.org2.0 Writer.
  • PDF files are portable read-only documents.
  • ZIP files are compressed archives that must be extracted after download.
