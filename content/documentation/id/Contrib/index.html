Bagaimana Melakukan Kontribusi?
  1. Daftarkan diri Anda sebagai anggota komunitas OpenOffice.org
  2. Buat satu issue baru yang dialamatkan pada proyek Documentation (pada pilihan Component) dan tuliskan nama willysr pada bagian Assigned to
  3. Upload file Anda dan disertai dengan keterangan yang cukup jelas
  4. Setelah di-review, maka dokumen Anda akan diupload pada situs OOo dan akan ditampilkan pada halaman ini
