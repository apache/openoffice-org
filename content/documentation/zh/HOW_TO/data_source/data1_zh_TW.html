從零開始建立資料源 (資料來源, Data Source)

建立資料源

這些步驟會帶你在 OpenOffice.org 系統中設定一個資料源.  我們將建立一個 dBase 格式的資料庫以符合跨平台的需求.

  1. 在功能表選擇 「工具」(Tools) - 「資料源」(Data Source).
  2. 按一下 「新資料源」(New Data Source).
  3. 輸入資料源的名字.  注意:  我將用 TEST 作為資料源的名稱.
  4. 按一下 Tab 鍵.
  5. 在「資料庫類型」(Databese Type)中選擇 dBase.  注意:  我們之所以選用 dBase 是因為它允許對本地端資料庫的完全控制.
  6. 按一下 Tab 鍵.
  7. 輸入資料庫儲存的路徑.  請確定你輸入的路徑目錄 *並不* 存在.  程式會自己建立.
  8. 按一下 「套用」 (Apply).

在資料源中新增表格

在這個部分, 我們將要告訴你如何在資料源中新增一個表格.  而我們假設你的資料源擁有對資料庫的完整控制權限.  目前只有 dBase 或 工作表文件(Spreadsheet) 是可以完全存取你電腦上檔案的類型.   如果你已經有像是MySQL, SQL Server, 或 ORACLE 的資料庫的話, 您可以選用 JDBC 或 ODBC.

  1. 按一下 「表格」(Tables) 分頁.
  2. 按一下「新建表格設計」(New Table)圖示.
  3. 然後會顯示「表格設計」(Table Design)視窗
  4. 輸入 Name.
  5. 按三次 Tab 鍵.
  6. 輸入 Telephone.
  7. 按三次 Tab 鍵.
  8. 輸入 Address.
  9. 按三次 Tab 鍵.
  10. 輸入 City.
  11. 按三次 Tab 鍵.
  12. 輸入 State.
  13. 按三次 Tab 鍵.
  14. 輸入 Zip.
  15. 在選單中選擇「檔案」(File).
  16. 選擇「儲存檔案」(Save).
  17. 輸入您想要的表格名稱.  注意:  我們在此教學中使用 TestTable 這個名字.
  18. 關閉「表格設計」(Table Design)視窗.
  19. 關閉「管理資料源」(Data Source)視窗.

在資料源中編輯表格

這個部分展現 OpenOffice.org 可以用來輸入, 編輯與刪除資料源中的資料.

    新增資料
  1. 按下 F4 鍵.
  2. 「資料源」(Data Explorer)視窗此時出現.
  3. 選擇我們剛剛建立的 Test 資料源. (譯註: 這邊用的「選擇」代表按兩下, 或者是按下旁邊的展開加號鈕)
  4. 選擇「表格」(Tables).
  5. 選擇 TestTable
  6. 如果表格中尚未存在任何資料, 你輸入任何東西後將會自動新增該筆資料. 否則請按下「新增資料」(New Record)按鈕.
  7. 輸入以下的資訊, 每個欄位間按一次 Tab 鍵:
    John Doe 999-999-9999 1234 Anywhere This Town This State 12345
    Jane Doe 122-122-1222 5431 Here Street This Town This State 12345
    編輯資料
  1. 按一下 John Doe 的 Telephone 欄位.
  2. 輸入 888-888-8888
  3. 按下鍵盤上的「下」鍵.  注意:  移動到新的資料會儲存您的變更 .
    刪除資料
  1. 按一下 Jane Doe 前的「選擇行」(Row Selector)鈕.  無論是 或是 都可以將整筆資料選取.
  2. 按滑鼠右鍵.
  3. 按一下 「刪除行」(Delete Rows).

然後?

你已經看到 OpenOffice.org 可以跟資料庫整合的某些部分了, 然後我會建議你:

  1. 試試看在新增表格時使用不同的欄位類型 (欄位型態, column type).
  2. 在「管理資料源」(Data Source)視窗中多看看關於建立 SQL 語句的部分.

注意:  使用本教學中建立的資料庫來做測試便不會傷害你任何資料.
