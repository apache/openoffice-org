Documentation - Samples and Templates

Last modified: $Date: 2009/01/26 13:44:27 $

Mission Statement

To demonstrate the capabilities of the programs in the OpenOffice.org suite, and provide usable templates for business and personal use.

Summary

The Samples and Templates area is broken up into two locations.  These areas represent the directory stucture where the files should be placed.

Note:  %OpenOffice.org% should be replaced by the location you originally installed the OpenOffice.org suite.

User Directories
Location Directory Description
Template %OpenOffice.org%/user/template/ Directory used to populate the Template sections of the various OpenOffice.org programs.
2.x Templates %OpenOffice.org%/share/template/en-US Directory used to populate the Template sections of the various OpenOffice.org programs. Files are in the 2.x format.
Gallery and Clipart %OpenOffice.org%/user/gallery/ Directory with graphics that can be used to populate the Gallery in OpenOffice.org programs or used as clipart.
OpenClipart %OpenOffice.org%/user/gallery/ External site that has Graphics/Clipart for use in OpenOffice.org programs.
Samples and Extras %OpenOffice.org%/user/samples/ Directory containing examples and extras for OpenOffice.org programs.
