If I run more than one operating system, do I need a key set for each system?

No. You can use the same public key file set on any machine you use to connect. The file set consists of two files: id_dsa and id_dsa.pub. Move these two files to a new machine, point to it during the terminal login, and you are ready to establish an SSH tunnel.
