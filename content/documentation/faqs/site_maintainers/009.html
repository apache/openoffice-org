How do I send my public key to OpenOffice.org?

Open an issue:
  • Log in to the OpenOffice.org web site. You must have your cookies enabled.
  • Under My Tools in the left menu, click on the File Issue link.
  • Click on proceed.
  • Click the www link near the bottom of the list of components.
  • Enter a Summary.
  • Type ssh2key@openoffice.org into the box labeled Assigned to.
  • In the description section, explain what modules (projects or areas of the web site) you wish to access.
  • Click Submit issue.
Attach your public key file:
  • Click the link to attach a file to this issue, or revisit the issue and click the Create an attachment link.
  • In the attachment form, browse to your id_dsa.pub file.
  • Add one line description.
  • Select file type: binary (application/octet-stream).
  • Submit.

For more information:

  • Submitting Your Certificate (Public Key).
