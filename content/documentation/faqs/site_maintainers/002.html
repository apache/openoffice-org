What is CVS and when will it be useful to me?

CVS, or a Concurrent Versions System, is a file system that allows multiple persons to use files at the same time. It records and time stamps all changes by all authors. This creates an excellent history log of all files in the system, and it flags editors if it finds differences, so they can't overwrite another person's work in the same file without review.
