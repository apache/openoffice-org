Is there a log of CVS changes?

Yes. You can find a log of changes in two places.

  1. Access the mail list that collects all CVS changes for each project. Every change in the file system generates a message for that project's cvs mail list. You can subscribe to these lists or browse the archives by clicking on the Mailing Lists link at the project's home page. You must be logged in to subscribe. You can access the individual project pages through the Projects tab at the top of most pages at the OpenOffice.org web site.
  2. Browse the source files for each project, where a history of changes for each file shows.
