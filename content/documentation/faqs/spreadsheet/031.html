If a cell is centre-justified, and the text is wider than the cell, it automatically left-justifies. Can I fix this?

This is a known bug in Calc that is shown in Issue #1171. This problem is fixed in the OOo1.9m26 developer snapshot version of OpenOffice.org and will be available in the stable version of OpenOffice.org 2.0. To obtain the latest versions of OOo, visit the Download Central page.

Note there is a work-around:

  1. Merge the cell with adjacent cells: Select the cells you wish to merge, then select Format > Merge Cells... > Define from the drop-down menus.
  2. Center the data in the merge cell: Click the Align Center Horizontally button on the tool bar, or use the drop-down menus to select Format > Cell, click the Alignment tab, then choose Center from the Horizontal drop-down menu in the Text alignment section.
