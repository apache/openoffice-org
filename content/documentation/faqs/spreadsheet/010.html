How do I create a chart in a spreadsheet that is automatically updated when the data is changed?

Follow these steps:

  1. From the pull down menus, select: Tools > Options.
  2. In the left column of the dialog that appears, select: Spreadsheet > General.
  3. On the page that appears, in the Updating section under Update links when opening, click the checkbox next to Always.

Note: This is a global setting and will affect all spreadsheet charts and linked information from spreadsheets. If this is not desirable, click the checkbox next to On request. With this selected, a dialog box will pop up when each file with links to a spreadsheet is opened, where you can choose whether or not to update the item with newer information from the linked spreadsheet.
