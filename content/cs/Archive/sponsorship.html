Sponzoring

I přesto, že našemu projektu nemůžete pomoci jako vývojář nebo uživatel, můžete nám pomoci jako sponzor. Máte-li zájem o sponzoring našeho projektu jinou formou, kontaktujte vedoucího projektu Pavla Janíka pro více informací.

Server pro build OpenOffice.org pro operační systém GNU/Linux 64 bit

Společnost Novell-Praha, s. r. o., zakoupila v září 2004 pro náš projekt zvláštní server pro build na platformě AMD64. OpenOffice.org není v 64bitové verzi zatím k dispozici, probíhá portace a tento server ji výrazně urychlí. Verze 2.0 již bude k dispozici i pro AMD64, jako první kancelářský balík na světě. Děkujeme za podporu!

Server je navržen tak, aby poskytl maximální možný výkon za přijatelnou sumu peněz. Jeho konfigurace je následující:

  • základní deska
    • MSI K8T Master
  • procesor
    • Dvakrát AMD Opteron 242 (samozřejmě SMP verze)
  • paměť
    • DDR3200 Kingston 4x512MB, Reg. ECC, CL3, celkem 2GB RAM
  • disk
    • Disk SATA HDD 74 GB WD740GD Raptor 10k RPM
  • grafika
    • ASUS A9200SE, ATI Radeon 9200SE
  • zdroj
    • Fortron 550W, Intel SSI, EPS
  • case
    • Cetus YY-5601, byla vybrána protože umožňuje prohánět vzduch pomocí dvou 120mm ventilátorů.

Na serveru je instalován SUSE Linux verze 9.1 v 64bitové verzi.

Server pro build OpenOffice.org pro operační systém GNU/Linux

V současné době používáme pro vývoj a sestavování binárních instalačních sad pro operační systém GNU/Linux počítač, který jsme zakoupili díky našim sponzorům. Parametry tohoto počítače jsou uvedeny níže:

  • základní deska
    • Intel Entry Server Board S875WP1-E (věnovala společnost Sun Microsystems Czech, s.r.o.)
  • procesor
    • Pentium 4, HT, 2,8 GHz, 800FSB boxovaný (odměna za spolupráci na časopise o OpenOffice.org od společnosti SoftwareMedia)
  • paměť
    • DDR3200 Kingston 2x512MB (věnovali Mgr. Daniel Hrbac a Viktor Matys)
  • disk
    • Disk SATA HDD 36.7GB WD360GD 10k rpm, 5RZ (zakoupeno z odměny za Projekt roku 2002 společností Nethost s.r.o.)
  • case
    • CASE, zdroj 350W, EZU (věnovala společnost CASUA)
  • UPS
    • UPS TRUST Energy Protector 650 (zakoupeno z odměny za Projekt roku 2002)

Server pro build OpenOffice.org na platformě Microsoft Windows

Vytváření instalačních sad pro operační systémy Microsoft Windows probíhá na Masarykově univerzitě v Brně. Počítač byl k tomuto účelu věnován společností ASW Systems, s.r.o., software byl věnován Ústavem výpočetní techniky MU. Parametry používaného počítače jsou uvedeny níže:

  • základní deska
    • Intel Entry Server Board S875WP1-E
  • procesor
    • Pentium 4, HT, 2,8 GHz, 800FSB boxovaný
  • paměť
    • 1GB
  • disk
    • Disk SATA HDD 36.7GB WD360GD

Nainstalovaný software: Microsoft Windows Server 2003 (Standard Edition), Microsoft Visual Studio .NET 2002 (Professional Edition).

Server pro build OpenOffice.org na platformě AMD64 (GNU/Linux)

Vývoj a sestavování OpenOffice.org pro platformu AMD64 umožnil www.linuxsoft.cz na svém serveru s těmito parametry (viz též zde):

  • základní deska
    • Tyan Thunder K8S S2880 UGNR
  • procesory
    • 2x AMD Opteron 240
  • paměť
    • 2x 512 MB Apacer DDR2700 ECC
  • disky
    • 2x U320 SCSI Maxtor ATLAS10K4_36WLS, 36.7 GB, 10000 rpm
    • 1x IDE Seagate ST3120026A, 120 GB, 7200 rpm
  • case
    • 2U rack Emko EM-2830L, zdroj Fortron 460W EPS12V
