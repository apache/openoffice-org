Expected Results

2

Select the cell range that you want to calculate subtotals for. (The contents in the first row will be regarded as column labels)

3

Click 'Data-) Subtotals'.

The 'Subtotal' dialog appears. The current tab page is '1st Group'. The combo box 'Group by' use the first column label as its default value.

4

In the 'Group by' combo box select label of a column which contains numbers or characters to group the data.

5

In the 'Calculate subtotals for' box, select a check box of the column labels that you want to subtotal.

6

In the 'Use function' box, select the function 'Product'.

7

Click 'OK' button.

The subtotal result is displayed in hierarchy according to the 1st group option specified.

8

Click the cell that contains the result of subtotal.

There is a formula =SUBTOTAL(6,...) shown in formula bar.

The function Product can multiplies the numbers. Ignore the characters and blanks.

For example, A1=3, A2=4, A3=5 then Product(A1:A3)=60

The result is correct.
