Test Case AOOTest-368: Data-)Subtotals_Options_Include format. [Version : 2]
Author:undefined - 18/03/2014 04:35:23
Last edit by:undefined - 03/04/2014 20:52:37
Summary:

Using the reference file "Subtotals_1_AOOTest-368" as a reference, this is what is generally expected from this testcase:

1. Left picure: Cell C2, C4's format is currency; C3, C5's format is number; C6's format is percent

2. Picture in middle: Uncheck "Include formats" - Item No.3 is moved from row 4 to row 3, by taking C3's format number while discarding C4's format currency, aka change from "?2.00" to "2". The change of other cells uses the same logic.

3. Right picture: Check "Include formats" - Item No.3 is moved from row 4 to row 3 with C4's format currency. The change of other cells uses the same logic.

In summary: "Include formats" means cell's format will be along with cell's data when sorting; Not "Include formats" means cell's format will inherit new position's cell's format when sorting.

Preconditions:

1)  Install Apache OpenOffice.

2) Attached files

   - Subtotals_1_AOOTest-368 (use as a reference)

   - AOO_Calc_Subtotal_Formatted_YuXehn.ods (sample spreadsheet)

#:Step actions:Expected Results:
1

Open sample spreadsheet AOO_Calc_Subtotal_Formatted_YuZhen.ods

Column C should now be highlighted

The 'Subtotals' dialog appears. The current tab page is '1st Group'.

 

 

4

"Data" should now be highlighted.

 

5

9

Worksheet is returned to it's original state.

 

 

10

Repeat step 2-6.
