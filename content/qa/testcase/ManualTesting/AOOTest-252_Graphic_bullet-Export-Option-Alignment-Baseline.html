Test Case AOOTest-252: Graphic bullet-Export-Option-Alignment-Baseline [Version : 2]
Author:undefined - 11/03/2014 07:19:28
Summary:

the purpose is AOO text document containing graphic bullets can be exported correctly into MS Word 2003 format
 

#:Step actions:Expected Results:
1

Create a new writer document, and create several paragraphs, apply bullets from toolbar "Bullets On/Off" for these paragraphs. Then use promote from toolbar to set these paragraphs at different level(at least have 4 levels)

The document ,paragraphs, bullets can be successfully created

2

Select paragraghs at different level, change their default bullets to different graphic bullets from right-click->Numbering/Bullets... and switch to graphic tab.

Different graphic bullets are correctly applied to paragraphs at different level

3

Go to one paragraph at level 2, insert a new paragraph between two paragraphs which have level 2 graphic bullets.

bullet will be added automatically beyond this text

4

Select level 3 graphic bullet and modify it to other format graphic bullet

can modify it successfully

5

select level 4 graphic bullet and delete it

can delete successfully

6

Promote 2 times on level 2 graphic bullet from toolbar 'Promote'
Demote 2 times on level 4 graphic bullet from toolbar 'Demote'

level 2 graphic bullets will be same with level 4 graphic bullets.

level 4 graphic bullets will be same with level 2 graphic bullets.

7

undo all previous steps.then redo all previous steps

undo\redo ok

8

Save the sample file to *.odt format then reload then save to *.doc format, then reload again

Both the two formats can be opened and all graphic bullet display well.

9

close the sample file and reopen it with MS word 2003

can reopen it and all graphic bullet display well.
