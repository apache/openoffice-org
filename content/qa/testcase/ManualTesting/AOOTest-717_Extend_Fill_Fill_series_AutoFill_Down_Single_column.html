Test Case AOOTest-717: (Extend) Fill_Fill series_AutoFill_Down_Single column [Version : 1]
Author:undefined - 19/11/2012 10:29:18
Summary:

To verify that automatically fill works properly - fill series - auto fill - down. Automatically generate series with the options in this dialog. Determine direction, increment, time unit and series type.. Before filling a series, first select a cell range. Left and Right are enabled only when selected range covers more than one columns. Up and Down are enabled only when selected range covers more than one rows. End value and Increment are disabled if series type is AutoFill. Time unit section is enabled only when series type is Date. Start/End value supports numbers, dates, and times.
