Author:undefined - 12/03/2014 07:52:07
Summary:

the list style with added paragraph style can be inported and exported in AOO

#:Step actions:Expected Results:
1

launch Apache OpenOffice, and new a writer document

new successfully

2

input some text,and enable "Styles and Formatting" dialog by "F11", select the "List style" from the toolbar at the top of the dialog

many styles will be shown in the style list box

3

select 'List 4' style

4

click "modify..." from context menu, and the "Numbering style" dialog pop up,

switch to "options" tab, and click "level 7"

5

Selelct a different numbering style from default one from 'Numbering' list box, and click "OK" button to close the dialog

7

Select some paragraphs and from 'Styles and Formatting' dialog, double click style 'List 4' to apply the style

The selected paragraphs are set default numbering

8

Promote paragraphs selected in step 6 to level 7 by clicking promote button from tool bar 6 times

The modified numbering in step 5 is applied to the text successfully,

9

Save to odt file and reopen it with AOO, and check the numbering

the text in the document shown same effect with previous step
