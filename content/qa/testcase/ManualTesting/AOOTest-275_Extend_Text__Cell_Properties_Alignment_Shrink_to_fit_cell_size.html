Test Case AOOTest-275: (Extend) Text * Cell Properties_Alignment_Shrink to fit cell size [Version : 1]
Author:undefined - 24/10/2012 14:21:16
Summary:

To verify that text * cell properties can be applied/modified properly - alignment - shrink to fit cell size. Possible states of Shrink to fit cell size check box. State Mockup Comment Unchecked Shrink to fit cell size is not applied to any cell in current selection. Checked Shrink to fit cell size is applied to all cells in current selection. Don't care Shrink to fit cell size is applied to some of cells in current selection.

.

3. State of Shrink to fit cell size is 'Unchecked' if none cell in selected range is wrapped.
