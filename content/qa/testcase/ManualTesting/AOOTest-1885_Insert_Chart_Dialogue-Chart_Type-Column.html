Test Case AOOTest-1885: Insert Chart Dialogue-Chart Type-Column [Version : 1]
Author:undefined - 28/11/2013 09:14:38
Summary:

To verify the Object information and display feature accessibility of "Chart Wizard" dialog in SC

Accessibility Checklist:

1 Visual focus indicator

2 Information about user interface objects

3 Labels

4 Text

Screen Reader Tools: Jaws or NVDA(open source) to

AT Support: Reference :https://wiki.openoffice.org/wiki/Accessibility

#:Step actions:Expected Results:
1

Launch Screen Reader Tools

Screen Reader Tools can be launched successfully.

2

Launch AOO and New a Calc

A new SC file will be created.

3

Select a range of data

4

Select Insert -> Chart on main menu by keyboard.

 It will be highlighted

5

Select Chart Type-Column in list at the left side, Check this dialog

Dialog has visual focus

6

Press Tab/Shift+Tab or Arrow key to navigate to each control in current dialog in order.

1. The controls can be navigated in order. (Top -> Bottom, Left -> Right)
2. There is a visible focus indicator at all times.
     The control with focus can be visually identified, such as, highlighting, focus rectangles and a visible insertion caret.
 

7

Listen with Screen Reader Tools

Screen Reader Tools announces the appropriate information.
1. The information of the control can be announced correctly, when get focus. (Name, Label, Type, Content, State, and how to interact with it...)
2. Follow the keyboard focus as move between the controls.
