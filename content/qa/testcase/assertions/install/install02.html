Test Case: install02

Solaris

Linux

Windows

Test Aim
Verify OpenOffice.org installs successfully and can be launched in network mode.

Test Area: Solaris

Description: Network Install

  • Log into  the test machine as root.
  • Obtain the openoffice.org tar-ball to be tested from:-
    • http://www.openoffice.org/dev_docs/source/download.html
  • Extract the tar-ball and install by executing  ./setup /net  from the  install directory.
  • Using the install wizard follow prompts.
    • Accept all license agreements.
    • When prompted Select Installation Type, "Standard Installation"
    • When prompted Select Installation Directory, <network-installation-set> directory.
    • When prompted for Java Setup select a valid Java release. Refer to system requirements at  http://www.openoffice.org/dev_docs/source/download.html
  • After the install has completed, logoff as root.
  • Login in as a new user on the target machine.
  • cd to the <network-installation-set> directory.
  • Install a workstation installation  by typing ./setup from the <network-installation-set> directory.
  • Using the install wizard follow prompts.
    • Accept all license agreements.
    • When prompted Select Installation Type, "WorkStation Installation"
    • When prompted Select Installation Directory, accept default directory.
  • After installation is complete launch openoffice.org by execting ./soffice from the installed directory.
  • Ensure openoffice.org launches successfully.

Test Area: Windows

Description: Network Install
  • Log into the test machine as Administrator.
  • Obtain the openoffice.org zip file to be tested from:-
    • http://www.openoffice.org/dev_docs/source/download.html
  • Extract the zip file and install by executing  ./setup /net  from the  install directory.
  • Using the install wizard follow prompts.
    • Accept all license agreements.
    • When prompted Select Installation Type, "Standard Installation"
    • When prompted Select Installation Directory, <network-installation-set> directory.
    • When prompted for Java Setup select a valid Java release. Refer to system requirements at  http://www.openoffice.org/dev_docs/source/download.html
  • After the install has completed, logoff as Administrator.
  • Login in as a new user on the target machine.
  • cd to the <network-installation-set> directory.
  • Install a workstation installation  by typing setup.exe from the <network-installation-set> directory.
  • Using the install wizard follow prompts.
    • Accept all license agreements.
    • When prompted Select Installation Type, "WorkStation Installation"
    • When prompted Select Installation Directory, accept default directory.
  • After installation is complete launch openoffice.org by execting soffice.exe from the installed directory, or from the windows Start menu..
  • Ensure openoffice.org launches successfully.



 
 
 
 

Expected Result:

1] Openoffice.org installs successfully, reporting no error messages during installation.

2] The Java install on the target machine is recognised during the installation.

3] Openoffice.org launches successfully after the installation.
