Test Case: aut_pre02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create an Autopilot presentation
Description:
    1. From the menu bar,  select File - Autopilot - Presentation.
    2. The Autopilot Presentation dialog should be launched.
    3. Ensure Empty presentation is selected on the Autopilot Presentation dialog.
    4. Select Create on the Autopilot Presentation dialog.
    5. Ensure OpenOffice.org creates an Autopilot Presentation page as well as launching a Modify slide dialog..
    6. The document should be empty.
    7. Close the Modify Slide Dialog by selecting cancel.
    8. Close the Presentation Document.

     
Expected Result:
All operations should complete successfully.

 

End of test case
