Test Case: aut_let02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create a letter from Autopilot
Description:
    1. From the menu bar,  select File - Autopilot - Letter.
    2. The Autopilot Letter dialog should be launched.
    3. Ensure Business letter is selected on the Autopilot Letter dialog.
    4. Select Create on the Autopilot Letter dialog.
    5. Ensure OpenOffice.org creates an Autopilot Letter.
    6. The document should show
        today's date,
        The start of the letter should begin with  "Dear Sir or Madam".
    7. Close the Letter Document.

     
     
Expected Result:
All operations should complete successfully.

 

End of test case
