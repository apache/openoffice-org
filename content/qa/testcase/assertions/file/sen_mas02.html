Test Case: sen_mas02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can send to a Master document >
Description:
    1. From the menu bar,  select File - New - Text Document
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message"
    4. From the menu bar,  select File - Send - Create Master Document.
    5. A Name and Path of Master Document dialog should be launched.
    6. Enter OOmaster in the File name box in the  Name and Path of Master Document dialog.
    7. Select Save in the Name and Path of Master Document dialog.
    8. Close the document.

Expected Result:
All operations should complete successfully.

End of test case
