Test Case: aut_rpt02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create a report from Autopilot
Description:
    1. From the menu bar,  select File - Autopilot - Report
    2. In the Choose Database dialog select Bibliography from the Data Source.
         select biblio in the tables or queries drop down box.
    3. In the existing fields, select the following and put them in the Fields in report
        window using the -> button.
         Address, Annote, Author, Booktitle.
    4. Select Next.
    5. In the Report Autopilot - Labelling Fields dialog select Next.
    6. In the Report Autopilot - Grouping dialog select Next.
    7. In the Report Autopilot - Sort Options, select Next.
    8. In the Report Autopilot - Choose Layout select Next.
    9. In the Report Autopilot - Save Report, select Create.
    10. A report should now be visible with headings
    'Address, Annote, Author and Booktitle'.
    11. Close the report document.

    Expected Result:
    All operations should complete successfully.

     

    End of test case
