Test Case: file_lab02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create a labels document
Description:
    1. From the menu bar,  select File - New - Labels.
    2. The Labels dialog should be launched.
    3. Ensure there are three tab keys on the dialog showing Labels, Format and Options
    4. Under options ensure Entire Page is selected on the Labels dialog.
    5. Select New Document on the Labels dialog.
    6. Ensure a Labels document is created.
    7. Close the Labels Document.
Expected Result:
All operations should complete successfully.

 

End of test case
