Test Case: aut_mem02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create a memo from Autopilot
Description:
    1. From the menu bar,  select File - Autopilot - Memo.
    2. The Autopilot Memo dialog should be launched.
    3. Ensure Modern  is selected on the Autopilot Memo dialog.
    4. Select Create on the Autopilot Memo dialog.
    5. Ensure OpenOffice.org creates an Autopilot Memo.
    6. The document should show
        A heading showing "Internal Memo".
        followed by a box showing
        From:
        To:
    7. Close the Memo Document.
     
     
Expected Result:

All operations should complete successfully.
 

End of test case
