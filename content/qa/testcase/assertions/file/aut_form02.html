Test Case: aut_form02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create an Autopilot form
Description:
    1. From the menu bar,  select File - Autopilot - Form.
    2. The Autopilot Form dialog should be launched.
    3.
         3.1 Ensure Data Source - Bibliography  is selected on the Autopilot Form dialog.
         3.2 Ensure Table or queries - biblio is selected on the Autopilot Form dialog.
         3.3 Select Author in  Exisiting fields in the Autopilot Form dialog.
         3.4 Place Author field in the Fields in form area of the Autopilot Form dialog
                by selecting the "->" key.
    4. Select Next>>  on the Autopilot Form dialog.
    5. Select Create  on the Autopilot Form dialog.
    6. Save document as test_form.sxw.
    5. Ensure OpenOffice.org creates an Autopilot Form.
    6. The document should show
        a list of names under heading Author.
    7. Close the Form Document.

     
Expected Result:
All operations should complete successfully.

 

End of test case
