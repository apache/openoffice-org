Test Case: exp_pdf

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can create and save a pdf document
Description:
    1. Open a new text document, enter some text, e.g. "short text message"
    2. Select FIle- Export as PDF.
    3. In the export as PDF dialog enter a name for the new file e.g. first
    4. Using ggv or acroread open the new file e.g first.pdf .
    5. The user should see the short text message in the opened file.
 

Expected Result:

All operations should complete successfully.


End of test case
