Test Case: aut_euro02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org Euro Converter functions correctly

Requirements:
A sample punts.sxc file is needed to perform this test.
samples

Description:
    1. From the menu bar,  select File - Autopilot - Euro Converter.
    2. The Autopilot Euro Convertr dialog should be launched.
    3. In the Autopilot Euro dialog select
        3.1 Extent - Complete directory
        3.2 Currencies - Irish Punt.
        3.3 Source Directory - the directory where punts.sxc is stored.
        3.4 Target Directory - directory where converted document is to be stored.
        3.5  Ensure all other options are checked off.
    4. Select Convert in the  Autopilot Euro dialog.
    5. The Euro Converter dialog should show the files which were converted.
    6. Select Finish on the Euro Converter dialog.
    7. Open the converted punts.sxc file, all currencies should be list in the euro format.
Expected Result:
All operations should complete successfully.

 

End of test case
