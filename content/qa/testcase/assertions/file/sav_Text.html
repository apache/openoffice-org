Test Case: sav_Text

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can save in text format
Description:
    1. From the menu bar,  select File - New - Text Document
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message"
    4. From the menu bar,  select File-Save As.
    5. The Save As dialog should be launched.
    6. Enter Texttestfile in the File name box in the Save As dialog.
    7. Select Text  in the Save as type box in the Save As dialog.
    8. Select Save in the Save As dialog.
    9.  Close the document.
    10. From the menu bar,  select File - Open.
    11. The file Open dialog should be launched.
    12. While viewing the details of the file list (third button to right of look in box)
           ensure that Texttestfile is of type
     
              Windows:-  Text Document
              Linux:-        Text File
              Solaris:-     Text File

    13. Select Texttestfile, ensure it  opens successfully.
    14. Close the document.

Expected Result:
All operations should complete successfully.


End of test case
