Test Case: sen_mail02

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can send a mail using an external mail program

Requirements:

Solaris/Linux         Netscape 6.1 or later
Windows                 Outlook 5.0 / Netscape 6.1  or later

Ensure OpenOffice.org is using a valid external program for mail, check under Tools - Options - OpenOffice.org - External Programs.

If no valid external program has been selected by OpenOffice.org edit one here.
Netscape or Outlook need to be configured to send mail before this test can be executed.
Description:
    1. From the menu bar,  select File - New - Text Document
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message"
    4. From the menu bar,  select File - Send - Document As Email.
    5. A New Message (Outlook) or Compose (Netscape) window will be launched.
    6. Send the mail as normal to recipient(s).
    7. Ensure the mail is delivered.
Expected Result:

OpenOffice.org should send short text message attached to an email.
 

End of test case
