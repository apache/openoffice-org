Test Case: config_06

Test Area: Solaris  - Linux - Windows

Test Aim
Verify the configuration dialog is functioning in OpenOffice.org.

Description:

      1. From the menu bar,  select - Tools - configure.
      2. The Configuration dialog should be launched.
      3. Ensure there are 5 tab settings,
           Menu, Keyboard, status Bar, Toolbars and Events.
           select each tab setting.
      4. Click on Cancel in the Configuration dialog.
Expected Result:
All operations should complete successfully.

End of test case
