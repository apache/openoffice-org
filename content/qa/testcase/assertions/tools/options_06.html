Test Case: options_06

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org scrolls through the tools-options tree successfully.

Description:

      1. From the menu bar,  select - Tools -options.
      2. The Options - OpenOffice.org dialog should be launched.
      3. In the Options - OpenOffice.org dialog, left click on the top most option in the options tree.
      4. Use the down arrow on the keyboard to scroll through all options.
      5. Ensure the tree expands to show all options as the scroll through is happening.
      6. Click on Cancel in the Options - OpenOffice.org dialog.
Expected Result:
All operations should complete successfully.

End of test case
