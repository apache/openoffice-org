Test Case: macro_06

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org records a simple macro, assign it and playback.

Description:

      1. Open a new text document.
      2. From the menu bar,  select - Tools - macros - record macro.
      3. Enter some text e.g. Short text message.
      4. Select all the text by pressing ctrl-a
      5. Select Format-Character, select the font effects tab, change the Fone colour to red, select OK.
      6. Select the B for bold in the Object Bar.
      7. Select Stop Recording.
      8. In the Macro dialog select soffice,Standard,Module1 and save as Main.
      9. Select Tools - Configure, select the keyboard tab.
      10. In the Shortcut keys window select F10,
             In the Category window select OpenOffice Basic, Standard, Module1.
             In the function window select Main.
      11. Select the Modify button, select OK
      12. Exit and restart OpenOffice with a new text document.
      13. Press F10, Short text message coloured red in bold font face should be entered to the text document.
Expected Result:
All operations should complete successfully.

End of test case
