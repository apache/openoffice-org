Test Case: aut_for06

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org autocorrect/autoformat functions correctly.

Description:

    1. From the menu bar,  select File - New - Text Document.
    2.  Type 'abotu' on the text line, hit <cr>, it should now read   About;
    3. Type '1/2' on the text line hit <cr>,  it should now read   ½
    4. Type 'abouta' on the text line , hit <cr>, it should now read 'about a'
    5. From the menu bar,  select Tools - AutoCorrect/AutoFormat.
    6. The Autocorrect/AutoFormat dialog should be launched.
    7. Ensure there are 5 tab settings,
        Replace, Exceptions, Options, Custom Quotes and Word Completion.
        Select each tab setting.
    8. In the Autocorrect/AutoFormat dialog  Select Cancel.
    9. Close the text document.
Expected Result:
All operations should complete successfully.

End of test case
