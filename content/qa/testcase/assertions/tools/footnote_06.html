Test Case: footnote_06

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org launches the footnote settings dialog successfully.

Description:

1. From the menu bar,  select - Tools - Footnotes.
2. The Footnote Settings dialog should be launched.
3. Ensure there are 2 tab settings,
     Footnotes and Endnotes.
     select each tab setting
4. Click on Cancel in the Footnote Settings dialog.
Expected Result:
All operations should complete successfully.
 

End of test case
