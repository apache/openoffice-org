Test Case: ins_sch03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can launch the insert special characters dialog in a text document.

Description:

1. From the menu bar,  select File - New - Text Document.
2. A new text document should be launched.
3. From the menu bar,  select Insert - Special Character.
4. The Special Characters dialog should be launched.
5. Select the '%' symbol in the Special Characters dialog.
6. Select Cancel in the Special Characters dialog.
7. Close the text document.
Expected Result:
All operations should complete successfully.

 

End of test case
