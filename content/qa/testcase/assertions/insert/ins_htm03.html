Test Case: ins_htm03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can launch the insert hyperlink and bookmark dialogs in a html document
Description:
 
1. From the menu bar,  select File - New - Html Document.
2. From the menu bar,  select Insert - Hyperlink.
3. The Hyperlink insert dialog should be launched.
  In the hyperlink dialog select each of the icons in the left hand menu bar.
  Internet, Mail&News, Document and NewDocument.
After selecting each icon ensure the heading on the left side of the dialog matches with the selected icon.
4.  Close the insert hyperlink dialog.
5. From the menu bar,  select  Insert Bookmark.
6. The Insert Bookmark dialog should be launched.
7. Select Cancel in the Insert Bookmark dialog.
8. Close the html document

 
 
 

Expected Result:

All operations should complete successfully.

End of test case
