Test Case: ins_mbk03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can insert a manual page break into a text document.

Description:

1. From the menu bar,  select File - New - Text Document.
2. A new text document should be launched.
3. From the menu bar,  select Insert - Manual Break
4. The Insert Break dialog should be launched.
5. In the Insert Break dialog select page break and click OK.
6. The cursor position in the document should now be on the second page.
7. Close the text document.
Expected Result:
All operations should complete successfully.

End of test case
