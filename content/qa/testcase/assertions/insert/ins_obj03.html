Test Case: ins_obj03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can insert a chart object into a text document.

Description:

1. From the menu bar,  select File - New - Text Document.
2. A new text document should be launched.
3. From the menu bar,  select Insert - Object - Chart.
4. The Autoformat Chart (1-4) should be launched.
5. Click on Create in the Autoformat Chart.
6. A Bar Chart should be inserted into the document.
7. Close the text document.
Expected Result:
All operations should complete successfully.

 

End of test case
