Test Case: ins_frm03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can insert a frame into a text document
Description:
      1. From the menu bar,  select File - New - Text Document.
      2. From the menu bar,  select Insert - Frame.
      3. The Frame  dialog should be launched.
      4. Ensure there are 8 tab settings,
          Type, Options, Wrap, Hyperlink, Borders, Background, Columns and Macro.
          Select each tab setting.
      5. Select OK in the Frame dialog.
      6. Ensure a frame is inserted to the documnet, rezie this frame be selecting one of the corners
          with the mouse and dragging it.
      7. Close the document.
Expected Result:
All operations should complete successfully.

End of test case
