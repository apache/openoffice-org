Test Case: ins_gra03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can insert a graphics ( type gif) file into a text document.
 
Requirements:
A *.gif graphic file is needed to perform this test.
samples

Description:

      1. From the menu bar,  select File - New - Text Document.
      2. From the menu bar,  select Insert - Graphics - From File.
      3. The Insert Graphics dialog should be launched.
      4. Select insert on the Insert Graphics dialog, ensure the type gif graphic is inserted correctly.
      5. Close the document.
Expected Result:
All operations should complete successfully.

End of test case
