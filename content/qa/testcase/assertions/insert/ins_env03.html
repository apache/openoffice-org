Test Case: ins_env03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can insert an envelope.

Description:

      1. From the menu bar,  select File - New - Text Document.
      2. A new text document should be launched.
      3. From the menu bar,  select Insert Envelope.
      4. The Envelope dialog should be launched.
      5. Ensure there are 3 tab settings,
          Envelope, Format and Printer.
          Select each tab setting.
      6. Select Insert in the Envelope dialog, ensure an envelope is inserted.
      7. Close the text document.
Expected Result:
All operations should complete successfully.

End of test case
