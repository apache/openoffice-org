Test Case: ins_fld03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can insert the Date field into a text document

Description:

      1. From the menu bar,  select File - New - Text Document.
      2. From the menu bar,  select Insert - Field - Date.
      3. Ensure today's date should be inserted into the document.
      4. Close the document.
Expected Result:
All operations should complete successfully.

End of test case
