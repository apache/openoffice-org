Test Case: ins_sec03

Test Area: Solaris  - Linux - Windows

Test Aim Test Aim
Verify OpenOffice.org can insert a 2 column section in a text document.

Description:

      1. From the menu bar,  select File - New - Text Document.
      2. A new text document should be launched.
      3. From the menu bar,  select Insert - Section
      4. The Insert Section dialog should be launched.
      5. Ensure there are 4 tab settings
          Section, Columns, Background and Footnotes/Endnotes.
          Select each tab setting.
      6. Under Columns in the  Insert Section dialog,
           set Columns to '2', click Insert in the Insert Section dialog,
      7. A line with 2 columns should be inserted to the text documnet.
      8. Close the text document.
Expected Result:
All operations should complete successfully.
End of test case
