Test Case: ins_flfr03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can insert a floating frame into a text document
Requirements:
Test case  sav_RTF must be completed before executing this test case

Description:

      1. From the menu bar,  select File - New - Text Document.
      2. From the menu bar,  select Insert - Floating Frame.
      3. The Floating Frame Properties  dialog should be launched.
      4. In the contents box select RTFtestfile as previously saved  in sav_RTF test assertion.
      5. Ensure a frame is inserted into the document containing the contents on the RTFtestfile.
      6. Close the document.
Expected Result:
All operations should complete successfully.

End of test case
