Test Case: ins_file03

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can insert an RTF type text document

Requirements:
Test case  sav_RTF must be completed before executing this test case

Description:

      1. From the menu bar,  select File - New - Text Document.
      2. From the menu bar,  select Insert - File
      3. The file Insert dialog should be launched.
      4. Select RTFtestfile as previously saved  in sav_RTF test assertion.
      5. Ensure "short text message" is inserted into the document.
      6. Close the document.
Expected Result:
All operations should complete successfully.

End of test case
