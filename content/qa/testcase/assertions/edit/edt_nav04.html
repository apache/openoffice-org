Test Case: edt_nav04

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org Navigator launches and closes successfully

Description:

    1. Open a new text file.
    2. From the menu bar,  select Edit- Navigator.
    3. The navigator window should be launched
    4. Press F5 , the navigator window should disappear.
Expected Result:
All operations should complete successfully.

End of test case
