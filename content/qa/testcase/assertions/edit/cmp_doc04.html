Test Case: cmp_doc04

Test Area: Solaris - Linux - Windows


Test Aim
Verify OpenOffice.org can compare two text documents
Description:
    1. From the menu bar,  select File - New - Text Document
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message"
    4. From the menu bar,  select File-Save As.
    5. The Save As dialog should be launched.
    6. Enter OO60-cmp1 in the File name box in the Save As dialog.
    7. Select OpenOffice.org 6.0 Text Document in the Save as type box in the Save As dialog.
    8. Select Save in the Save As dialog.
    9. Close the document.
     
    10. From the menu bar,  select File - New - Text Document
    11. A new text document should be launched.
    12. Enter a short text message e.g. "short text message messagessss"
    13. From the menu bar,  select File-Save As.
    14. The Save As dialog should be launched.
    15. Enter OO60-cmp2 in the File name box in the Save As dialog.
    16. Select OpenOffice.org 6.0 Text Document in the Save as type box in the Save As dialog.
    17. Select Save in the Save As dialog.

    18. From menu bar select  Edit - Compare Document. An Insert dialog should appear.
    19. Select  OO60-cmp1, select Insert.
    20. An accept or reject changes dialog should appear, and the word ' messagessss '  should be highlighted.
    21. Select Accept on the dialog. The word ' messagessss ' should remain in the document.
    22. Close the dialog and close the document.

 
Expected Result:
All operations should complete successfully.


End of test case
