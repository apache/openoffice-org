Test Case: help_01

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org launches the help contents successfully.

Description:

    1. From the menu bar, select Help - Contents.
    2. The OpenOffice.org Help Contents page should be launched.
    3. In the Help Contents page select each of the following Tabs
    Contents, Index, Find and Bookmarks.
    4. Under Contents ensure there are help contents for
    Writer, Draw, Basic, Math, Impress and Calc.
Expected Result:
All operations should complete successfully.

End of test case
