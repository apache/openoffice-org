Test Case: help_04

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org shows help for draw correctly.

Description:

    1. From the menu bar, select Help - Contents.
    2. The OpenOffice.org Help Contents page should be launched.
    3. In the Help Contents page select the Contents Tab.
    4. Double click on  Help for Draw.
    5. Under Help for Draw double click on Overview
    6. Ensure  the contents in the right hand frame shows
        "Welcome to the OpenOffice.org Draw Help"

     
Expected Result:
All operations should complete successfully.

End of test case
