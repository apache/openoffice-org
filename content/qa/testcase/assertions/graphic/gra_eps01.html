Test Case: gra_eps01

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org opens a type .eps graphic file correctly.

Requirements:
A *.eps graphic file is needed to perform this test.
samples

Description:

      1.  From the menu bar,  select File - open.
      2. The File Open dialog should be launched.
      3. Enter the name of the .eps file type, select open in File Open dialog.
      4. Close the graphic file.
Expected Result:
OpenOffice.org should launch a new window displaying the graphic file.

 

End of test case
