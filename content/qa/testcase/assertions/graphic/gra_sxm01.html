Test Case: gra_sxm01

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org opens a type .sxm file correctly.

Requirements:
A *.sxm file is needed to perform this test.
samples

Description:

      1.  From the menu bar,  select File - open.
      2. The File Open dialog should be launched.
      3. Enter the name of the .sxm file type, select open in File Open dialog.
      4. Close the file.
Expected Result:
OpenOffice.org should launch a new window displaying the file.

 

End of test case
