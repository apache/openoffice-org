Test Case: vew_zoom04

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org text document can be zoomed to 200%

Description:

    1. From the menu bar,  select File - New - Text Document
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message".
    4. From the menu bar, select - View - Zoom.
    5. The Zoom dialog should be launched.
    6. Select 200% in the Zoom dialog, click on OK.
    7. The text should now appear to be much larger in the document.
    8. Close the text document.
Expected Result:
All operations should complete successfully.


End of test case
