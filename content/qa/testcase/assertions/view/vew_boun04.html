Test Case: vew_boun04

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can toggle selected view options on/off

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. A new text document should be launched.
    3. From the main menu bar select view,
         for each of the following toggle them off/on or on/off .
         View - Text Boundaries  on/off
         View - Field Shadings - toggle off/on
         View - Fields toggle on/off
         View - Nonprinting Characters - toggle on/off
         View - Hidden Paragraphs - toggle on/off
    4. Close the text document.
Expected Result:
All operations should complete successfully. Ensure one can view the rulers, status bars etc. being toggled off and on.


End of test case
