Test Case: vew_data04

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org can view the data source bibliography data source

Description:

     
    1. From menu bar select View - Data Sources.
    2. Two new frames should appear in the window.
    3. In the left frame expand the Bibliography data source.
    4. From the expanded tree select Tables - Biblio.
    5. A list of Identifiers, Type, Address etc. should appear in the right hand frame.
    6. From menu bar select View - Data Sources.
    7. The two new Datasource frames will now close.
Expected Result:
All operations should complete successfully.


End of test case
