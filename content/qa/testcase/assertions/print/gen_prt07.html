Test Case: gen_prt07

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org prints a plain text document successfully.

Requirements
A default printer is setup for the user.

Description:

    1. From the menu bar,  select File - New - Text Document
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message".
    4. From the menu bar select File - Print.
    5. The Print dialog should be launched.
    5. Select the printer to print the file to using the drop down box beside Name.
    6. Ensure the document printed successfully.
    7. Close the text document.
Expected Result:
All operations should complete successfully.


End of test case
