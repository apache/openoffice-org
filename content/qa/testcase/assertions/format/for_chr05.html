Test Case: for_chr05

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org text document can format characters to colour red.

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message"
    4. From the menu bar,  select Edit - Select All.
    5. From the menu bar,  select Format - Character.
    6. The Format Character dialog should appear.
    7. Ensure there are 5 tab settings,
        Font, Font Effects, Position, Hyperlink & Background.
        Select each tab setting.
    8. Under the  Font tab select the Font color to red.
    9. Ensure the color of the "short text message" is now red.
    10. Close the text document.
Expected Result:
All operations should complete successfully.


End of test case
