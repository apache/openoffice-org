Test Case: for_nubl05

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org launches the numbering/bullets dialog successfully.

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. A new text document should be launched.
    3. From the menu bar,  select Format - Numbering/Bullets.
    4. The Numbering/Bullets dialog should be launched.
    5. Ensure there are 6 tab settings
        Bullets, Numbering Type, Outline, Graphics, Position and Options
        Select each tab setting.
    6. Close the text document.
Expected Result:

All operations should complete successfully.

End of test case
