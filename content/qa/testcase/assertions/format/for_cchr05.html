Test Case: for_cchr05

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org text document can switch characters to/from lowercase/uppercase.

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message"
    4. From the menu bar,  select Edit - Select All.
    5. From the menu bar,  select Format - Case Character - Uppercase.
    6. Ensure the documnet now reads "SHORT TEXT MESSAGE"
    7. From the menu bar,  select Format - Case Character - Lowercase.
    8. Ensure the documnet now reads "short text message".
    9. Close the text document.
Expected Result:
All operations should complete successfully.


End of test case
