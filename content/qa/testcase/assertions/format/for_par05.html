Test Case: for_par05

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org text document can colour paragraph gray.

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message"
    4. From the menu bar,  select Format - Paragraph.
    5. The Format Paragraph dialog should appear.
    6. Ensure there are 8 tab settings,
         Indents & Spacing, Alignment, Text Flow, Numbering, Tabs, Drop Caps, Borders and Background.
         Select each tab setting.
    7. Under the  Background tab select the Background color to light gray.
    8. Ensure the Background color of the "short text message" is now light gray.
    9. Close the text document.
Expected Result:
All operations should complete successfully.


End of test case
