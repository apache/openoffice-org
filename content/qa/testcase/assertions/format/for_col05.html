Test Case: for_col05

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can insert 4 columnes to a text document.

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. A new text document should be launched.
    3. From the menu bar,  select Format - Columns
    4. The Format Columns dialog should appear.
    5. Set the number of Columns to 4 in  the dialog and press ok.
    6. Ensure 4 columns are shown in the document.
    7. Close the text document.
Expected Result:

All operations should complete successfully.

End of test case
