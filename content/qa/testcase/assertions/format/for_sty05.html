Test Case: for_sty05

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org can insert numbering styles 'numbering 1' into a text document.

Description:

    1. From the menu bar,  select File - New - Text Document.
    2. A new text document should be launched.
    3. Enter a short text message e.g. "short text message"
    4. From the menu bar,  select Format - Styles - Catalog.
    5. The Format Styles dialog should appear.
    6. Using the drop down menu ensure there are five style options
        Paragraph Styles, Character Styles, Frame Styles, Page Styles and Numbering Styles.
        Select each style option.
    7. Under Numbering Styles select, Numbering 1, click ok on the Format Styles dialog.
    8. The document should now read,
         "1. short text message"
    9. Close the text document.
Expected Result:

All operations should complete successfully.

End of test case
