OpenOffice.org

We need your help to complete the translation of Apache OpenOffice into Croatian!

This note is in English because we have no one to translate it into your language. The links on the page will help you download and install OpenOffice.org 3.2.1, a very old version of our product. It is missing many bug fixes, performance enhancements and even new features that are in Apache OpenOffice 4.x.

We would be happy to make Apache OpenOffice available in Croatian, but we need help completing the translation of the user interface.

If you are interested in helping translate, please review this information. You can send a note to our mailing list, L10N@openoffice.apache.org, if you want to volunteer or if you have questions.

Dobrodošli na OpenOffice.org

OpenOffice.org je projekt otvorenog koda sa motivom zajedničkog stvaranja skupa uredskih alata koji će raditi na svim značajnijim platformama i imati podršku za većinu svjetskih jezika.

Hrvatski potprojekt OpenOffice.org-a namjerava približiti OpenOffice.org hrvatskim korisnicima.

Nabrojimo neke od ciljeva:

  • Prevesti sučelje OpenOffice.org i dokumentaciju na hrvatski jezik
  • Dati podršku (preko mailing lista) novim korisnicima
  • Izgraditi mrežu iskusnih korisnika u svrhu učenja i iskorištavanja funkcionalnosti OpenOfficea
  • Promovirati korištenje OpenOffice.org, ne kao alternativnog uredskog alata, već kao jedinog multiplatformskog uredskog skupa alata
