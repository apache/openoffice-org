Java och OpenOffice.org

OpenOffice.org har vissa delar som kräver stöd för Java. För att få OpenOffice.orgs fulla funktionalitet krävs stöd för Java. Stödet används för att driva databasmotorn HSQLDB (används i våt databasprogram Base) samt vissa guider, assistativa funktioner samt hjälpmedel för personer med särskilda behov.

Om du är säker på att denna funktionalitet inte behövs, behöver datorn inte ha stöd för Java. Du rekommenderas dock att ha stöd för Java, eftersom installation.

Installation av OpenOffice.org med Java

Vid nerladdning av OpenOffice.org kan du välja om installationen skall innehålla Java-stöd eller inte. En installation med Java-stöd är ungefär 15 MB större än en installation utan Java-stöd.

Om du vet att maskinen redan har stöd för Java behöver stöd för Java inte installeras igen. Installation av ytterligare Java-stöd skadar inte maskinen, men är helt onödig.

Java installeras genom att besöka webbplatsen www.java.com.

Installation av Java i efterhand

Stöd för Java kan installeras i efterhand för att få den saknade funktionaliteten.

Java installeras genom att besöka webbplatsen www.java.com.

Java för utvecklare

För utvecklare är ett s.k. Software Development Kit mer intressant. Det innehåller stöd för Java, samt ytterligare material användbart för Java-utvecklare. Java SE SDK kan hämtas på java.sun.com
