Använda ordkomplettering

Guiden ger dig grundläggande hjälp om hur man använder och anpassar ordkomplettering.

Guiden kräver inte särskilt mycket förkunskaper.

Inledande förklaring

Ordkomplettering visar slutet av ord som OpenOffice känner igen när man skriver.
Exempel på hur det ser ut:
Ordkomplettering kan användas för att öka skrivhastigheten och brukar vara aktiverat som standard. Det kan anpassas via verktygsmenyn.

Acceptera komplettering av ord

Gör så här:

När ett ord börjar skrivas och resten av ordet visas bekräftar du med en tangenttryckning för att acceptera kompletteringen.
Oftast är det tangenten Retur som används för att acceptera kompletteringen. Vilken tangent som används kan ändras, se nedan under Anpassa ordkomplettering.

Hoppa över ordkomplettering

Gör så här:

Om du inte vill acceptera ordkompletteringen är det bara att strunta i den. Fortsätt bara skriva så försvinner kompletteringsförslaget.
Ett problem kan uppstå om kompletteringen visas när du vill trycka Retur, trots att du inte vill acceptera kompletteringen. Då kan man trycka pil-höger och sedan Retur.

Anpassa ordkomplettering

Gör så här:

  1. Klicka på Verktyg-menyn.
  2. Klicka sedan på Autokorrigering...
  3. Välj fliken Ordkomplettering
    • Om du inte vill ha ordkomplettering alls kan du klicka bort bocken framför Aktivera komplettering av ord
    • Om du vill ändra till en annan tangent för att acceptera kompletteringen väljer du ett annat alternativ i listan under Acceptera med
  4. Bekräfta ändringen genom att klicka på knappen OK eller tryck Retur
