Ansluta till MySQL-databas med JDBC

Denna guide visar hur du ansluter till en MySQL-databas med JDBC i OpenOffice.org Base.

Krav: Denna guide kräver viss vana med OpenOffice.org samt att en Java-körmiljö finns installerad.

Installation

  1. Ladda ner JDBC-drivrutinen MySQL Connector/J från MySQLs webbplats.
  2. Packa upp och installera drivrutinen enligt separat dokumentation.
  3. Starta något program i OpenOffice.org, Writer är oftast lättast.
  4. Välj Verktyg - Alternativ, och klicka på OpenOffice.org - Java.
  5. Låt markerad Java-körsmiljö vara markerad. Klicka på Klassökväg.... Dialogrutan Klassökväg visas.
  6. Klicka på Lägg till arkiv. Navigera till .jar-filen som kom med MySQL Connector/J och klicka på OK.
  7. Klicka på OK två gånger för att stänga alla dialogrutor och återvända till OpenOffice.org.
  8. När uppmaningen om att starta om OpenOffice.org kommer, stäng alla öppna dokument och OpenOffice.org-program. Avsluta även Snabbstart-programmet (vid klockan), genom att högerklicka och välja Avsluta.

Skapa databaskoppling

  • Starta OpenOffice.org Base. Databasguiden visas.
  • Välj Ansluta till en befintlig databas. Välj JDBC och klicka på Nästa.
  • I fältet Datakällans URL anger du sökvägen till databasen, enligt förmatet:
    jdbc:mysql://serveradress:3306/databasnamn
  • I fältet JDBC-drivrutinsklass anger du klassnamnet för JDBC-drivrutinen. Den är oftast:
    com.mysql.jdbc.Driver
  • Klicka på Nästa.
  • I fältet Användarnamn anger du ett ev. användarnamn som behövs för autentisering mot MySQL-databasen. Kryssa i Lösenord krävs om så är fallet. Klicka till sist på Testaanslutning för att testa anslutningen. Klicka på Nästa.
  • På nästföljande och sista sidan, gör de val som motsvarar dina behov. Dessa val kommer inte närmare att beskrivas här.
  • Klicka på Slutför.
  • Tabellerna visas nu i Base under Tabeller, och kan därifrån öppnas för ändring.

    • Kom ihåg: Du kan endast redigera tabeller som har ett fält som är Primary Key.

    Ha så roligt!
