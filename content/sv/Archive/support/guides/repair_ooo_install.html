Reparera en installation som börjat strula

Om du har en installation av OpenOffice.org som börjat visa konstiga felmeddelanden, eller där rättstavningen eller vissa tillägg slutat fungera, använder du denna guide för att försöka avhälpa felen. Felen kan uppstå om OpenOffice.org någon gång kraschat eller om en användare eller ett program ändrat i användarprofilen manuellt.

Denna guide hjälper dig återskapa profilen eller att göra en fullständig ominstallation.

Återskapa profilen

Om OpenOffice.org någon gång krachat kan din användarprofil ha skadats. Följ följande steg för att återskapa din användarprofil.

OBS! Eventuella inställningar och installerade tillägg kommer att behöva installeras om.

  1. Stäng alla öppna fönster i OpenOffice.org samt stäng Snabbstarten (är placerad vid klockan).
  2. Gå till mappen för din användarprofil som heter "user". Mappen är placerad enligt följande:
    3. Windows XP: C:\Documents and Settings\<användarnamn>\Application Data\OpenOffice.org\3\
    Windows Vista: C:\Users\<användarnamn>\AppData\Roaming\OpenOffice.org\3\
    Linux: /home/<användarnamn>/.openoffice.org/3/
    Mac OS X: /Users/<användarnamn>/Library/Application Support/OpenOffice.org/3/
  3. Byt namn på mappen till t.ex. "user_bak".
  4. Starta OpenOffice.org. Välkomstguiden kommer att visas, och din användarprofil kommer att återskapas.
  5. Ändra de inställningar som du ev. gjorde vid första installation samt installera de tillägg som du hade tidigare.
  6. Testa om problemen med OpenOffice.org är lösta. Om inte se Full ominstallation.

Full ominstallation

Om en återskapning av profilen inte löste problemet kan programfilerna ha skadats. Följ följande steg för en full ominstallation av produkten.

OBS! Eventuella inställningar och installerade tillägg kommer att behöva installeras om.

  1. Stäng alla öppna fönster i OpenOffice.org samt stäng Snabbstarten (är placerad vid klockan).
  2. Avinstallera OpenOffice.org.
  3. Se till att programmappen där OpenOffice.org var installerad är tom.
  4. Se till att OpenOffice.org-mappen för din användare är borttagen. Mappen är placerad enligt följande:
    5. Windows XP: C:\Documents and Settings\<användarnamn>\Application Data\OpenOffice.org\
    Windows Vista: C:\Users\<användarnamn>\AppData\Roaming\OpenOffice.org\
    Linux: /home/<användarnamn>/.openoffice.org/
    Mac OS X: /Users/<användarnamn>/Library/Application Support/OpenOffice.org/
  5. Installera OpenOffice.org.
  6. Ändra de inställningar som du ev. gjorde vid första installation samt installera de tillägg som du hade tidigare.
  7. Problemen med OpenOffice.org bör nu vara lösta.
