Formatera första sidan utan sidhuvud/sidfot

Denna guide hjälper dig att ändra så att den första sidan i ett dokument inte visar sidhuvud eller sidfot.

Guiden kräver viss förståelse av formatmallar och sidhuvud/sidfot samt dess syften.

Inledande förklaring

Om du använder sidhuvud eller sidfot men inte vill att dokumentets första sida ska visa dem, behöver du byta sidformatmall för dokumentets första sida.

Byt sidformatmall för första sidan

Gör så här:

  1. Se till att markören är på första sidan i dokumentet. Det kan du göra genom att klicka på första sidan eller trycka på tangenterna Ctrl + Home.
  2. Klicka på Format-menyn och välj Formatmallar och formatering.
  3. Du får då upp ett litet fönster (se bilden) med formatmallar. Välj sidformatmallar genom att klicka på den symbol som visar en sida.
  4. Du ser att Standard är valt från början. Om du använder sidhuvud eller sidfot kommer de att visas på alla sidor med sidformatmallen Standard. Klicka på Första sidan för att välja den sidformatmallen istället. Nu kommer den första sidan i dokumentet inte visa sidhuvud eller sidfot.
