Läs mer om systemkrav för olika operativsystem i dokumentationen Installations- och anpassningshandbok

Stöd för Java krävs för att viss funktionalitet skall aktiveras. Som standard medföljer inte Java i installationen. Klicka på "Med Java-stöd" under rätt operativsystem för att ladda ner ett installationspaket med Java. För mer information, se Java och Apache OpenOffice.
