OpenOffice.org Base

Det nya sättet att arbeta med databaser

OpenOffice.org Writer Calc Impress Draw Base Math

I OpenOffice.org 2 låter Base dig att arbeta med databaser på ett helt nytt sätt. Skapa och modifiera tabeller, formulär, frågor och rapporter, antingen genom att använda din egna databas eller Base inbyggda databas HSQLDB. Base ger dig valmöjligheter med guider, designvyer eller SQL-vyer för såväl nybörjare som avancerade användare.

Underhåll din information

Med Base kan du

  • Skapa tabeller för dina data - och även ändra dem efter behov
  • Underhålla index för tabeller för att komma åt data snabbare
  • Visa en tabell på olika sätt, och lägga till, ändra eller ta bort poster
  • Använda rapportguiden för att skapa imponerande datarapporter
  • Använda formulärguiden för att skapa databastillämpningar

Använd din information

Med Base kan du
  • Genomföra enkla till avancerade sorteringar för en eller flera kolumner
  • Visa utdrag av data med enkla eller avancerade filter
  • Skapa kraftfulla frågor för att visa dina data på flera sätt, inklusive sammanfattningar och flertabellsvyer
  • Skapa rapporter på många sätt i flera format med hjälp av Autopilot

I Base ingår en fullversion av HSQLDB-databasen som lagrar data i ett XML-format. Base kan även använda dBASE-filer för enkelt databasarbete. För mer avancerade användare hanterar Base även många populära databaser såsom Adabas D, ADO, Microsoft Access, MySQL samt alla databaser som stödjer ODBC- och JDBC-drivrutiner. Base stödjer även alla LDAP-kompatibla adressböcker samt vanligt förekommande format såsom Microsoft Outlook, Microsoft Windows och Mozilla.
