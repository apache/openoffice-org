Downloading Tips

  1. Download sites are organized by country; and within each country by site name. Each named site is hyperlinked. It should show as red and underlined as you hover over it with your mouse.
  2. Click on the hyperlinked named site in the country closest you. Downloads will begin once you click on the hyperlinked named site.
  3. If the download is too slow, you may stop it and choose another site (mirror). You are free to choose other countries.
  4. You will have downloaded a zip file. The next steps of checking, uncompressing, and running the installer are simple, but we recommend you read the Installation Instructions, if you have any questions. Actually, it might be best to read the instructions, anyway. Note if you are using Windows 98, you really should read the Instructions.

Md5Sums: The "md5sum" allows you to verify the integrity of the download.
