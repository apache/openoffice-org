Release Notes for the OpenOffice.org 1.0.3.1 Release

Last updated April 2003

OpenOffice.org 1.0.3.1 is a respin of the 1.0.3 release and fixes a bug in printing. OpenOffice.org 1.0.3 is the result of development on the cvs branch tagged "OOO_STABLE_1". Some builds have already been released from this branch, namely OpenOffice.org 1.0.2, 1.0.1, 1.0.0, and earlier the snapshots 641, 641C and 641D. Development was focussed on bug fixes. Thus, in these release notes, you won't find the 'features' section as in other (developer branch) releases but a section describing bug fixes.

News and Issues

  • Solaris/sparc patches
    The Solaris/sparc version needs the following patches:

    • If you have Solaris 8 (sparc), patches 108434-01 and 108435-01
    • If you have Solaris 7 (sparc), patches 106387-8
    • The patches are available at sunsolve.sun.com.
        Instructions:
      • Search for the appropriate patch numbers and download
      • Uncompress the files
      • cd to the directory containing the patches
      • As root, execute the following: 
        prompt> patchadd [patchnumber]
      • (repeat as needed for Solaris 8)
