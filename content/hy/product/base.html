Apache OpenOffice Base

Base-ը լիովին գործառնական աշխատասեղանի տվյալների շտեմարանի կառավարման համակարգ է, որը նախատեսված է բավարարել օգտագործողների լայն զանգվածների կարիքները՝ սկսած անհատական CD հավաքածուներին հետևելուց, մինչև կորպորատիվ ամսական վաճառքի հաշվետվություններ կազմելը:

Base-ն առաջարկում է հրաշագործեր՝ օգնելու օգտվողներին տվյալների շտեմարանի ձևավորման գործում (կամ Base) ստեղծելու աղյուսակներ, հարցումներ, ձևեր և հաշվետվություններ, ինչպես նաև նախապես սահմանված աղյուսակների սահմանումներով ակտիվներին, հաճախորդներին, վաճառքի պատվերներին վճարահաշիվներին և ավելին հետևելու համար:

Երբ անհատական օգտագործման շտեմարանն է միայն անհրաժեշտ, Base-ն առաջարկում է ամբողջական HSQL հարաբերական տվյալների շտեմարանի շարժիչ, որը կազմաձևված է մեկ օգտվողի համար, և պահված տվյալներով հենց Base նիշքում, ինչպես նաև dBase հարթ նիշքերի բնիկ աջակցություն:

Ձեռնարկությունում փորձված օգտագործողների համար Base-ը մատակարարում է աջակցության բնիկ վարորդներ բազմաբնույթ տվյալների շտեմարանի մի շարք շարժիչների համար՝ MySQL, Adabas D, MS Access և PostgreSQL։ Բացի այդ, JDBC և ODBC ստանդարտ վարորդներին աջակցելը թույլ է տալիս գործնականում միանալու ցանկացած տվյալների շտեմարանի:

Base-ը անխափանորեն ինտեգրվում է մնացած Apache OpenOffice հավաքածուի կիրառություններում, օրինակ ՝

  • Հասցեագրքերի տվյալների մատակարարումը փոստի միաձուլման համար Writer-ում` օգտագործելով արդյունաբերական LDAP արձանագրության ստանդարտ կամ հասցեագրքի ընդհանուր ձևաչափերը, ինչպիսիք են Microsoft Outlook-ը, Microsoft Windows-ը և Mozilla-ն;
  • Calc նիշքերումԿապված տվյալների միջակայքերի ստեղծում, տվյալների փորձնական վերլուծության համար կամ որպես գծապատկերների հիմք:

Base- ի հետ աշխատելու վերաբերյալ լրացուցիչ տեղեկություններ ստանալու համար տես՝ Apache OpenOffice Wiki Database էջերը։

Տեխնիկական տեղեկատվության կամ Base-ն էլ ավելի լավ դարձնելու համար կանգ առեք այստեղ՝ Base նախագծի էջ։
