struct OBroadcastHelperVar

virtual abstract interface template

Description
This struct contains the standard variables of a broadcaster. Helper classes only know a reference to this struct instead of references to the four members. The access to the members must be guarded with rMutex. The additional template parameter keyType has been added, because gcc can't compile addListener( const container::keyType &key ).
