680 m45 Snapshot

Release Notes

these notes contains changes between 680_m42 and  680_m45.
this release will install as OpenOffice.org 680, it comes with ugly hacked splash screen to make clear, that this is an iterims build between 1.1 and 2.0.
for snapshot builds there will be no extra source tar balls created, sources can be received from cvs by tag SRC680_m45.
