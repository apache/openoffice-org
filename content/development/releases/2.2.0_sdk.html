OpenOffice.org 2.2.0 Software Development Kit (SDK)

download

The OpenOffice.org SDK is an add-on for the OpenOffice.org office suite. It provides the necessary tools and documentation for programming the OpenOffice.org APIs and creating own extensions (UNO components) for OpenOffice.org.

What's new in OpenOffice.org 2.2.0 SDK?

  • API's
    We have introduced 63 new UNO IDL types (1 new published interface). Please see "Overview of new API's" for a roughly overview of new API's.
  • Improved Inspector example
    The Inspector example gets improved. It's now more than a simple example. It's a really useful developer tool to inspect UNO objects or simply browsing through the object hierarchy. The tool supports now the generation of code snippets for Java. Try it out, you will like it!
