OpenOffice.org 2.4.0 Software Development Kit (SDK)

The OpenOffice.org SDK is an add-on for the OpenOffice.org office suite. It provides the necessary tools and documentation for programming the OpenOffice.org APIs and creating own extensions (UNO components) for OpenOffice.org.

You can download the SDK from here!

What's new in OpenOffice.org 2.4.0 SDK?

  • API's
    The OpenOffice.org API covers now 3915 UNO IDL type definitions. For OpenOffice.org 2.4 we have introduced 37 new UNO IDL types (whereas 3 are new published types). Please see "Overview of new API's" for a roughly overview of the new types.
    The UNO IDL reference documentation gets improved by inserting missing or correct existing documentation as well by fixing errors in our autodoc tool that is used for the generation of the reference.
  • Developer's Guide update and move into the Wiki
    The Developers Guide gets updated and moved into the OpenOffice.org Wiki. That means that the SDK itself don't come with a Developer's Guide and the IDL reference documentation linked directly into the Wiki. The main purpose of moving the guide into the wiki is for maintenance reasons and the hope to get more contributions. We also hope that we will get additional localized versions of the guide to make the information available for more users all over the world.
