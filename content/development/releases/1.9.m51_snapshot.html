1.9 m51 Snapshot

Release Notes

these notes contains changes between 680_m49 and  680_m51.
this release will install as OpenOffice.org 1.9.m49, it comes with ugly hacked splash screen to make clear, that this is an interims build between 1.1 and 2.0.
for snapshot builds there will be no extra source tar balls created, sources can be received from cvs by tag SRC680_m49.

UNO: Single-interface--based services with no constructors i31194 feature-info:
A single-interface--based service written in UNOIDL as service SomeService: XSomeIfc {}; used to be interpreted the same as service SomeService: XSomeIfc; that is, both the former and the latter were considered to have a default constructor. This has been changed so that the former is now considered to have no constructors at all. This is useful for services that are not intended to be instantiated by clients, like the com.sun.star.uri.UriSchemeParser_XXX services which are intended to be accessed by client code via the com.sun.star.uri.UriReferenceFactory service only. 		speclink
