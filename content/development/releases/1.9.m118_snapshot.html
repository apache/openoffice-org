Language packs for es, fr, it, sv, de,nl, pt-BR, ja, ko, zh-CN, zh-TW are available on http://oootranslation.services.openoffice.org/pub/OpenOffice.org/680m118/ .


What's new ?

- if you miss some features in the list or the list is empty, it is possible that the feature-announcement is missing.
- or the issue wasn't added to a CWS, have a look at the issue in EIS.


Feature title TaskId Spec. abstract Spec. link Component
option to open existing databases from within the File| New| Database wizard i42477 The new database application needs an easy to use entry point to create a new database file. These database files serve as an interface between the user and the actual data source. They contain all meta information about the database. The wizard will be started by the menu sequence “File/New”. By means of this file the database can be opened similarly to an office document. speclink dba
