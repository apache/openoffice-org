OOO_STABLE_1 Branch

The OOO_STABLE_1 branch have been created on a SRC641 basis. SRC641 is the codeline for StarOffice 6.0 and the OOO_STABLE_1 branch is the codeline for OpenOffice.org 1.0.x releases.

  • Bugfixes from StarOffice 6.0 codeline (SRC641) merged into OOO_STABLE_1

  • Bugfixes for OOo 1.0.x

  • No development of features on this branch

For doing new ports (e.g. MacOSX, Irix) the OOO_STABLE_1_PORTS branch has been created. Once a port is finished, these changes will be merged back into OOO_STABLE_1 and dependent codelines.

Nominations for a OOo 1.0.x release can be made by setting the ooo1.0 keywords within IssueZilla. Which fixes got into the 1.0.x release are listed in developement.openoffice.org/releases/changelog.html.

Status:

OOo 1.0.x releases are stable.

IssueZilla Bughandling:

Query for OOo 1.0.2 fixes, nominations for OOo 1.0.3 release. Once these fixes got into a release the Issues should be verified and closed by the Issue submitter (or members of the QA project ? )

Plan:
