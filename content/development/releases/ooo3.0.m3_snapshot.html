OOO300 m3 Snapshot - Release Notes


The build includes all code changes until DEV300 m28 but they are not included in this document. Please have a look for older Release Notes (example: http://development.openoffice.org/releases/3.0.m28_snapshot.html).
These notes contains changes between OOO300 m1 and OOO300 m3.
This release will install as OOo-Dev3.0.
Sources can be received from cvs by tag OOO300_m3.

What's new ?

Currently there are no new things in OOO300 m3 since DEV300 m28.
