1.9 m56 Snapshot

Release Notes

these notes contains changes between 680_m55 and  680_m56.
this release will install as OpenOffice.org 1.9.56, it comes with ugly hacked splash screen to make clear, that this is an interims build between 1.1 and 2.0.
for snapshot builds there will be no extra source tar balls created, sources can be received from cvs by tag SRC680_m56.
 

What's new ?

- if you miss some features in the list or the list is empty, it is possible that the feature-announcement is missing.
- or the issue wasn't added to a CWS, have a look at the issue in EIS.

Chinese translation i30300 Very close to the features of the Japanese 'Hiragana' and 'Katakana' and the Korean 'Hangul/Hanja Conversion' Chinese users are accustomed to having a built in conversion between Simplified and Traditional Chinese. This conversion allows them to easily transfer Chinese written text in between simplified Chinese and traditional Chinese characters. speclink
Add Keyboard Support for Assigning Paragraph Styles i29441 OpenOffice.org (OOo) allows to assign styles (i.e. paragraph styles) via keyboard. This allows to easily assign shortcuts while typing in a document without mouse or dialog interaction. This is a big improvement for professional writers and people with disabilities who need keyboard accessibility. A set of predefined shortcuts for paragraph styles is provided for Writer only. Calc, Draw and Impress are supporting this feature too but do not offer presets by default. speclink
