3.3 m65 Snapshot - Release Notes

These notes contain changes between DEV300_m60 and DEV300_m65.
Please have a look for older Release Notes if you need to know former changes (example: http://development.openoffice.org/releases/DEV300_m60_snapshot.html).
This build will still install as OOo-Dev 3.2 but this will be changed.
Sources can be received from SVN by tag DEV300_m65.

Important note:

DEV300_m60 was the last build towards OOo 3.2 from this codeline. The next 3.2 build will be from the OOO320 codeline.
DEV300_m61 is the first build that goes on with the unstable trunk. It does not belong to OOo 3.2.
See also the blog article on GullFOSS.

buildverbosity 
quiten the build.

The idea is to bring some new life into the former idea of three
different build verbosity levels
(http://blogs.sun.com/GullFOSS/entry/shut_up_or_better_tell)
where "quiet" ("verbose=false") emits nearly no output, "debug"
("verbose=true") quite excessive output, and "normal" ("verbose"
not set) something inbetween.

The current output of a build is more like one would expect from
a "debug" level, so a lot of the current output will be silenced,
and only emitted if explicitly requested.
