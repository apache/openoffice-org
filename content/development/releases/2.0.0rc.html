2.0.0rc1 Release Candidate1 - Release Notes

these notes contains changes between SRC680_m130 and  OOO680_m1.
this release will install as OpenOffice.org 2.0.0
Sources can be received from cvs by tag OOO680_m1.

localized installation packs for es, fr, it, sv, de, nl, pt-BR, ja, ko, zh-CN, zh-TW are available on http://oootranslation.services.openoffice.org/pub/OpenOffice.org/2.0.0rc1/ .

Known issue: OpenOffice.org 2.0.0rc1 complains that there is a newer version of OpenOffice.org ist installed, if there is a 1.9.130 already installed. To install the RC1, please uninstall 1.9.130 first (this is a Windows platform problem only).
