2.0.0rc3 Release Candidate3 - Release Notes

these notes contains changes between OOO680m2 and  OOO680_m3.
this release will install as OpenOffice.org 2.0.0
Sources can be received from cvs by tag OOO680_m3.

localized language packs for es, fr, it, sv, de, nl, pt-BR, ja, ko, zh-CN, zh-TW are available on http://oootranslation.services.openoffice.org/pub/OpenOffice.org/2.0.0rc2/ .They don't need to be updated for rc3.

Fixed issues since rc2:
  • 55330 Incorrect attributes of graphics objects after saving in .odt format
  • 55627 Mac OS X icons are broken in m2 ( -kb options forgotten for commit)
  • 55679 create-package.sh and version numbers
  • 55719 No printing of text when UI language is RTL
