OpenOffice.org 2.1.0 Software Development Kit (SDK)

download

The OpenOffice.org SDK is an add-on for the OpenOffice.org office suite. It provides the necessary tools and documentation for programming the OpenOffice.org APIs and creating own extensions (UNO components) for OpenOffice.org.

What's new in OpenOffice.org 2.1.0 SDK?

  • API's
    We have introduced 13 new UNO IDL types. Please see "Overview of new API's" for a roughly overview of new API's.
  • 2 new examples
    With this version of the SDK we provide two new examples showing some of the latest new features. One is an example showing how to use the simple license feature for extensions, see <sdk>/examples/DevelopersGuide/components/SimpleLicense. The second one is a new C++ example for complex toolbars, see <sdk>/examples/cpp/comlpextoolbarcontrols. Both examples are worth to try it out!
