Apache Forrect y OpenOffice.org

Pregunte a Reinhard Poetz sobre el proyecto Forrest de apache. y suinteracción con OOo, el respondió diciendo:

Forrest es un proyecto orientado a documentación estandarrd basado en Apache Cocoon, proveiendo de hojas de estilo XSLT y esquemas, imagenes, y otros recursos. Forrest usa esto para renderizar el contenido fuente de XML en una pagina web via la linea de comando, robots, o una aplicación web dinamica. Una posible fuente de XML son los documentos de Writter en OpenOffice.org.

Actualmente solo trabaja en la dirección OOo -> Apache Forrest. Yo integré el soporte para OOo por que ofrece un ambiente amigable para editar archivos. Aunque soy un desarrollador Apache-Cocoon, no me gusta editar archivos XML en orden de escribir un documento.
