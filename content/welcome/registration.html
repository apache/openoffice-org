Thank you for using OpenOffice.org - now Apache OpenOffice

Get the latest version

OpenOffice.org is now Apache OpenOffice.

Please note: You have just installed an outdated version of OpenOffice and are missing new features and bug fixes. The latest version is OpenOffice 3.4.1, released in August 2012. You can download the latest version, free, from: the official OpenOffice download site.

If the latest version of Apache OpenOffice is not yet available in your language, this means that we need volunteers to help us with the translation. If you are interested, we have a page on How to Help Translate Apache OpenOffice. Your volunteer contributions help us bring free software to the entire world.
