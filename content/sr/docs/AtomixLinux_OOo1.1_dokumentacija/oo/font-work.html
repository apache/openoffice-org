Font work ( Word Art )

Alat Font Work služi za dodatno ukrašavanje teksta u Text Box-u.

Kada selektujete Text Box birate u padajućem meniju Format / Fontwork tada se otvori prozor.

  1. Oblik ispisa teksta
  2. Zakrivljenost teksta
  3. Širina teksta i poravnanje
  4. Udaljenost u odnosu na poziciju tekstboksa
  5. Konture teksta senčenje i boja senke
