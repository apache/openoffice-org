Windows XP/2003/Vista 下 Apache OpenOffice 3.4 快速入门指南

欢迎到Apache OpenOffice的Windows环境下快速入门指南。 下面你会找到一步一步的指示在您的系统上下载并安装Apache OpenOffice版本3.4的方法。您可能需要在开始下载和安装Apache OpenOffice之前打印本说明书以供参考。

如果您对Windows操作系统有经验，您可以使用在下一节中提供的说明操作。 否则，跳转到详尽的安装说明部分.

下载和安装过程概述

要下载和安装Apache OpenOffice 3.4，请按照此清单:

  1. 查看Apache OpenOffice的系统要求.
  2. 下载和安装Java JRE如果你需要依赖Java的功能.
  3. 下载Apache OpenOffice 3.4.
  4. 请以作为管理员权限登录您的电脑(如果需要).
  5. 解压并安装下载的Apache OpenOffice 3.4的文件.
  6. 运行Apache OpenOffice 3.4 以确保安装成功.

详尽的安装说明

本节详细介绍了如何在您的电脑上安装Apache OpenOffice。

你将需要做的第一件事是彻底阅读一遍这些说明，并使用 Apache OpenOffice网站的解答你可能有的任何问题。该网站包含的文档为各层次用户，从初学者到高级用户准备的。是新用户起步一个好地方，新的Apache OpenOffice用户可以浏览一下FAQ。

注意: 以管理员的身份登录如果您的计算机只允许具有管理员权限人来安装程序。

查看系统要求

您的电脑需要这些资源作为最低限度运行Apache OpenOffice 3.4:

  • Windows XP, Windows 2003, Windows Vista
  • 256MB内存(推荐512MB)
  • 至少650 MB的可用磁盘空间（包括JRE）通过下载默认安装。 安装完成后，删除临时安装文件，Apache OpenOffice将使用约440 MB的磁盘空间。
  • 1024 x 768 or higher resolution with at least 256 colours

下载Apache OpenOffice 3.4

从我们的网站下载Apache OpenOffice，你不需要登录。此外，它不花费什么，你的成本是连接到互联网，并下载一个大文件。

如果您的连接不是非常快（或您是在一个拨号连接），你可以考虑一个程序，使您可以暂停/继续下载。我们这里列出了一些流行的下载管理器.

要下载最新版本的Apache OpenOffice:

  1. 在 下载 Apache OpenOffice中,点击显示在网页顶部的较大的绿色下载按钮。

注：可以找到很多的资料，下面是在最近来自开发者维基的 Win 86 安装向导，第四节. 安装程序安装 .

解压和安装已下载的Apache OpenOffice3.4文件

  1. 无论是使用Windows Explorer或Mozilla/Netscape的下载管理器，请导航到您下载的Apache OpenOffice的文件。

    Windows资源管理器用户会希望找到的文件，然后双击以启动解包和安装程序。 另外，Mozilla/ Netscape的用户需要打开下载管理器，选中文件，然后单击“启动文件。 在这两种情况下，解压程序将启动。

    欢迎窗口将会提示您已经下载好了Apache Openoffice 3.4.点击 下一步 继续安装。

  2. 选择文件夹窗口显示出来。接受的文件夹名称显示在文本框中，如果需要更好可以输入保存解压缩文件的文件夹的名称。单击解压，程序将开始解压。
  3. 文件解压缩后，Apache OpenOffice3.4 安装向导被显示出来。单击“下一步”继续安装Apache OpenOffice 3.4。按照提示，Apache OpenOffice将被安装到您的系统上，安装完成您就可以使用了。
  4. 阅读“欢迎”窗口中的信息，然后单击“下一步”。

    许可协议窗口出现。

    提示: 您可以使用滚动条来查看所有的许可证文本。如果你不打算接受许可，您将 在安装程序关闭之前收到提示确认您的选择。

  5. 阅读许可，选择“我接受许可协议中的条款”，单击“下一步”。

    客户信息窗口出现。

    你可以不填这个表单，也可以输入你的用户名，如果有组织名也可以一并输入。

    提示: 如果你以管理员身份登录系统会提示您是否安装给所有用户.

  6. 填完客户信息然后单击“下一步”

    安装类型窗口显示出来.

  7. 选择一个安装类型(我们推荐 完整)，然后点击“下一步”.

    接下来就是文件类型窗口.

  8. 选择你希望用Apache OpenOffice打开的文件类型。

    默认所有文件类型是选中的，这就意味着打开这些类型的时候会启动Apache OpenOffice来打开。反选这些类型就不会把这些文件关联到Apache OpenOffice。

    注意：如果您改变了主意，返回到以前的状态是非常繁琐的。您需要将所有文字处理，电子表格或演示文件重新关联相应的文件类型。

    您可以使用没有与Apache OpenOffice关联的文件类型，通过启动Apache OpenOffice然后打开该文件： 从“文件”菜单中选择“打开”。

  9. 选定了文件类型，单击“下一步”。

    “准备安装程序”窗口会显示出来。

  10. 点击“返回”按钮，返回到上一个的窗口更改安装选项，单击“安装”开始安装过程。

  11. 安装向导完成的窗口显示时，单击“完成”按钮，。

  12. 如果你以管理员身份登录，注销。之后每个用户都可以从程序菜单中调用Apache OpenOffice。这将使用户在配置Apache OpenOffice工作站副本的时候不需要投入更大精力在那部分用户身上。

运行Apache OpenOffice。以确保安装成功

当你第一次运行Apache OpenOffice安装程序成功后，你将被提示是否接受 许可证，请输入您的用户名，注册您的副本。

  1. 登录使用您的用户帐户，如果成功安装了Apache OpenOffice，它会出现在“程序” “开始”菜单子菜单。

  2. 在一个Apache OpenOffice组件上点击（我们推荐Writer），将会显示欢迎窗口。

  3. 单击“下一步”，会显示“许可协议“窗口。

  4. 滚动滚动条阅读许可，然后单击“接受“以继续。转移个人数据窗口中显示出来。

    如果您希望重新使用以前的安装的个人数据单击[下一步]。

    如果你不希望从以前安装的版本重用任何设置，取消选中该复选框并单击 [下一步]。

  5. “用户信息”窗口显示出来。

    系统会提示您输入您的姓名。这是用来在文档属性，模板，记录所做的更改的人的。这是非常有用的，如果你与他人工作在同一个文档，但你并不需要为了继续而完成此信息。使用程序时这些信息也可以稍后输入。

  6. 完成表单并单击“下一步”，会显示“注册”窗口。

    你将需要连接到互联网，如果你选择“我要注册”，然后单击“完成”。您的浏览器将会打开Apache OpenOffice网站并显示一个欢迎消息，然后将您重定向到用户调查页面。

    如果您没有连接到互联网，你可以选择稍后注册。

    Apache OpenOffice的注册和完成用户调查是可选的，不需要你充分利用Apache OpenOffice.

    提示: 在Apache OpenOffice程序的注册与订阅各种OpenOffice邮件列表和注册自己的“www.openoffice.org”网站帐号，提交错误报告，增强请求和对项目作出贡献是完全独立的，也就是说，二者不是同一回事。

您已成功安装并运行ApacheOpenOffice。您现在应该，联机帮助（选择“Apache OpenOffice帮助“，从”帮助“菜单中），或连接到互联网，通过点击技术支持选项卡提供的任何网页，从 网站上获得支持。

非常欢迎并感谢您使用Apache OpenOffice！

Linux安装

下面的预安装步骤是我们推荐对Linux分发版而言。这些应采取的措施保证你有一个干净的Apache OpenOffice安装由于内部因素和/或安装有LibreOffice，如果在您的系统存在的话。

  1. 检查LibreOffice是否吧OpenOffice二进制文件“改道”.
    在终端中输入"whereis soffice".

    如果被链接到LibreOffice，删除符号链接。 （找您最喜欢参考资料去看如何做到这一点）

    这不会删除LibreOffice，这将禁止LibreOffice重定向正常的OpenOffice二进制文件-- 典型地 /usr/bin/soffice -- libreoffice把它替换了.
    提示: 根据您的具体情况，您可能需要卸载LibreOffice才能使完全Apache OpenOffice正常工作。
  2. 如果您有较旧的，3.3.0版本（或更低）的OpenOffice.org安装，请将其完全删除。
    这对于大多数用户来说最安全的方式是使用GUI软件包管理器，搜索OpenOffice并移除所有的包。

基于RPM的Linux安装

前提条件

如果你希望Java集成，你要确保你有安装最新的JRE。它的JRE应至少为1.6版。 你可以找到JRE for Linux在 Java JRE for Linux 下载站点。 另外，JRE可能被包含在你的发行版的安装介质作为一个完整的Java 开发环境的一部分。

安装步骤

  1. 查看系统要求
  2. 下载Apache OpenOffice你最喜欢的Linux版本
  3. 在 安装向导查看有用的信息和安装选项。
  4. 解压下载的镜像，为安装做好准备.
    使用下面的命令：tar -xvzf “linux包的名字”.tar.gz
    "linux包的名字"应该替换你刚刚下载的tar.gz包的前一部分，也就是除了tar.gz以外的那一串字符.

    这将创建一个安装目录。
    安装目录的名称很可能会有语言名的缩写，例如，en-US.
  5. su到 root 用户,如果有必要，浏览到Apache OpenOffice的安装目录（解压缩后的档案的位置）.
    您可能需要以root身份运行rpm命令来安装软件。
  6. cd到安装目录下RPMS子目录.
    你应该能看到许多rpm包和一个名为"desktop-integration"的子目录.
  7. 输入rpm -Uvih *rpm，安装新版本.
    默认将会安装/升级Apache OpenOffice到/opt目录.

    或者不用命令行,你也可以使用GUI程序安装rpm包, 引用安装目录,安装所有rpm包在top level.这也可以帮助你监测依赖问题，如果有的话.
  8. 安装桌面集成功能
    在安装目录下cd 到 desktop-integration,
    根据软件包管理器/系统，安装相应的桌面界面的RPM包。
  9. 最后，启动 Apache OpenOffice 3.4.x确保它正常工作.

前提条件

如果你希望Java集成，你要确保你有安装最新的JRE。它的JRE应至少为1.6版。 你可以找到JRE for Linux在 Java JRE for Linux 下载站点。 另外，JRE可能被包含在你的发行版的安装介质作为一个完整的Java 开发环境的一部分。

安装步骤

  1. 查看系统要求
  2. 下载Apache OpenOffice你最喜欢的Linux版本
  3. 在 安装向导查看有用的信息和安装选项。
  4. 解压下载的镜像，为安装做好准备.
    使用下面的命令：tar -xvzf “linux包的名字”.tar.gz
    "linux包的名字"应该替换你刚刚下载的tar.gz包的前一部分，也就是除了tar.gz以外的那一串字符.

    这将创建一个安装目录。
    安装目录的名称很可能会有语言名的缩写，例如，en-US.
  5. su>到 root 用户,如果有必要，浏览到Apache OpenOffice的安装目录（解压缩后的档案的位置）.
    您可能需要以root身份运行deb命令来安装软件。
  6. cd到安装目录下DEBS子目录.
  7. 输入 sudo dpkg -i *.deb.
    默认将会安装/升级Apache OpenOffice到/opt目录.

  8. 安装桌面集成功能
    在安装目录下 cd到desktop-integration ,
    根据软件包管理器/系统，用dpkg安装相应的桌面界面。
  9. 最后，启动 Apache OpenOffice 3.4.x确保它正常工作。

Linux发行版的软件包(未经Apache OpenOffice开发者编译的)

在Apache OpenOffice的网站（镜像）提供的Linux下载 对于基于deb或rpm的包管理器是通用的。如果你有经验，在您的系统上直接安装这些或类型的程序包，这些类型的 安装也可以为你工作。他们不是指定到一个特定的Linux发行版。

现在，许多Linux发行版本的Apache OpenOffice在它们自己的原生软件包管理器中可以找到(.rpm, .deb,等.) 无论是从他们自己资料库或社区资料库的支持。这些软件包被构建 并由发行版自己的团队或社区成员支持，而不是由Apache OpenOffice。许多发行版都有 严格的政策的许可，这意味着，他们可能会选择禁用Apache OpenOffice不符合他们的标准依赖软件（例如，在Fedora RPM包的Apache OpenOffice已禁用所需要的Java）。

提示: 请使用第三方构建的版本前仔细检查它是否实际上包括所有Apache OpenOffice的你将会用到的部分。替换一个'native'发行版构建的完整的原始版本Apache OpenOffice可能会非常棘手，但通常可以成功地完成通过完全移除现有的'native'安装。
