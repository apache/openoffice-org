Apache OpenOffice 3.4.1 正式发布

Apache OpenOffice™ 3.4.1 正式发布，支援更多语言，更快更稳定。

Apache OpenOffice 团队正式发布 Apache OpenOffice 3.4.1 。这是开放免费的文书软件 Apache OpenOffice 的最新版本。维护更新版 3.4.1 版，以全球 228 国下载超过一百二十万次的 Apache OpenOffice 3.4.0 为基础，支援更多语言，相容更多平台，修正错误并强化性能。 OpenOffice 3.4.1 可由 http://www.openoffice.org/download/ 下载，或由 OpenOffice 3.3 、 3.4 版的「说明(H)／检查是否有更新(U)...」选单更新。

Apache OpenOffice （原名 OpenOffice.org ）目前为 Apache 的育成专案。

3.4.1 版详情，可参阅版本说明。其中重点包括：

  • 新增支援高棉文、芬兰文、英式英文、斯洛伐尼亚文及斯洛伐克文，并更新荷兰文、西班牙文、义大利文、匈牙利文。加上原有的美式英文、德文、日文、捷克文、法文、巴西葡萄牙文、俄文、简体中文、正体中文版本、加利西亚文及阿拉伯文，支援达 20 种语言。
  • 支援由 Windows XP 到 Windows 8 ， Linux 32 位元及 64 位元，及苹果 Mac 作业系统，包括最新的山狮 (Mountain Lion) 。本版修正相容性支援，在 Windows 8 上执行更顺畅。
  • 性能、相互操作性及安全性加强。
  • 修正 69 个使用者回报的错误。
  • 社区正在移植到 BSD 、 Solaris 及 OS/2 平台上。详情请参阅 OpenOffice 平台移植计划网页。

Apache OpenOffice 使用者还可享有众多协力厂商团体的周边产品。由 SourceForge 改写重开的扩展组件及模板站上，共有 2388 个模板及 628 个扩展组件。

    未来版本走向，请不吝给我们建议

    除了 Apache OpenOffice 3.4 维护更新版外，我们团队成员也已同步开始 3.5 版的工作。 3.5 版的主要目标包括：提高和 MS Office 的相容性，更快、更稳、更好用，并支援更多语言。 Apache OpenOffice 3.5.0 预计将在 2013 年第一季推出。

    接著将是重大版本 Apache OpenOffice 4.0 版，将包含最近捐给 Apache 的 IBM Lotus Symphony ，其新增及改善的功能。这一版，有机会让我们跳脱原有框架，加上额外的创新设计。

    我们团队已经在著手脑力激荡 Apache OpenOffice 4.0 的各种可能性。我们也邀请使用者、社区成员及有兴趣的各方人士，和我们一起讨论，让 4.0 版再次缔造十年前 1.0 版的伟大成就。我们讨论的平台架设在 Google 好问上，欢迎加入我们，分享您的想法。

    社区志愿者

    Apache OpenOffice 是由社区主导的开源专案。 Apache OpenOffice 团队欢迎志愿者，协助程式撰写、品管、界面设计、翻译／本地化、文件撰写、网站设计及行销推广各方面的工作。若您有兴趣参与，请写信到我们的公开邮件通讯群组 ooo-dev@openoffice.apache.org 自我介绍，并告诉我们您想参与什么工作。
