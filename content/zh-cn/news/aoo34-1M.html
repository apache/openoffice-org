Apache OpenOffice™ 3.4 下载突破百万

上周 Apache OpenOffice 3.4 发表后，感谢使用者和媒体的热情支持，下载已突破一百万次。

2012 年 5 月 16 日 Apache OpenOffice 专案管理委员会最新消息— 在 Apache 软件基金会及 Apache 孵化计划下， Apache OpenOffice 团队今天公布了 Apache OpenOffice™ 3.4 的初步下载数字。

  • Apache OpenOffice 3.4 已由 SourceForge 下载了 1,000,663 次。多国语言包、软件开发套件及原始码下载另计。
  • 下载的使用者来自超过 200 个国家。
  • 下载的版本中有 87% 为 Microsoft Windows ， 11% 为 MacOS ， 2% 为 Linux 。

自上周发表后， Apache OpenOffice 3.4 即成为媒体和社群网路的焦点：

  • 在 12 种以上语言的媒体上，报导了超过 60 个以上的新功能。
  • 在 Twitter 帐号上发表的新闻被转贴了 61 次，读者超过 43,000 人。
  • Facebook 、 Identi.ca 及 Google+ 专页上也非常活跃。

媒体报导

BetaNews 记者 Nick Peers 抢先报导「 Apache 发表了 OpenOffice 的重大更新版—马上升级吧！」并总结新功能如下：

3.4 版的新功能一看就知道—新版最明显的变化有：启动速度变快很多，强化 OLE 物件处理、改进图表绘制及支援可缩放向量图形 (SVG) 。此外还支援了 ODF 1.2 加密，并改以 Apache License 2 引擎发表。

OMG! Ubuntu! 的 Joey Sneddon 则说：「 Apache OpenOffice 3.4 没有全新风貌，而是善加保持原来的优良传统。」

ZDNet 的 Paula Rooney 也作了相关报导。在介绍完 Apache OpenOffice 3.4 新版公告上的「新奇玩具」后， Paula 强调了未来发展的重要性：

…[对] Apache 软件基金会—及其他开放原始码文字软件的死忠支持者—来说，脱离商业公司掌控后的全新开发流程，正是最能吸引使用者及赞助商之处。

The Register 的 Iain Thomspon 强调了 OpenOffice 在图形处理上的重大突破，特别提到了程式设计师 Armin Le Grand 的贡献。

TweakTown 上的 Anthony Garreffa ，在文章中将 AOO 3.4 推为「今日下载」，称 Apache OpenOffice 3.4 是：

…超过 20 年软件工程的结晶。由一个小软件创始到现在，它的稳定性，使其他软件难以望其项背。完全开放的开发流程，每个人都可以回报错误、提出建议，协助改进软件。
