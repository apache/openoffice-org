Lingucomponent Sub-Project: Grammar Checking

One of the goals of the Lingucomponent project is to design, develop, and implement a Grammar checker for English and other supported languages. A grammar checker API has been available in OpenOffice.org since version 3.0. More information about the API is available at the following sources:

  • Grammar Checking page in Wiki
  • Specification (.odt)
  • All grammar checker issues

If you have any interest in helping to implement a grammar checker for the OpenOffice.org project, please subscribe to the mailing list dev@lingucomponent.openoffice.org and introduce yourself, your skills, and your willingness to help this project.
