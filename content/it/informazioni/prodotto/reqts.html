Requisiti di Sistema

Microsoft Windows

  • Windows 98, Windows ME, Windows 2000 (Service Pack 2 o superiore), Windows XP, Windows 2003
  • 128 Mbytes RAM
  • 200 Mbytes di spazio disponibile su disco
  • 800 x 600 o risoluzione superiore con almeno 256 colori

Solaris: SPARC platform edition

  • Solaris 8 OS o superiore
  • 128 Mbytes RAM
  • 250 Mbytes di spazio disponibile su disco
  • X-Server con risoluzione 800 x 600 o superiore con almeno 256 colori

Solaris: x86 platform edition

  • Solaris 8 OS o superiore
  • 128 Mbytes RAM
  • 250 Mbytes di spazio disponibile su disco
  • X-Server con risoluzione 800 x 600 o superiore con almeno 256 colori

GNU/Linux (‘Linux’):

  • Linux kernel versione 2.2.13 o superiore, glibc2 versione 2.2.0 o superiore
  • 128 Mbytes RAM
  • 200 Mbytes di spazio disponibile su disco
  • X-Server con risoluzione 800 x 600 o superiore con almeno 256 colori

Note

  1. Per avere la completa funzionalità di Openoffice.org è richiesta la presenza di un Java Runtime Environment (JRE). Per poter prelevarlo, senza costi aggiuntivi, per il vostro sistema, visitate il sito web Java .
  2. L'ultima versione di OpenOffice.org ufficialmente supportata per Windows 95 era la versione 1.0.3.
