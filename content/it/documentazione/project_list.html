Documentazione presente in altri progetti di OpenOffice.org


Questa sezione elenca i progetti di OpenOffice.org che forniscono documentazione addizionale su funzionalità specifiche. La documentazione è in lingua inglese.

Collegamento

Descrizione

Lingua
Database Access Project Il progetto Database Access è dedicato a facilitare le connessioni tra OpenOffice.org e database esterni. Inglese
API Project Il progetto OpenOffice.org API è basato sulla tecnologia componenti OpenOffice.org. Le API di OpenOffice.org definiscono l'interfaccia di accesso alle funzionalità del programma indipendentemente dal linguaggio di programmazione.  Inglese
Application Framework Project Questo progetto contiene i moduli che definiscono il comportamento generale dei singoli componenti dei programmi di OpenOffice ( Writer, Calc ecc. ). Questo comprende le operazioni comuni, oggetti comuni o funzionalità ed moduli di integrazione con l'ambiente esterno.  Inglese
OpenOffice.org Project Management Tool Project Questo contiene la documentazione per lo sviluppo degli strumenti di gestione del progetto di OpenOffice.org. Inglese
OpenOffice.org Wordprocessing Project Il progetto Word Processing contiene il componente di elaborazione del testo, il componente editor HTML  editor formule, componenti linguistici e filitri testo. Inglese
OpenOffice.org Lingucomponent Project Il progetto Lingucomponent fornisce gli strumenti di correzione ortografica: correttore ortografico, sillabazione e dizionario dei sinonimi.  Inglese
