Documentazione - Guida all'Installazione


Missione

Fornire documentazione comprensibile in diverse lingue per guidare gli utenti attraverso il processo di installazione di OpenOffice.org.

Sommario

Questa area fornisce i file sorgenti (file Writer *.sxw e immagini), i file  PDF, ed i file compressi zip (sorgenti e file pdf) per le Guide di Installazione di OpenOffice.org. La versione inglese della Guida di Installazione è il documento di riferimento. I file sorgenti sono disponibili per le seguenti versioni di OpenOffice.org:

  • OpenOffice.org 2.0 (versione in preparazione)
  • OpenOffice.org 1.1 (versione corrente)
  • OpenOffice.org 1.0x

Guida all'Installazione per OpenOffice.org 2.0
