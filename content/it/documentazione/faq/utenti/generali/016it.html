Quale versione del Java Runtime Environment devo usare?

OpenOffice.org non necessita di un ambiente Java per funzionare (è scritto in C++).

Se si vuole utilizzare Java all'interno di OpenOffice.org serve almeno il jre SUN 1.3.1 , liberamente scaricabile qui.

Altri ambienti java potrebbero funzionare ma non sono testati. La comunità sta lavorando per migliorare l'interoperabilità tra la suite e gli altri ambienti java.

Per installare java dopo l'installazione di OpenOffice.org bisogna eseguire il programma jvmsetup che si trova nella directory program all'interno della directory dove è stato installato OpenOffice.org
(ad es. se OpenOffice.org è stato installato in C:\Programmi\OpenOffice.org1.0\, il programma si troverà in C:\Programmi\OpenOffice.org1.0\program\jvmsetup.exe).
