Legenda ai formati file

  • HTML file visualizzabili nel vostro browser web.
  • SXW file che possono essere aperti ed modificati in OpenOffice.org Writer.
  • ODT file che possono essere aperti ed modificati in OpenOffice.org2.0 Writer.
  • PDF file che sono documenti portabili a sola lettura.
  • ZIP file che sono archivi compressi che devono essere decompressi dopo il download.
