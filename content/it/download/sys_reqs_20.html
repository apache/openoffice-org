Requisiti di Sistema per OpenOffice.org 2.0

Microsoft Windows

  • Windows 98, Windows ME, Windows 2000 (Service Pack 2 o superiore), Windows XP, Windows 2003, Windows Vista  (integrazione con Vista migliorata dalla versione 2.2)
  • 128 Mbytes RAM
  • Almeno 800 Mbyte di spazio disponibile per l'installazione predefinita (includendo una JRE) tramite download. Dopo l'intallazione e la cancellazione dei file di installazione temporanei, OpenOffice.org occuperà approssimativamente 440 Mbyte di spazio su disco.
  • 800 x 600 o risoluzione superiore con almeno 256 colori

Solaris: SPARC platform edition

  • Solaris 8 OS o più recente
  • 128 Mbytes RAM
  • 250 Mbytes di spazio disponibile su disco
  • X-Server con 800 x 600 o risoluzione superiore con almeno 256 colori

Solaris: x86 platform edition

  • Solaris 8 OS o più recente
  • 128 Mbytes RAM
  • 250 Mbytes di spazio disponibile su disco
  • X-Server with 800 x 600 o superiore con almeno 256 colori

Linux:

  • Linux kernel versione 2.2.13 or superiore, glibc2 versione 2.2.0 o superiore
  • 128 Mbytes RAM
  • 200 Mbytes di spazio disponibile su disco
  • X-Server con 800 x 600 o risoluzione superiore con almeno 256 colori

Java

La versione minima di JDK/JRE richiesta per usare le funzionalità OpenOffice.org che richiedono java è JDK/JRE versione 1.3.1.

Per usare i filtri XSLT con JDK 1.3.1 dovreste prelevare i file xalan.jar e xml-apis.jar dal sito web Xalan e crimson.jar dal sito web Crimson. I file devono essere aggiunte al classpath di OpenOffice.org, questo può essere fatto con Strumenti->Opzioni->OpenOffice.org->Sicurezza

Per la piena funzionalità , jdk/jre 1.4.0_02 o più recente o jdk/jre 1.4.1_01 o più recente è richiesto

Per accessibilità:

Tutte le piattaforme:

Java runtime environment 1.4.0_02 / 1.4.1_01 o più recente

Java Access Bridge v1.0.3 - per prelevare Java Access Bridge, visitate www.sun.com/star/accessibility

Linux:

Installazione di GNOME 2.2

Solaris:

Installazione di GNOME 2.0

Vedete anche Indice dei Requisiti di Sistema per ogni versione
