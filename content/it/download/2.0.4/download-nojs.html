Introduzione

La più recente versione di OpenOffice.org è disponibile qui. 
  • I file disponibili qui sono per Windows, Linux, Solaris. Per le altre piattaforme, andate sui siti collegati: LinuxPPC, Mac OS X (X11), FreeBSD.

Scegliete il sito mirror più vicino a voi. Il collegamento va direttamente nella cartella di riferimento. Non tutti i mirror hanno confermato la disponibilità di tutti i file. 


Struttura

Tutti i mirror, indistintamente, devono seguire la seguente struttura per i file di OpenOffice.org. 

Struttura cartelle server
Cartella Descrizione
/stable/[numero versione]/[nome file] Versioni finali inglese
/developer/[numero versione sviluppo]/[nome file] Versioni in sviluppo inglese
/localized/[lingua]/[numero versione]/[nome file] Versioni finali localizzate, compreso la versione italiana
/contrib/[nome cartella]/[nome file] Versioni in sviluppo localizzate, compreso la versione italiana
/contrib/iso/[lingua] Immagini ISO per CD disponibile in inglese e tedesco
/versioni-precedenti (per archivio) Versioni precedenti

Dimensioni versione 1.1.5 : Windows: 64 MB | Linux: 76 MB | Solaris (S): 87 MB | Solaris (I): 78 MB
