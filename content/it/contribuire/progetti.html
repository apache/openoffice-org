Progetti & Moduli

I programmi accessibili dall'utente:

dba/dbaccess L'applicazione database (Base)
graphics/sd Il programma di presentazione e disegno (Draw / Impress)
graphics/sch Lo strumento per diagrammi (Chart)
installation/setup2 Il programma d'installazione
installation/wizard L'Autopilota (composizioni guidate)
sc/sc Il foglio di calcolo (Calc)
sc/scaddins Strumenti addizionali per il foglio di calcolo (Calc)
script/basctl L'Editor(IDE) di StarBasic
sw/sw L'elaboratore di testi (Writer)
sw/smath L'editor di formule matematiche (Math)
whiteboard/Groupware La re-implementazione delle applicazioni di posta, news e agenda

Moduli per i quali non sono necessarie conoscenze di programmazione:

installation/extras Questo progetto è relativo alle presentazioni, la Gallery,...
installation/helpcontent Il testo e il sistema dell'aiuto in linea
installation/readlicense I dati per i Readme
de la versione tedesca di OOo
fr la versione francese di OOo
it la versione italiana di OOo
ui Qui vengono discussi i cambiamenti all'interfaccia grafica della Suite
website Questo Progetto si occupa del sito web di OOo
documentation Produce la Documentazione di OOo
marketing Comunicazione, Informazione, Marchio, Collocazione sul Mercato, Contatti con la Stampa,...
whiteboard/mirrorsOffre alternative per il download di OOo

Moduli importanti per il Programmatore:

api/udkapi Contiene la parte di API che appartiene al kit di sviluppo UNO
api/offapi A questa sezione appartiene la parte di API di ogni singolo componente di OpenOffice.org
api/drafts Contiene nuovi elementi che non sono stati ancora completamente implementati
api/offuh Genera gli UNO-Header per C++ dalle descrizioni IDL di API
api/unoil Genera le Classi Java dalle descrizioni IDL di API
dba/connectivity Contiene le interfacce per database esterni
dba/dbaccess L'applicazione database (Base)
external/aspell Il correttore ortografico Open Source (più intelligente di ISPELL)
external/Berkeleydb Il database Berkley
external/freetype Libreria piattaforma-indipendente per la rappresentazione dei caratteri(CJK)
external/MathMLDTD Libreria per l'editor di formule
external/moz Librerie e header di Mozilla
external/pspell Interfaccia portabile per il correttore ortografico
external/sablot Sablotron Transformations Processor
external/stlport Libreria standard ANSI C++ multi-piattaforma
framework/sfx2 Il kernel del framework di OOo
framework/offmgr Elementi utilizzati globalmente (esempio:le finestre di dialogo)
framework/res Le risorse (Bitmap, Icone,...)
framework/idl Descrizione dell'interfaccia dei componenti del Framework
framework/framework Integra complessivamente i vari componenti
framework/desktop Contiene il codice che viene richiamato alla partenza
framework/DocumentPropertiesRende disponibile le Proprietà dei Documenti
graphics/svx Contiene i componenti grafici comuni
graphics/goodies Contiene alcuni filtri e funzioni grafiche di supporto
gsl/psprint Il Generatore di Codice PostScript
gsl/padmin Il Sistema d'Amministrazione della Stampa su Unix
gsl/vcl Visual Class Library, si occupa del controllo del sistema a finestre
gsl/rsc Il Compiler delle Risorse
gsl/toolkit Implementazione dei controlli UNO basati su VCL
gsl/UnoControls Implementazione dei controlli UNO con com.sun.star.awt
gsl/forms Implementazione dei formulari
installation/scp Script d'Installazione
installation/scptools Genera gli script d'installazione
installation/instsetoo Questo modulo assiste nella produzione delle frasi d'installazione per OpenOffice.org
l10n/I18npool Framework d'internazionalizzazione con supporto per CTL e CJK
l10n/transex Strumenti di localizzazione
oi/so3 Compound document model
oi/sj2 Supporto per le Java-Applets in OpenOffice
oi/ie Integra Internet Explorer in OOo (WIN32)
porting/sal Adattamento per i differenti Sistemi Operativi
script/basic L'interprete Basic e la libreria Basic di Run-time
sw/linguistic L'interfaccia UNO per il Correttore Ortografico
sw/lingu Le basi per un Correttore Ortografico portabile
tools/solenv Ambiente Solar
tools/dmake Programma Make per OOo
tools/rscpp Resource Compiler Preprocessor
tools/xml2cmp Processore per la descrizione dei componenti UNO
tools/jtools Programmi Java, che sono necessari durante la fase di compilazione
tools/boot Bootstrapping degli strumenti di compilazione
tools/config_office Configurazione dell'ambiente di compilazione
tools/soltools Strumenti dell'ambiente Solar
ucb/ucb Universal Content Broker
ucb/ucbhelper Classi d'aiuto C++ per l'Universal Content Broker
ucb/uui GUI per l'Universal Content Broker
ucb/inet Protocolli Internet (FTP, HTTP, IMAP, NNTP, POP3, SMTP, MIME)
ucb/ldapber Supporto LDAP
ucb/store Contenitore per file, i form e i data-stream
udk/cppu Libreria di Run-time UNO per tutti i linguaggi tranne JAVA
udk/idlc Il Compiler IDL (Interface Description Language)
udk/cppuhelper Aiuto d'implementazione per C++ UNO
udk/javaunohelper Aiuto d'implementazione per Java UNO
udk/jurt Libreria di Run-time UNO per JAVA
udk/bridges Bridges verso altri linguaggi di programmazione
udk/remotebridges Servizi UNO per Bridges tra Processi
udk/stoc Servizio Base UNO
udk/cpputools raccolta di Strumenti UNO
udk/registry Implementazione del registro
udk/codemaker Genera i dati degli header C++ e COBRA IDL
udk/rdbmaker Genera le descrizioni di Tipo binarie
udk/sandbox Manager di sicurezza JAVA
udk/jut Strumenti JAVA UNO
udk/product UDK-Makefiles e programma d'esempio UNO
ui Progetto per l'interfaccia utente di OOo
ui/Accessibility Componenti per l'accessibilità al programma da parte di individui disabili
util/tools Contiene le classi di base come String, Date, Time, Streams
util/svtools Strumenti che si basano su VCL(Visual Class Library)
util/std2 Contiene il porting STL di SGI/STL
util/io Contiene le funzioni di base per lo Streaming e la comunicazione tra processi
util/eventattacher Componenti di base per la gestione degli eventi
until/unzip Libreria Infozip
util/unotools Strumenti d'aiuto per le API basate su UNO
util/extensions Componenti UNO per OLE, PGP, ...
util/external Base per Makefiles e Componenti esterni
util/configmgr Componenti UNO per la configurazione
util/officecfg Struttura della configurazione per Applicazioni e Componenti
util/sot Implementazione della struttura dati di Microsoft
util/automation Base per i test del Framework e dei test automatizzati
whiteboard/bonobo Mette a disposizione i componenti UNO per GNOME e viceversa
whiteboard/Groupware La re-implementazione delle applicazioni di posta, news e agenda
whiteboard/Inspector Strumenti di sviluppo per mostrare le informazioni sugli oggetti durante il Run-time
whiteboard/lingucomponent Implementazione del correttore ortografico di OpenOffice.org
whiteboard/OOBean mette a disposizione i moduli OpenOffice.org come Java-Bean
xml/sax Parser XML e accesso alla scrittura XML
xml/xmloff Implementazione del formato dati OO-XML
xml/package genera i dati XML compressi (ZIP)
