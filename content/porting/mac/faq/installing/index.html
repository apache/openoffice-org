Installing OpenOffice.org for Mac OS X (with X11 windowing System and Aqua)

There are a number of steps involved with installing OpenOffice.org on the Mac.

  • Install the X11 Windowing System (not required for Aqua)
  • Download OpenOffice.org for Mac OS X, if you haven't already
  • Install the OpenOffice.org Application
