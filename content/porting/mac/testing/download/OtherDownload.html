Download OpenOffice.org (X11) for Mac OS X

Currently we provide separate builds for PPC and Intel based Macs in some languages. There is no Universal Binary available at this time. We have not yet released a preview build of our latest Aqua development

You have selected to download a version of OpenOffice that is not an official release.

These builds are still the official Mac builds, however many are development builds, or have not been tested yet.

If you have any problems with these builds, please contact us by sending a mail to mac@porting.openoffice.org. It is also recommended that you subscribe to our mailing list so that you get all the responses from other members of the list. You can subscribe by sending a mail to mac-subscribe@porting.openoffice.org. You can also look at the mailing list archives.

Go to the Development build FTP server

It is recommended that you use a Mozilla based web browser, or a dedicated FTP client, such as Cyberduck for downloading, as there can be problems with Safari and the Finder, especially on Mac OS X 10.3.
