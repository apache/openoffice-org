OpenOffice.org PPC Linux Download Sites

Last updated 2004-08

 

PPC Linux OpenOffice.org 1.1.2 is able to run on any PPC Linux system with glibc 2.3.1 or higher, XFree86 4.0 or higher, etc.

Note. The home for PPC Linux OpenOffice.org releases is at:

  • ftp://ftp.yellowdoglinux.com/pub/yellowdog/software/openoffice/

Please support our PPC Linux distributions any way you can! They are helping to bring great new software to you!

Thanks,

Kevin Hendricks

khendricks at openoffice.org

PPC Linux Links
