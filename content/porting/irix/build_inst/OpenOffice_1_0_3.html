Building OpenOffice.org 1.0.3

Getting The OpenOffice 1.0.3 Source

Follow the instructions on the GENERAL Build Instructions page for obtaining the source from CVS.

Use the following Tag to obtain the source: 

  
OpenOffice_1_0_3
OR alternatively, you can download the tarball for the OpenOffice.org 1.0.3 source from this page

Further Pre-Build Instrictions

There is a readme file missing from the installation set in: 
  $SRC_ROOT/readlicense/source/readme
Simply change to this directory and copy one of the other directories, to the IRIX specific directory needed: 
  % cp -r unxlngi4 unxirgm
