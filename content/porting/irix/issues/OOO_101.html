OpenOffice.org 1.0.1 Issues

Missing files

Up until April 2003, all developments to get OO running on IRIX were done in this branch for the following reasons:
  • We had a complete list of patches to build this tag
  • We had both a debug and non debug build of this tag
  • We didnt want to hit a moving target (eg: OOO_STABLE_1)
Basically any problems with this build were ironed out, so it made sense to continue working off it, besides the fact that it needed to be heavily patched. Any problems we encountered along the way, we raised issues on, and managed to get the majority of them checked into 1.0.3.

As we were concentrating on finding our showstopper bug, we weren't very fussy about small details like files missing from certain modules, and incorrect versions of files in other places.

Since we found out showstopper bug, the majority of our efforts in porting and debugging were complete, so we finally made the switch to 1.0.3, as it is a much cleaner build process.
