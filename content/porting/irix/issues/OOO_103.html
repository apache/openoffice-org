OpenOffice.org 1.0.3 Issues

This section will be used to list all major issues that impose problems in building or running the OpenOffice.org 1.0.3 release on IRIX.

Missing README file

The README file for IRIX never made it into the 1.0.3 release (along with 2 dozen or so patches), but hopefully will be in for the 1.0.4 release. This is not a major issue, but rather an annoyance, as it will disturb the installation process.

Patches that never got checked in...

There are still about 2 dozen patches that need to be applied to 1.0.3 to configure, build and run it. Unfortunately these will never be a part of 1.0.3, but we hope to get them checked into 1.0.4 (well the more important ones anyhow). Once this happens, we should see a more seemless build process on IRIX, and hopefully more IRIX users will start building OpenOffice.org on IRIX!
