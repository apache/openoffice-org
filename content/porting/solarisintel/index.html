Introduction

OpenOffice.org is available for Solaris x86. Version 1.1.1 has been built and sucessfully tested.
If you have difficulties installing or using the product, please join the users mailing list to get a helping hand.

Since OpenOffice.org 1.1.2 Solaris x86 is one of the standard platforms available from Download Central.
