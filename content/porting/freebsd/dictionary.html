  • ports/editors/ooodict-all:dictionary for OpenOffice.org in ports tree
    Even though DicoOOo installes dictionaries, we love ports, and of course we have ports/editors/ooodict-all.you can install by (cvsup ports tree is mandatory or highly recommended): 
    # cd /usr/ports/editors/ooodict-all
# make install
    If you want localized version: 
    # cd /usr/ports/editors/ooodict-all
# make install LOCALIZED_LANG=abc
    and if you are using 2.0 or 2.0-devel 
    # cd /usr/ports/editors/ooodict-all
# make install LOCALIZED_LANG=abc INSTALLATION_BASEDIR=openoffice.org2.0.0
    you can check INSTALLATION_BASEDIR by 
    # cd /usr/ports/editors/openoffice.org-2.0
# make -V INSTALLATION_BASEDIR
openoffice.org2.0.0
