Tools required


gcc -3.2.X or higher
Blackdown JDK 1.3.1-02d - augmented with crimson.jar, xalan.jar, and xml-apis.jar from Apache.org
Ant -1.5 available from Apache.org

Build Notes


For a complete build from source you must add in a specially built set of Mozilla Runtime components as a special zip archive. They are available prebuilt from the maintainer for gcc 3.2.X and later. Please see the "source additions" archive on the download sites for complete build pieces and instructions.
