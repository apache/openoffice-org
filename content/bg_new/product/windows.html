Apache OpenOffice for Windows

Поддръжани версии на Windows

Настоящият Apache OpenOffice поддържа Microsoft Windows XP, Vista, Windows 7 и Windows 8.

Специфични бележки по отношение на платформите:

  • Windows 2000: OpenOffice може да работи на Windows 2000, но ние не сме тествали или не препоръчваме.
  • Windows XP: Тестван, стабилен, не са известни специфични проблеми.
  • Windows Vista: Тестван, стабилен, не са известни специфични проблеми.
  • Windows 7: тестван, стабилен.
  • Windows 8: стартирахме като “десктоп приложение”, не като Metro “Апликация”. Изисква x86 или 64-bit Windows. Не поддържаме Windows RT, ARM-базирани версии, предназначени за таблети.

Моля обърнете внимание: Apache OpenOffice се разпространява като 32-bit приложение. Ще върви успешно на 32-bit режим на 64-bit версия на Windows 7 и Windows 8. Но в тези случай, за допълнителна функционалност се изисква и 32-bit JVM (Java). За повече детайли тук.

Хардуерни изисквания

Хардуерните исизквания са доста скроми и дори по стари машини би трябвало да могат да стартират OpenOffice:

  • Памет: Минимум 256 Mbytes RAM (512 MB RAM препоръчително)
  • Дисково пространство: Поне 650 Mbytes налично дисково пространство за инсталация по подразниране през изтегляне. След инсталирането и изтриването на временните файлове, Apache OpenOffice ще използва приблизително 440 Mbytes дисково пространство
  • Графика: 1024 x 768 или по – висока разделителна способност с поне 256 цвята

Специфични за Windows Характеристики

Apache OpenOffice има следните специални характеристики в режим на работа под Windows:

  • интегриране на мейл – клиент през Интерфейс приложение за съобщения (MAPI)
