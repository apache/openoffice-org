Master Server

The following servers are available as masters for downstream mirrors. As an end user please use our Download Webpage. If you are providing OpenOffice.org downloads from your server, please read more about our Mirrors Project and let us know by sending a message to Mirrors mailing list.

Tier-1 / Category 1 / main set

The Tier-1 / Category 1 servers mentioned below carry the main data set and are accessible without prior registration. Access to the extended data set is described at the bottom of this page.

Server for extended dataset

The following server can be used to mirror the extended dataset.

rsync.services.openoffice.org

  • rsync://rsync.services.openoffice.org/openoffice-extended/
  • http://rsync.services.openoffice.org (HTTP access is not guaranteed to work. If it sucks too much bandwith we will shut it down in favor of downstream servers rsyncing the data.)
