Apache OpenOffice para Windows

Suporte para versões Windows

A versão actual do Apache OpenOffice suporta Microsoft Windows XP, Vista, Windows 7 e Windows 8.

Notas específicas das plataformas:

  • Windows 2000: o OpenOffice pode funcionar em Windows 2000, mas nós não o testámos e não recomendamos a utilização;
  • Windows XP: testado, estável, não há problemas conhecidos nesta plataforma;
  • Windows Vista: testado, estável, não há problemas conhecidos nesta plataforma;
  • Windows 7: testado, estável;
  • Windows 8: executado como “aplicação do desktop”, não uma “App” Metro . Requer Windows x86 ou 64-bit. Não suportamos Windows RT, a versão baseada em ARM, destinada a tablets.

Por favor note: o Apache OpenOffice é distribuído como uma aplicação 32-bit. Funciona normalmente em modo 32-bit nas versões 64-bit do Windows 7 e Windows 8. Mas nesses casos, é necessário um JVM de 32-bit para alguma funcionalidade. Veja os detalhes aqui.

Requisitos do equipamento

As exigências de hardware são modestas e mesmo máquinas antigas devem ser capazes de executar o OpenOffice:

  • Memória: Mínimo 256 Mbytes RAM (512 MB RAM recomendado).
  • Armazenamento: pelo menos 650 Mbytes disponíveis em disco para uma instalação típica por download. Após instalação e eliminação de ficheiros temporários, o Apache OpenOffice ocupa aproximadamente 440 Mbytes de espaço em disco.
  • Placa gráfica: resolução de 1024 x 768 ou superior com pelo menos 256 cores.

Funcionalidades específicas do Windows

O Apache OpenOffice tem as seguintes funcionalidades específicas no Windows:

  • integração de clientes de correio electrónico por Messaging Application Programming Interface (MAPI)
