Apache OpenOffice Descrição do produto

Compatível com outros importantes conjuntos de produtividade, o Apache OpenOffice é grátis para transferir, usar e distribuir. Transfira já e obtenha:

  • Writer um processador de texto que pode usar para escrever uma carta rapidamente ou para escrever um livro.
  • Calc uma poderosa folha de cálculo com todas as ferramentas necessárias para calcular, analisar e mostrar os seus dados em relatórios numéricos ou gráficos fabulosos.
  • Impress a maneira mais rápida, mais poderosa para criar apresentações multimédia.
  • Draw permite-lhe produzir tudo, desde simples diagramas a ilustrações 3D dinâmicas.
  • Math permite-lhe criar equações matemáticas com um ambiente gráfico de utilização ou digitando as suas fórmulas no editor de equações.
  • Base permite-lhe manipular bases de dados facilmente. Crie e modifique tabelas, formulários, pesquisas e relatórios, tudo dentro do Apache OpenOffice.

Porquê o Apache OpenOffice?

Apache OpenOffice é sinónimo de qualidade:

  • As raízes do Apache OpenOffice têm vinte anos, criando um produto bem amadurecido e poderoso
  • Tem muitos milhões de utilizadores
  • Críticos independentes à volta do mundo recomendam o produto
  • Com um processo de desenvolvimento completamente aberto, o Apache OpenOffice não tem nada a esconder - o produto mantém-se ou cai com a sua reputação

O Apache OpenOffice é fácil de usar:

  • O programa parece familiar e é utilizável instantaneamente por qualquer pessoa que tenha usado outro produto deste tipo
  • É fácil mudar para o Apache OpenOffice - o programa lê a maioria dos ficheiros da concorrência
  • Poucas barreiras linguísticas - se ainda não está disponível no seu idioma, as hipóteses são de que brevemente estará
  • O Apache OpenOffice é suportado por uma comunidade global de voluntários amigáveis, felizes por ajudar tanto os recém-chegados como os utilizadores avançados

O Apache OpenOffice é um programa gratuito:

  • Pode transferir o Apache OpenOffice completamente grátis e livre de taxas
  • Instale-o em tantos computadores quantos queira
  • Use-o para qualquer propósito - privado, educacional, Estado e administração pública, comercial…
  • Distribua cópias grátis à família, amigos, estudantes, empregados, etc.

O Apache OpenOffice cumpre com os padrões:

  • O primeiro programa no mundo a usar o formato OASIS OpenDocument Format (ISO/IEC 26300) como seu formato nativo de ficheiros.
