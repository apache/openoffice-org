Apache OpenOffice para novos computadores

Poupar dinheiro no seu novo computador

Está a pensar comprar um computador novo? Ou a compra de um para outra pessoa? Se assim for, aqui vai uma sugestão de poupança de dinheiro que o vendedor de computadores pode não lhe dar.

Não é obrigado a gastar dinheiro numa cópia do Microsoft Office. Pode comprar um computador sem o Microsoft Office instalado e poupar esse dinheiro. Então, quando for para casa, experimente o [Apache OpenOffice (AOO)] 1 gratuitamente. Dispõe de um processador de texto flexível, uma poderosa folha de cálculo, gráficos dinâmicos, acesso a bases de dados e muito mais. Pode ler arquivos do Microsoft Office. Pode exportar para PDF. Funciona desde o Windows XP até ao Windows 8, em Mac OS e Linux. E é totalmente gratuito!

Como é que podemos fazer isto de graça? Somos uma organização sem fins lucrativos com a missão caritativa de publicar programas grátis para o público.

É isto que nós fazemos. Nós queremos que tenha programas grátis. Portanto, quando o vendedor tentar carregar a sua factura do computador com todos os “extras”, basta dizer “Obrigado, mas Não. Vou instalar o AOO”.
