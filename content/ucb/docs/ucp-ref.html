Abstract

This document is a Reference Guide to the Content Providers (UCPs) currently available for the Universal Content Broker (UCB).

Note: There are some other UCPs available. Documentation may follow soon.

File Content Provider

The File Content Provider (FCP) provides access to the file system of the machine the Provider is running on.

Available documents:

  • Reference Guide

  • Specification for the UNO service com.sun.star.ucb.FileContentProvider

  • Specification for the UNO service com.sun.star.ucb.FileContent

FTP Content Provider

The FTP Content Provider provides access to the contents (folders and documents) made available by FTP servers.

Available documents:

  • Reference Guide

  • Specification for the UNO service com.sun.star.ucb.FTPContentProvider

  • Specification for the UNO service com.sun.star.ucb.FTPContent

WebDAV Content Provider

The WebDAV Content Provider (DCP) provides access to WebDAV and standard HTTP servers.

Available documents:

  • Reference Guide

  • Specification for the UNO service com.sun.star.ucb.WebDAVContentProvider

  • Specification for the UNO service com.sun.star.ucb.WebDAVDocumentContent

  • Specification for the UNO service com.sun.star.ucb.WebDAVFolderContent

Hierarchy Content Provider

The Hierarchy Content Provider ( HCP ) provides access to a virtual, persistent, customizable hierarchy of contents.

Available documents:

  • Specification for the UNO service com.sun.star.ucb.HierarchyRootFolderContent
