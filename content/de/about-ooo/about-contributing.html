Code Beisteuern

Wie man eigenen Code beisteuert:

Alle Vorschläge für Änderungen des Quellcodes werden via IssueZilla eingereicht, dazu wählt man unter Issue-Type die Option "PATCH" aus. Der Code wird dann automatisch dem für dieses Projekt zuständigen Committer (jemand, der für diesen Bereich des Quellcodes Schreibrechte hat) zugestellt.

  1. Bitte füllen Sie das Copyright-Formular aus: PDF
    und faxen oder schicken es an die angegebene Nummer/Adresse.

  2. Damit Ihr Code übernommen werden kann, müssen zwei Bedingungen erfüllt sein:
    1. Für jeden Beitrag mit mehr als 10 Zeilen muss Ihre Copyright-Erklärung vorliegen. Sie können hier nachsehen, ob das der Fall ist.

    2. Der für dieses Projekt Verantwortliche muss der vorgeschlagenen Änderung zustimmen.

Wir freuen uns auf Ihre Beiträge!
