Benötigen Sie Hilfe oder haben Sie ein Problem?

Fragen zur Anwendung des Programms

Sie haben eine Frage oder wissen z. B. nicht, wie Sie eine Funktion in einer Komponente von OpenOffice aufrufen können? Oder Sie haben ein Verhalten festgestellt, das Sie sich nicht erklären können?

Antworten finden Sie möglicherweise in den Handbüchern, die wir im Dokumentationsportal für Sie bereithalten. Eine Reihe häufig auftretender Fragen und Probleme haben wir außerdem in den entsprechenden FAQs beantwortet. Eine weitere informative Quelle ist das OpenOffice.org-Wiki, auch hier finden Sie eine Fülle von Material.

Ließ sich Ihr Problem mit diesen Hilfen nicht lösen, dann stellen Sie Ihre Frage am besten auf der Anwender-Mailingliste. Auf dieser Liste helfen Anwender anderen Anwendern auf freiwilliger Basis, denn OpenOffice ist keine kommerzielle Software und bietet damit keinen eigenen kommerziellen Support.

Sie sollten sich anmelden, bevor Sie Ihre Frage stellen, damit Sie auch über alle eingehenden Antworten informiert werden:

  • Senden Sie bitte eine E-Mail an users-de-subscribe@openoffice.apache.org.
  • Sie erhalten daraufhin eine Bestätigungsmail (auf Englisch), die Sie einfach mit der Antwort-Funktion Ihres E-Mail-Programms beantworten, um die Anmeldung abzuschließen (sog. Double Opt-In).

Fragen können Sie nun an die Mitglieder der Mailingliste richten, indem Sie eine E-Mail mit einem möglichst aussagekräftigen Betreff an users-de@openoffice.apache.org senden.

Um sich wieder von der Mailingliste abzumelden, gehen Sie bitte wie folgt vor:

  • Senden Sie eine Mail an users-de-unsubscribe@openoffice.apache.org (mit der selben E-Mail-Adresse, die Sie für die Anmeldung verwendet hatten).
  • Sie erhalten daraufhin eine Bestätigungsmail (auf Englisch), die Sie einfach mit der Antwort-Funktion Ihres E-Mail-Programms beantworten, um die Abmeldung abzuschließen (sog. Double Opt-Out).

Weitere Hinweise zur Anwender-Mailingliste finden Sie auf der Seite Hilfe per Mail.

Wichtige Hinweise für die Benutzung von Mailinglisten auf OpenOffice.org:
Sie erhalten nach der Anmeldung alle an die Liste geschickten Beiträge aller Benutzer. Wenn Sie diese Mitteilungen nicht mehr erhalten möchten, melden Sie sich bitte selbst wieder von der Liste ab (s.o.).
Die Mailinglisten sind öffentlich. Eine an die Liste gerichtete E-Mail wird automatisch im Internet veröffentlicht und archiviert, ohne dass das Projekt dies steuern oder beeinflussen könnte. Wenn Sie eine Nachricht an die Liste senden, stimmen Sie damit deren Veröffentlichung zu! Vermeiden Sie deshalb bitte die Angabe persönlicher Daten wie Telefonnummer oder Adresse im Text bzw. in der Signatur der Nachricht, wenn Sie nicht wollen, dass diese Angaben mitveröffentlicht werden und damit auch von Internetsuchmaschinen gefunden werden können.

Fragen zur Entwicklung und zur Mitarbeit

Sie haben eine Frage, die nicht die Anwendung des Programms betrifft, sondern eher in den Bereich Entwicklung zielt? Oder Sie sind an einer aktiven Mitarbeit interessiert? Dann melden Sie sich bitte auf der Entwickler-Mailingliste; über diese Liste werden die Tätigkeiten des deutschsprachigen Projektes von OpenOffice koordiniert.

  • Melden Sie sich an, indem Sie eine E-Mail an dev-de-subscribe@openoffice.apache.org senden und die Bestätigungsmail beantworten (Double Opt-In).
  • Senden Sie Ihre Fragen oder Anregungen an dev-de@openoffice.apache.org.
  • Zum Abmelden senden Sie eine Mail an dev-de-unsubscribe@openoffice.apache.org und beantworten Sie die Bestätigungsmail.
