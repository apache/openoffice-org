Java

OpenOffice.org funktioniert auch ohne Java. Um aber alle Funktionalitäten z. B. im Bereich Datenbanken nutzen zu können, wird Java benötigt. Bei der OpenOffice.org-Installation wird automatisch nach einem installierten Java gesucht. Die aktuelle Java-Version können Sie kostenlos von den Seiten der Firma Sun laden. http://java.sun.com/j2se/1.5.0/download.jsp oder http://java.sun.com/j2se/1.4.2/download.html
Für OpenOffice.org 1.1 wird Java 1.4.1_02 oder neuer benötigt.
