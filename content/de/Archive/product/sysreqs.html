Systemanforderungen für OpenOffice.org 3.x

Microsoft Windows

  • Windows 2000 (Service Pack 2 oder höher), Windows XP, Windows 2003, Windows Vista, Windows 7
  • 256 MByte RAM (512 MB empfohlen)
  • 650 MByte freier Speicherplatz auf der Festplatte für die Installation als Download (einschl. JRE). Nach der Installation und Entfernen der temporären Dateien benötitgt OpenOffice.org ca. 440 MB Speicherplatz auf der Festplatte
  • Bildschirmauflösung 1024 x 768 oder höher mit mindestens 256 Farben

Solaris: SPARC oder X86

  • Solaris 10 OS oder höher
  • 256 MByte RAM
  • 450 MByte freier Speicherplatz auf der Festplatte
  • X-Server mit Auflösung 1024 x 768 oder höher und mindestens 256 Farben

GNU/Linux ("Linux")

  • Linux Kernel Version 2.4 oder neuer, glibc2 Version 2.3.2 oder höher
  • 256 MByte RAM (512 MB empfohlen)
  • 400 MByte freier Speicherplatz auf der Festplatte
  • X-Server mit Auflösung 1024 x 768 oder höher und mindestens 256 Farben

Mac OS X

  • Intel Prozessor oder PowerPC G4
  • Mac OS X 10.4 (Tiger) oder höher
  • 512Mbyte RAM
  • 400 Mbyte freier Speicherplatz auf der Festplatte
  • Bildschirmauflösung 1024 x 768 oder höher mit 16,7 Millionen Farben

Java

Die minimale JDK/JRE-Version für die Verwendung von OpenOffice.org-Features, die Java erforden, ist JDK/JRE Version 1.5.x.

Zugänglichkeit

Alle Plattformen

  • Java Laufzeitumgebung 1.4.0_02 / 1.4.1_01 oder neuer
  • Java Access Bridge v1.0.3 - erhältlich unter www.sun.com/star/accessibility

Linux:

  • Installation von GNOME 2.6

Solaris:

  • Installation von GNOME 2.6
