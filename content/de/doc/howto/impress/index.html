OpenOffice.org Präsentation

Diese Anleitung soll Ihnen anhand eines kleinen Beispiels einen ersten Eindruck der Fähigkeiten von Impress vermitteln. Vorausgesetzt wird, dass Sie bereits Erfahrungen im Umgang mit Ihrer Systemumgebung (z. B. Windows, Linux mit grafischer Benutzeroberfläche) gesammelt haben.
Für die Arbeit mit diesem Dokument sollten Sie etwa eine Stunde einkalkulieren. Sie werden im Anschluss daran in der Lage sein, mit OpenOffice.org Impress eine einfache Präsentation zu erstellen.
