Ich zeichne einen Banner für eine Web-Seite. Wie kann ich das in einer 'web-sicheren' Farbpalette tun?

Sie können die Farbpalette von OpenOffice.org in eine Palette, in der die Farben als hexadezimale Werte oder web Farbnamen gespeichert sind, ändern, wenn Sie dieser Prozedur folgen:

  1. Zeichnen Sie ein Objekt auf der Dokumentenfläche.
  2. Klicken Sie auf das Objekt um es auszuwählen.
  3. Klicken Sie mit der rechten Maustaste darauf, um das Kontextmenü zu erhalten.
  4. Wählen Sie den Eintrag 'Fläche...' aus.
  5. In dem Popup-Fenster, das erscheinen wird, wechseln Sie in die Registrierkarte 'Farben'.
  6. Klicken Sie auf das 'Laden der Farbtabelle'-Icon.
  7. Wählen Sie die Datei html.soc aus und drücken Sie die 'OK'-Schaltfläche. (standardmäßiger Ordner: OpenOffice60/user/config/)
