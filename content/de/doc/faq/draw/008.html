Wie kann ich eine OpenOffice.org Zeichnung in einem Standard-Bildformat (wie zum Beispiel BMP, JPG,...) speichern?

Das von dem Zeichen-Programm verwendete .sxd Format ist kein Grafik-Format; es ist ein Dokumenten-Format.

Um eine Zeichnung in einer Bilddatei zu speichern machen Sie Folgendes:

  1. Wählen Sie Datei-Exportieren... aus der Menüleiste aus.
  2. Wählen Sie das gewünschte Graphik-Format aus der Auswahlliste 'Dateityp' aus.
  3. Drücken Sie die 'Speichern'-Schaltfläche
