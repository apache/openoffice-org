Ich habe ein Bild (JPEG-Format), bei dem ich die Farben ändern muss, bevor ich es in einem OpenOffice.org Dokument verwende. Wie verändere ich die Farben?

Wenn Sie ein JPEG-Bild öffnen, zeigt OpenOffice.org die Symbolleiste für Bilder gleich unter der Symbolleiste für Befehle an. Wenn das Bild geöffnet ist:

  1. Wähle: Extras-Pipette aus dem Menü. Es zeigt das 'Pipetten'-Fenster an.
  2. Klicken Sie auf das ganz links oben befindliche Icon: [Pipette].
  3. Geben Sie einen Toleranzwert ein, wie genau die Farbauswahl sein soll. Beginnen Sie mit einer Toleranz von 10%. Eventuell müssen Sie diesen Wert erhöhen, damit Sie eine Änderung sehen. Ein Wert um die 30% ist gewöhnlicherweise ausreichend. Sie können ihn bis zu 99% erhöhen, aber das könnte eine weit größere Fläche verändern, als Sie möchten.
  4. Wählen Sie im Bild die Farbe aus, die ersetzt werden soll. Im Pipettenfenster wird diese Farbe automatisch sichtbar.
  5. Wählen Sie bei der 'Ersetzen durch...'-Auswahlliste die Farbe aus, mit der ersetzt werden soll.
  6. Klicken Sie auf die 'Ersetzen'-Schaltfläche um Ihre Änderung zu bestätigen. Wenn nichts passiert, erhöhen Sie die Toleranz.
  7. Um mehr Farben zu ersetzen, wählen Sie eine andere Pipetten-Fläche auf der linken Seite des Pipetten-Dialogfensters aus, dann wiederholen Sie die Schritte von 2 bis 5. Abschließend bestätigen Sie Ihre Änderung durch Drücken der 'Ersetzen'-Schaltfläche.
