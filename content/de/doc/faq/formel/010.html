Wie verändere ich die Größe eines Unterausdrucks?

Es gibt 5 Möglichkeiten die Größe zu verändern:

  • size 20: die Größe 20pt wird gesetzt.
  • size +5: die momentane Größe +5pt wird gesetzt.
  • size -5: die momentane Größe -5pt wird gesetzt.
  • size *1.5: setzt die Schriftgröße auf das Eineinhalbfache der momentanen Größe
  • size /2: setzt die Schriftgröße auf die Hälfte der momentanen Größe

Damit die Änderung auf einen ganzen Unterausdruck wirksam ist, muss er in diesen Klammern { } oder in anderen Klammern gesetzt sein. Beispiel:

  • a b size *2 {c d} e erzeugt also: abcde.

Wenn mehrere Schriftgrößenänderungen angegeben werden, hat die innerste Größenangabe Vorrang:

  • size *1.5 {a size 12 b size *1.5 c d} erzeugt im Ausdruck: abcd.

Um die Grundgröße einer Formel zu ändern, siehe FAQ #008.
