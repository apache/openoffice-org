Wie verändere ich die Schrift eines Unterausdrucks?

Sie können eine der folgenden Schrifteigenschaften verwenden:

  • italic ist ein Schriftsatz, um es schräggestellt darzustellen.
  • das macht auch ital vorangestellt.
  • nitalic macht die Schrift aufrecht.
  • bold macht die Schrift fett.
  • nbold steht für normal gewichtete Schrift.

Zusätzlich verwendet Formel drei generische Schriftfamilien, und zwar:

  • font serif für eine serife Schrift, gewöhnlich Times.
  • font sans für eine sans serife Schrift, gewöhnlich Helvetica oder Arial.
  • font fixed für eine Festbreitenschrift, gewöhnlich Courier.

Natürlich können die Attribute für die Schrift auch kombiniert werden, hier ein Beispiel:

  • bold font sans A erzeugt ein A.

Um Schriftattribute für einen kompletten Unterausdruck geltend zu machen, muss er in diesen Gruppierungsklammern: { } oder anderen Klammern eingeschlossen sein. Beispiel:

  • geben Sie a b bold {c d} e ein um abcde zu schreiben.

Für den Fall, dass mehrere Attribute gleichzeitig gesetzt werden, wird dem innersten Attribut Vorrang gegeben, Beispiel:

  • bold {a b nbold c d} ergibt: abcd.

Um die Schriften, die in Formel verwendet werden zu ändern, siehe FAQ #008.
