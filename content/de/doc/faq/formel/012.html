Ich benötige ein Symbol, das Formel nicht zur Verfügung stellt. Was kann ich da machen?

Wenn Sie eine Schriftart haben, die dieses benötigte Symbol enthält, können Sie ein benutzerdefiniertes Symbol erstellen. Das geht wie folgt:

  • öffnen Sie den Dialog (Extras-Katalog...).
  • klicken Sie auf 'Edit'.
  • Geben Sie den Namen für das Symbol in das Feld 'Symbol' ein.
  • Wählen Sie in 'Symbolset' die Symbolgruppe aus (oder geben Sie eine neue ein), die das von Ihnen gewünschte Symbol enthalten soll.
  • Wählen Sie nun die Schriftart in der Liste 'Schriftart' aus, zum Beispiel StarMath.
  • Außerdem wählen Sie die Formatierung in der Liste 'Schriftschnitt' aus.
  • Nun wählen Sie das Symbol aus, das Sie hinzufügen möchten.
  • Abschließend noch ein Klick auf 'Hinzufügen' um das Symbol zu dem gewünschten 'Symbolset' hinzuzufügen.

Nun können Sie das Symbol wie jedes andere auch im Dialogfeld mit dem Syntax %namedessymbols direkt verwenden.

Suchen Sie ein bestimmtes Symbol, schauen Sie bitte zuerst in der Schriftart StarMath nach, diese enthält 25 zusätzliche Symbole.

Eine andere Quelle für Symbole ist die sogenannte AMS Schriftart von der "American Mathematical society".

Sie können diese Schrift im Postskript 1 Format von http://www.ams.org/tex/amsfonts.html downloaden.

Wenn Sie Microsoft Windows verwenden, bevorzugen Sie vielleicht die BaKoMa TrueType Version von http://www.ctan.org/tex-archive/fonts/cm/ps-type1/bakoma/.

Weitere Erklärungen entnehmen Sie bitte der Dokumentation dieser Schriftarten.
