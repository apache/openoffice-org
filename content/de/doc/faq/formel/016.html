Wie verändere ich die Abstände um meine Formeln?

Wenn Sie Ihre Formel in ein Text Dokument eingebettet haben, verfahren Sie wie folgt:

Wenn Sie die Abstände um eine einzelne Formel ändern möchten, tun Sie das Folgende:

  • Wählen Sie die Formel aus.
  • Wählen Sie im Menü Format-Objekt aus.
  • Gehen Sie in das Register 'Umlauf'.
  • In der Gruppe 'Abstände' können Sie die Abstände um das Formelobjekt einstellen.

Wenn Sie die Abstände um alle Formeln ändern möchten, tun Sie dies so:

  • Öffnen Sie den Stylist (Format-Stylist).
  • Klicken Sie auf das Icon Rahmenvorlage (oben, das dritte Icon von links).
  • Wählen Sie 'Formel' mit der Maus aus und dann klicken Sie mit der rechten Maustaste darauf.
  • Wähle Ändern... im Popup-Menü aus.
  • Nun das Register Umlauf.
  • In der Gruppe 'Abstände' können Sie die Abstände um das Formelobjekt einstellen.

Zusätzlich kann auch Formel Abstände um die Formeln einfügen. Befolgen Sie hierfür diese Schritte:

  • Öffnen Sie Ihre Formel durch Doppelklick.
  • Wählen Sie im Menü Format-Abstände.
  • Klicken Sie nun auf 'Kategorie' und wählen Sie 'Ränder' in dem Popup-Menü aus.
  • Jetzt können Sie die Abstände rund um Ihre Formel ändern.
