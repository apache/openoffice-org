Wieso habe ich auf meinem Windows 9x/Me System so viele Allgemeine Schutzverletzungen (General Protection Fault - GPF) Abstürze während ich OpenOffice.org Anwendungen benütze?

Gut, GPF Abstürze kommen auf Windows-Systemen häufig vor, und natürlich werden nicht alle von OpenOffice.org Programmen verursacht! Wie auch immer, manchmal ein OpenOffice.org Programm diese Art von Fehler erzeugen. OpenOffice.org ist ein Open Source Programmpaket, das auf den Code von StarOffice von Sun Microsystems Inc. basiert. Sun hat dessen Code der Open Source Gemeinschaft übergeben - manche Teile unterliegen aber noch immer anderen Lizenzen: zum Beispiel die Rechtschreibprüfung oder das Hilfesystem. Deshalb funktionieren einfach einige Funktionen von OpenOffice.org noch nicht, oder nicht richtig, was einen GPF Absturz verursachen kann.
Die Gruppe der OpenOffice.org Entwickler arbeitet hart um diese Probleme zu beseitigen. Es wurde diesbezüglich schon viel erledigt, aber noch viel mehr bleibt zu tun. Sie, als ein Benutzer können helfen, indem Sie die IssueZilla Sektion auf der OpenOffice.org Webseite besuchen. Wenn Sie einen reproduzierbaren Fehler (nicht einen einmaligen Absturz) in einem OpenOffice.org Programm gefunden haben, melden Sie diesen mittels IssueZilla. Als Gemeinschaft ist OpenOffice.org bestrebt, diese Fehler zu beseitigen; Ihre Hilfe ist erwünscht.
