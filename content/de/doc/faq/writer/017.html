017 - Wie schütze ich einen Textbereich so, dass er nicht verändert werden kann?

Führen Sie nacheinander folgende Schritte durch:

  • Schreiben (oder laden) Sie Ihren Text.

  • Markieren Sie den zu schützenden Text.

  • Aus dem Pulldown Menü wählen Sie: Einfügen -> Bereich

  • In der Abteilung Schreibschutz aktivieren Sie "Schützen"

  • Sie können diesem Bereich nun einen Namen geben. Tun Sie dies nicht, wird automatisch der Name "Bereich1" vergeben.

  • Klicken Sie auf den "Einfügen" Knopf.

  • Sie können für einen solchen geschützten Bereich auch ein Kennwort vergeben. Wer das Kennwort nicht kennt, kann dann auch den Schutz von diesem Bereich nicht entfernen.

Ein Kennwort für einen geschützen Bereich vergeben: Wir gehen davon aus, dass Sie nachträglich - für einen bereits geschützten Bereich - ein Kennwort vergeben wollen.

  • Aus dem Pulldown Menü wählen Sie: Format -> Bereiche

  • Wählen Sie den Namen des Bereiches, den Sie bearbeiten wollen.

  • In der Abteilung "Schreibschutz" ist "Geschützt" bereits aktiviert. Sie aktivieren nun das Feld "Mit Kennwort"

  • In dem sich öffnenden Fenster geben Sie das Kennwort ein. (Das Kennwort muss mindestens fünf Zeichen lang sein!) Bestätigen Sie das Kennwort.

  • Klicken Sie auf OK (im "Kennwort eingeben" Fenster)

  • Klicken Sie auf OK (im "Bereiche bearbeiten" Fenster)

Sie wollen einen geschützten Bereich verändern? Das geht wie folgt:

  • Aus dem Pulldown Menü wählen Sie: Format -> Bereiche

  • Wählen Sie den Bereich aus, den Sie verändern wollen. Wenn der Bereich mit einem Kennwort geschützt ist, müssen Sie zunächst das Kennwort eingeben.

  • Dazu klicken Sie in der Abteilung "Schreibschutz" auf "Mit Kennwort" und geben dann in dem sich öffnenden Fenster das entprechende Kennwort ein.

  • Klicken Sie OK.

  • Nun Klicken Sie auf "Aufheben" button

  • und dann auf OK

  • Nun können Sie in dem ehemaligen geschützten Bereich jegliche Änderungen vornehmen.

  • Um den Schutz für den Bereich wieder herzustellen, folgen Sie bitte den o. a. Schritten in "Wie schütze ich einen Textebereich"".

Wenn Sie einen Bereich mit einem Kennwort versehen haben, werden Sie automatisch nach dem Kennwort gefragt, wenn Sie irgendwelche Veränderungen an diesem Bereich durchführen wollen.
