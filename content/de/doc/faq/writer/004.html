004 - Wie erscheint ein Logo auf jeder Seite meines Dokumentes?

Folgendermaßen:

  • Öffnen eines neuen Textdokumentes

  • Aus dem Pulldown Menü wählen: Einfügen -> Kopfzeile -> Standard

  • Klicken Sie in den Rahmen, der die Kopfzeile darstellt

  • Aus dem Pulldown Menü wählen: Einfügen -> Grafik -> Aus Datei...

  • Suchen Sie das gewünschte Logo, dann klicken Sie Öffnen

Wenn das gewünschte Logo nicht auf allen Seiten erscheint, kann dies zwei Gründe haben: Die z. Zt. benutzte Seitenvorlage kann die falschen Kopfzeileneinstellungen haben. Die Einstellungen prüfen wir wie folgt:

  • Aus dem Pulldown Menü wählen: Format -> Seite

  • Klick auf den Reiter 'Kopfzeile'

  • Prüfen Sie, ob "Inhalt links/rechts gleich" aktiviert ist. Wenn dies nicht der Fall ist, so erscheint das Logo nur auf ungeraden oder geraden Seiten, aber nicht auf beiden.

Wenn die Grafik auf einer oder mehreren Seiten erscheint, und dann aber plötzlich nicht mehr erscheint, dann liegt dies wahrscheinlich daran, dass auf der Seite, auf der das Logo nicht mehr erscheint, eine veränderte Seitenvorlage eingeschaltet wurde. Beachten Sie die Information in der Statuszeile neben der Seitennumerierung. Der Name, der dort erscheint ist der Vorlagenname. Ändert sich dieser Name von einer Seite zur anderen, so wurde das Dokument in verschiedene Teile aufgeteilt. Wenn Sie dies ändern wollen:

  • Cursor auf die Seite, welche die falsche Vorlage enthält

  • Aus dem Pulldown Menü wählen: Format -> Stylist

  • Klicken Sie auf das Symbol "Seitenvorlagen" (Es ist das VIERTE Symbol von links)

  • Doppelklick auf die gewünschte Seitenvorlage (diejenige, die Ihre Kopfzeile mit dem Logo richtig anzeigt). Nun können Sie den Stylist wieder schließen.
