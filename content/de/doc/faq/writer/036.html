036 - Wie füge ich eine Seitennummerierung in der Form 'Seite 1 von N' ein?

Bevor Sie beginnen, müssen in Ihrem Text eine Fuß- oder eine Kopfzeile existieren. Wollen Sie noch eine Fuß- oder Kopfzeile einfügen, dann wählen Sie Einfügen ->Fußzeile ->Standard oder Einfügen ->Kopfzeile ->Standard

  • Klicken Sie in den Bereich der Fuß- oder eine Kopfzeile Ihres Dokumentes

  • Geben Sie den Text "Seite" ein

  • Wählen Sie Einfügen-Feldbefehl-Seitennummer damit die Nummer der jeweiligen Seite eingefügt wird

  • Schreiben Sie den Text "von"

  • Wählen Sie Einfügen-Feldbefehl-Seitenanzahl damit der Gesamtzahl der bisher existierenden Seiten eingefügt wird

Nun erscheint die Seitennummerierung in der Form "Seite 1 von n" auf jeder Seite Ihres Dokumentes.
