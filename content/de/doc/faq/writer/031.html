031 - Einige '#' Zeichen, die ich in einer Reihe eingegeben habe, erzeugen eine dicke schwarze Doppellinie. Wie kann ich diese Linie wieder löschen?

Einige # Zeichen (genauer gesagt genügen drei solche Zeichen) ist ein "Tastaturkürzel" für eine durchgezogene Linie unter einem Absatz in einem Dokument. Wenn sie nun die Linie löschen wollen, stellen Sie den Cursor unmittelbar über die Linie. Wählen Sie Format ->Standard, und die Linie verschwindet.

Wie kann man das erklären? Erzeugen Sie wiederum eine solche Linie. Klicken Sie in den Absatz über der dicken Linie und wählen: Format ->Absatz. Die Linie erscheint, wenn Sie den Reiter "Umrandung" klicken. Sie haben als die Standardformatierung ihres Absatzes verlassen. Deshalb kann - wie oben beschrieben - ein Format ->Standard auch die Linie wieder löschen.
