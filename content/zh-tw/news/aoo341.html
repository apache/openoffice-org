Apache OpenOffice 3.4.1 正式發表

Apache OpenOffice™ 3.4.1 正式發表，支援更多語言，更快更穩定。

Apache OpenOffice 團隊正式發表 Apache OpenOffice 3.4.1 。這是開放免費的文書軟體 Apache OpenOffice 的最新版本。維護更新版 3.4.1 版，以全球 228 國下載超過一百二十萬次的 Apache OpenOffice 3.4.0 為基礎，支援更多語言，相容更多平台，修正錯誤並強化效能。 OpenOffice 3.4.1 可由 http://www.openoffice.org/download/ 下載，或由 OpenOffice 3.3 、 3.4 版的「說明(H)／檢查是否有更新(U)...」選單更新。

Apache OpenOffice （原名 OpenOffice.org ）目前為 Apache 的育成專案。

3.4.1 版詳情，可參閱版本說明。其中重點包括：

  • 新增支援高棉文、芬蘭文、英式英文、斯洛伐尼亞文及斯洛伐克文，並更新荷蘭文、西班牙文、義大利文、匈牙利文。加上原有的美式英文、德文、日文、捷克文、法文、巴西葡萄牙文、俄文、簡體中文、正體中文版本、加利西亞文及阿拉伯文，支援達 20 種語言。
  • 支援由 Windows XP 到 Windows 8 ， Linux 32 位元及 64 位元，及蘋果 Mac 作業系統，包括最新的山獅 (Mountain Lion) 。本版修正相容性支援，在 Windows 8 上執行更順暢。
  • 效能、相互操作性及安全性加強。
  • 修正 69 個使用者回報的錯誤。
  • 社群正在移植到 BSD 、 Solaris 及 OS/2 平台上。詳情請參閱 OpenOffice 平台移植計劃網頁。

Apache OpenOffice 使用者還可享有眾多協力廠商團體的周邊產品。由 SourceForge 改寫重開的擴充套件及範本站上，共有 2388 個範本及 628 個擴充套件。

    未來版本走向，請不吝給我們建議

    除了 Apache OpenOffice 3.4 維護更新版外，我們團隊成員也已同步開始 3.5 版的工作。 3.5 版的主要目標包括：提高和 MS Office 的相容性，更快、更穩、更好用，並支援更多語言。 Apache OpenOffice 3.5.0 預計將在 2013 年第一季推出。

    接著將是重大版本 Apache OpenOffice 4.0 版，將包含最近捐給 Apache 的 IBM Lotus Symphony ，其新增及改善的功能。這一版，有機會讓我們跳脫原有框架，加上額外的創新設計。

    我們團隊已經在著手腦力激盪 Apache OpenOffice 4.0 的各種可能性。我們也邀請使用者、社群成員及有興趣的各方人士，和我們一起討論，讓 4.0 版再次締造十年前 1.0 版的偉大成就。我們討論的平台架設在 Google 好問上，歡迎加入我們，分享您的想法。

    社群志工

    Apache OpenOffice 是由社群主導的開放原始碼專案。 Apache OpenOffice 團隊歡迎志工，協助程式撰寫、品管、界面設計、翻譯／本地化、文件撰寫、網站設計及行銷推廣各方面的工作。若您有興趣參與，請寫信到我們的公開郵件通訊群組 dev@openoffice.apache.org 自我介紹，並告訴我們您想參與什麼工作。
