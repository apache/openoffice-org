Apache OpenOffice™ 3.4 下載突破百萬

上周 Apache OpenOffice 3.4 發表後，感謝使用者和媒體的熱情支持，下載已突破一百萬次。

2012 年 5 月 16 日 Apache OpenOffice 專案管理委員會最新消息— 在 Apache 軟體基金會及 Apache 育成計劃下， Apache OpenOffice 團隊今天公佈了 Apache OpenOffice™ 3.4 的初步下載數字。

  • Apache OpenOffice 3.4 已由 SourceForge 下載了 1,000,663 次。多國語言包、軟體開發套件及原始碼下載另計。
  • 下載的使用者來自超過 200 個國家。
  • 下載的版本中有 87% 為 Microsoft Windows ， 11% 為 MacOS ， 2% 為 Linux 。

自上周發表後， Apache OpenOffice 3.4 即成為媒體和社群網路的焦點：

  • 在 12 種以上語言的媒體上，報導了超過 60 個以上的新功能。
  • 在 Twitter 帳號上發表的新聞被轉貼了 61 次，讀者超過 43,000 人。
  • Facebook 、 Identi.ca 及 Google+ 專頁上也非常活躍。

媒體報導

BetaNews 記者 Nick Peers 搶先報導「 Apache 發表了 OpenOffice 的重大更新版—馬上昇級吧！」並總結新功能如下：

3.4 版的新功能一看就知道—新版最明顯的變化有：啟動速度變快很多，強化 OLE 物件處理、改進圖表繪製及支援可縮放向量圖形 (SVG) 。此外還支援了 ODF 1.2 加密，並改以 Apache License 2 引擎發表。

OMG! Ubuntu! 的 Joey Sneddon 則說：「 Apache OpenOffice 3.4 沒有全新風貌，而是善加保持原來的優良傳統。」

ZDNet 的 Paula Rooney 也作了相關報導。在介紹完 Apache OpenOffice 3.4 新版公告上的「新奇玩具」後， Paula 強調了未來發展的重要性：

…[對] Apache 軟體基金會—及其他開放原始碼文書軟體的死忠支持者—來說，脫離商業公司掌控後的全新開發流程，正是最能吸引使用者及贊助商之處。

The Register 的 Iain Thomspon 強調了 OpenOffice 在圖形處理上的重大突破，特別提到了程式設計師 Armin Le Grand 的貢獻。

TweakTown 上的 Anthony Garreffa ，在文章中將 AOO 3.4 推為「今日下載」，稱 Apache OpenOffice 3.4 是：

…超過 20 年軟體工程的結晶。由一個小軟體創始到現在，它的穩定性，使其他軟體難以望其項背。完全開放的開發流程，每個人都可以回報錯誤、提出建議，協助改進軟體。
