Apache 軟體基金會宣佈 Apache OpenOffice™ 為一級專案

風行全球 228 個國家的文書軟體領導品牌， 2012 年 5 月發表後，已超兩千萬次下載。

馬里蘭州 Forest Hill — 2012 年 10 月 18 日—由超過 150 個開放源始碼專案、新創計劃的開發義工、管理義工及育成義工，組成的 Apache 軟體基金會 (ASF) ，今天宣佈 Apache OpenOffice 已脫離 Apache 育成中心，昇級為一級專案 (TLP) 。這象徵著 OpenOffice 的社群和產品，已在 Apache 軟體基金會英才主義的流程原則下，順利運作。

「 OpenOffice 結業，見證了在 The Apache Way 下，由原型專案成功轉型，成為完整消費者產品的轉換過程。」 Apache 軟體基金會副主席、 Apache OpenOffice 導師 Ross Gardler 指出。「在育成過程中，資深的 Apache 義工輔導專案，協助 OpenOffice 的新舊義工們，建立 Apache 式多元、開盎的社群。」

「 OpenOffice 結業，代表官方認可， OpenOffice 不論在技術開發、社群運作上，皆可自治運作。」 Apache OpenOffice 的副主席 Andrea Pescetti 說。「『 Apache Way 』及其實踐方法，如所有決策徹底公開透明的原則，讓 OpenOffice 得以不斷吸引新的義工投入，票選出多元、活躍的專案管理委員會，確保 Apache OpenOffice 的穩健未來。」

OpenOffice 最初是在 1990 年代由 Star Divison 開發，分別在 1999 年和 2010 年被昇陽電腦和甲骨文公司購併，並在 2011 年中轉移給 Apache 軟體基金會。

在 Apache 育成中心的開發過程中，普及的免費文書軟體 Apache OpenOffice ，翻修了將近一千萬行的程式碼，加上無數的新功能，修正數十個使用者回報的錯誤。此外， OpenOffice 亦榮獲五項業界獎項，囊括個別功能、最多下載及最佳開放源始碼文書軟體等。

自 Apache OpenOffice 3.4 於 2012 年 5 月，以 20 國語言發表以來，已經在 228 個國家下載超過兩千萬次，使用者囊括個人、企業、教育單位、政府機關等。之後， OpenOffice 團隊隨即投入開發創新功能，預計於 2013 年第一季與第四季發表。

「 OpenOffice 成為 Apache 的一級專案，實在太酷了！」Apache OpenOffice 版本經理 Juergen Schmidt 說。「為實現這個目標，我們面臨了重重挑戰。為發表第一個版本 Apache OpenOffice 3.4 版，社群不只要把程式碼由 Oracle 套件庫中轉移到 Apache 下，還要改寫不相容的函式庫，以符合 Apache 授權規定。現在 Apache OpenOffice 的原始碼，可以造福其他組織的各種專案。」

「我們對此極為自豪，雙手歡迎 OpenOffice 加入 Apache 世界領導品牌專案的大家庭！」 Gardler 補充。

使用和展望

不論任何人，都可以由 http://openoffice.org 任意免費下載 Apache Office 。 OpenOffice 不限電腦數、不限使用者數，皆可下載安裝，授權完全免費。 OpenOffice 致力於支援開放標準，包括 ODF（ ISO/IEC 26300 的第一個實作），未來並計劃支援 CMIS、OpenSocial 及 OData。

如同所有 Apache 軟體， Apache OpenOffice 以 Apache License v2.0 授權發表，由社群自己選出的活躍義工團隊監督運作。專案管理委員會 (PMC) 管理 OpenOffice 日常的運作，包括社群發展及新版發表等。 Apache OpenOffice 的原始碼、下載、說明文件、通信群組和相關資訊，請參閱 http://openoffice.apache.org/ 。

關於 Apache 軟體基金會 (ASF)

自 1999 年成立起，由全義工組成的基金會，管理超過一百五十個世界領導品牌的開放源始碼專案，包括世界最多人使用的網站伺服器 Apache HTTP 伺服器。透過稱為「 The Apache Way 」的 Apache 軟體基金會英才主義流程，超過 400 個個人會員、 3500 位貢獻者共同合作，開發免費的企業級軟體，造福全世界百萬使用者：上千個軟體方案以 Apache 授權散佈，社群並積極參與 Apache 軟體基金會通信群組、新創計劃輔導，及基金會的官方年會、培訓、展覽活動 ApacheCon。Apache 軟體基金會是美國註冊的 501(c)(3) 非營利慈善團體，由個人及企業贊助成立，贊助者包括 AMD、Basis Technology、Citrix、Cloudera、Facebook、Go Daddy、Google、HP、Hortonworks、Huawei、IBM、InMotion Hosting、Matt Mullenweg、Microsoft、PSW Group、SpringSource/VMware、WANdisco 及 Yahoo!。詳情請參閱 http://www.apache.org/ ，或追隨 Twitter 上的 @TheASF 。

「 Apache 」、「 Apache OpenOffice 」、「 OpenOffice 」及「 OpenOffice.org」為 Apache 軟體基金會的商標。其他品牌及商標為其註冊人所有。
