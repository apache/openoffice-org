This is the tool category of Education Project


The tool category is a sharing repository of links or tools that work with OpenOffice.org. It could be as an extension or as an add-on, or simply as an editor. The purpose is to make those tools available to the community in a central point. And also to try to help localization and enhancement of these tools.
Don't hesitate to propose links or tools you are aware of in your language!

Multilingual tools working with OOo 
  • DMaths is a dedicated tool for OpenOffice.org and StarOffice to help Math teachers writing formulas. It's available in several languages
  • Lodel is a high quality web publishing design system. An interesting tools for researcher publications.
Monolingual tools working with OOo
  • OOoHG: [French] site and tools dedicated to History and Geography teachers
