What's new in Build SRC609

New in Build Environment

  • fixed dependencies for building static libraries on Unix platforms

  • fix for the kill targets -> dmake killobj and dmake killbin work again

  • introduced macros for STATIC and DYNAMIC linking



Changes in the CVS Tree


  • new project jut (Java UNO Tools. It is a collection of Java UNO utilities. It is the Java parallel to the cpputools project)

  • new project ucb( UCB core implementation and related helper services. Also includes various UCPs (FILE, HIERARCHY, REMOTE, WEBDAV).)
