I file binari di OpenOffice.org possono essere usati per fini commerciali/d'affari?

Si, puoi usare i file binari di OpenOffice.org per fini commerciali. Prima di farlo, per favore assicurati di aver compreso che il software è in fase di sviluppo e può avere delle funzioni non complete. Inoltre, leggi e comprendi bene la limitazione di garanzia che trovi in cima alla pagina di download dei file binari:

http://www.openoffice.org/dev_docs/source/get_binaries.html

In alternativa, se sei interessato a StarOffice per fini commerciali, per favore visita questa URL:

http://www.sun.com/staroffice/starofficenow
