How can I save a Draw document into a standard file format (BMP, JPG, ...)?

The format .sxd used in Draw is not a graphics format, but a document format.

To save a Draw document into a graphics format:

  1. Select File > Export from the Menu Bar.
  2. Select the graphics format you wish to use from the File Type combo box.
  3. Click the Export... button.
  4. Define the options you wish to use, if options are presented, then click OK.
