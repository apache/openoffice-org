Windows - Zip and GZ Files


Select your program:
  • WinZip

    • Using Winzip

  • Download file into a directory. (Be sure to remember the directory where the file was downloaded.)
  • Minimize your browser window.
  • Double Click on My Computer.
  • Locate the Directory where you downloaded the file.
  • Right Click on the file
  • Select Extract To...
  • Enter the Directory where you want the files to be extracted
