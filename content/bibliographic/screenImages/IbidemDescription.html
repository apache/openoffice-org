Bibliographic Applications - screen images and menu structure



Applications -

  • IBIDEM -The Bibliographic component of Nota Bene, a Scholar`s Word Processing Application.

  • Biblioexpress

This information is provided as a guide to developing a GUI interface design for the OpenOffice Bibliographic facility. Nota Bene is probably the best example of integrated word processor and bibliographic functions that we have found.

Note: that we are not planning to copy these interface panels but they provide examples of the features required.
