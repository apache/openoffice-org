Shown below a sample OO0 Writer document that contains only the following bit of text a bibliographic entry and a bibliographic table, with custom formatting.

Test Message [1]

Bibliography

1: Wilson. D, Testing Bibliographic databases, ( Melbourne: Free Press, 2003)
