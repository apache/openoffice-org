If you want user support and don't want to file a bug...

Please navigate to the Support Page where you'll find a variety of ways to get help with Apache OpenOffice including a "users" mailing list and the OpenOffice.org Community Forum.

If you want to contact the Apache OpenOffice developers...

Please check out the mailing list page and use the developer mailing list.

If you want to search existing issues, you can use this simple interface...

It searches only for bugs that are marked "Open" in the "www" project. This means that it may miss bugs which have been fixed in internal builds. Using this will open the search in a new browser window, Bugzilla Window. If your problem shows up in the issue listing, please add your own comment to the original report. You will have to Log In properly to Bugzilla from the Bugzilla Window to do that.

Additionally, if your issue is not found in the issue list, and you want to submit a new issue, you can apply for a Bugzilla account from the BugZilla Window.
