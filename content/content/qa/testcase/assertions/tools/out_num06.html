Test Case: out_num06

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org launches the outline numbering dialog successfully.

Description:

     
      1. From the menu bar,  select - Tools - Outline Numbering.
      2. The  Outline Numbering dialog should be launched.
      3. Ensure there are 2 tab settings,
                  Numbering and Position.
                  select each tab setting
             4. Click on Cancel in the  Outline Numbering dialog.
Expected Result:
All operations should complete successfully.

End of test case
