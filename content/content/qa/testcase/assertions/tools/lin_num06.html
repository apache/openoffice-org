Test Case: lin_num06

Test Area: Solaris  - Linux - Windows

Test Aim
Verify OpenOffice.org inserts line numbers to a text document successfully.

Description:

      1. From the menu bar,  select  tools - line numbering.
      2. The Line Numbering dialog should be launched.
      3. Check the show numbering box on in the Line Numbering dialog.
      4. Click OK, Line Numbering dialog.
      5. Hit several carriage returns, the margin of the page should show a line count, 5, 10, 15, 20 etc.
      6. From the menu bar,  select  tools - line numbering.
      7. The Line Numbering dialog should be launched.
      8. Check the show numbering box off in the Line Numbering dialog.
      9. Click OK, Line Numbering dialog.

     
Expected Result:
All operations should complete successfully.

End of test case
