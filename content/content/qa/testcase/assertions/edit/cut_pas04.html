Test Case: cut_pas04

Test Area: Solaris - Linux - Windows

Test Aim
Verify OpenOffice.org cut and paste functions correctly

Description:

    1. Open a new text file, enter a short text message e.g. "short text message"
    2. From the menu bar,  select Edit-Select  All.
    3. The entered text should be highlighted.
    4. From the menu bar,  select Edit - Cut, the  highlighted text should be removed from the text document.
    5. From the menu bar,  select Edit - Paste, "short text message" should now be reinserted
Expected Result:
All operations should complete successfully.

 

End of test case.
