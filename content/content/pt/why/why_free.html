Porquê o Apache OpenOffice: programa gratuito

O Apache OpenOffice é um programa grátis. Isso significa que é livre para o transferir, livre para instalá-lo em quantos computadores quiser, livre para dar cópias a quantas pessoas quiser. Pode usar o OpenOffice para qualquer fim, sem restrição: privado, educacional, administração pública, comercial … Livre, realmente livre.

  • Gratuito

    É difícil acreditar que um programa de alta qualidade e fácil de usar como o Apache OpenOffice pode ser realmente livre. Mas, graças a contribuições de tempo e dinheiro, de pessoas e empresas (grandes e pequenas), o OpenOffice é livre para usar hoje. Além disso, a licença de código aberto usada pelo OpenOffice significa essa liberdade nunca lhe pode ser tirada. A comunidade OpenOffice ficará muito contente se achar o nosso programa útil. Por favor ajude, falando sobre o Apache OpenOffice e, se puder, ofereça uma cópia. Mais pessoas usam o OpenOffice como resultado de recomendação pessoal mais do que por qualquer outra forma.

  • A liberdade de se preocupar

    No entanto, uma licença de software grátis significa muito mais do que uma economia de custo único. Significa que nunca precisará de se preocupar novamente se o seu programa é legal, ou se a licença vai expirar algum dia. Para as pessoas responsáveis por uma série de computadores - desde apenas alguns numa pequena empresa, centenas numa escola, milhares numa empresa - este é um enorme peso. Não há necessidade de auditorias a programas, de manter as facturas durante anos, nem de se preocupar em acabar em Tribunal porque interpretou mal algumas letras pequenas num contrato de licença.

  • Nós valorizamos a sua liberdade

    Como parte de uma comunidade dedicada ao software grátis, também valorizamos a sua liberdade de escolha. O Apache OpenOffice vai ler e escrever ficheiros que podem ser usados noutro programa comum de escritório. Foi o primeiro programa no mundo a suportar totalmente o padrão ISO para formatos de ficheiro, formato que está a ser adoptado por um número crescente de fornecedores de programas. Se quiser utilizar outro programa, nós queremos que seja livre para fazê-lo. Se quiser mudar do Microsoft Windows para Apple Mac ou para Linux, nós queremos que seja livre para fazê-lo. Se gosta desta abordagem, não hesite em fazer parte da comunidade Apache OpenOffice.
