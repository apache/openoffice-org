Come posso aggiornare una vecchia versione di OpenOffice con una nuova?

Si raccomanda di disinstallare la versione precedente di OpenOffice prima di procedere all'aggiornamento. Ciò può essere fatto, sotto Windows, usando il programma Setup dal Menu Start e selezionando 'Deinstallation'.

Puoi anche copiare e rinominare la cartella principale di OpenOffice se vuoi mantenere alcuni o tutti i file contenuti in essa. Se provi a installare in un sistema Windows una nuova versione di OpenOffice.org sopra una più vecchia, durante la procedura di installazione potranno essere segnalati degli errori.
