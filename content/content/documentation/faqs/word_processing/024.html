If I click in the middle of a table cell or in an empty paragraph, OpenOffice.org insert tabs in front of the cursor position. How do I stop this?

This is caused by a default setting of the OpenOffice.org interface called 'Direct Cursor'. To change this behavior:

  • From the pull down menus, select: Tools -> Options

  • In the dialog that appears, select: Text Document - Cursor

  • Change the 'Insert' value to 'Left Paragraph Margin' (it was set to 'Tabs')

  • Click on OK
