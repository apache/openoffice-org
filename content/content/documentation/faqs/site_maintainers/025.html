Is there a place where I can see the CVS directory tree?

Every project has an interface that allows you to see the file list. Click the Version control link in the left menu of any project page to view a directory tree of the files in that project. You can put together the URL for your specific project once you see the pattern in their names.

Examples of URLs for browsing the directory tree view of openoffice.org files

  • Visit the http://documentation.openoffice.org/source/browse/documentation/www/ web page to see files in the documentation project.
  • Visit the http://documentation.openoffice.org/source/browse/documentation/www/faqs/ to see the web page that shows the directory of FAQs.
  • Visit the http://documentation.openoffice.org/source/browse/documentation/www/faqs/index.html to see the web page that shows information for the documentation project's faqs index page.
  • Visit the http://marketing.openoffice.org/source/browse/marketing/www/index.html to see the web page that shows information for the marketing project index page.
  • Visit the http://qa.openoffice.org/source/browse/qa/www/index.html to see the web page that shows information for the qa project index page.

If you remove the URL and the /source/browse/ directories from the above examples, the path from your home directory remains. You can use this information to navigate through the OpenOffice.org file system at the command line.
