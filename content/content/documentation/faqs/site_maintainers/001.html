What is SSH and when will I need it?

SSH, or Secure SHell, is a secure connection to a file server, such as OpenOffice.org, via shell (or terminal) access. You require a passphrase to connect, and cannot connect without one. The secure connection also requires that a set of key files match. The key files are generated on your machine: one part passes to OpenOffice.org, and one parts stays on your machine. The passphrase is also chosen at this time. After the uploaded public key at OOo is mated to the private key file on your machine during the login process, you are able to connect to the OpenOffice.org file server. An SSH tunnel like this is used whenever you checkout web pages and source files from OpenOffice.org, or when you update or add new files to OpenOffice.org's file system.
