No sounds under Linux?

  1. OpenOffice.org 2,x uses the Java Media Framework (JMF) to render media such as film and sound under Linux.
  2. Download and install and configure this tool so that gallery sounds and others are available
  3. For instructions, see the Troubleshooting chapter (# 14) of the User Guide (http://documentation.openoffice.org/manuals/OOo2.x/user_guide2_draft.pdf)
