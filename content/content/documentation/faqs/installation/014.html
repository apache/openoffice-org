How do I perform a silent install?

In corporate environments, when rolling out OpenOffice.org to many machines, the easiest way to do it is via a silent install.

For Openoffice.org1.1.x, the How To Install OOo with a Response File document explains how to do it.

For 2.x, the process is covered in Setup Guides for OpenOffice.org 2.x

Windows users can use Orca tool to manage this
