How do I install OpenOffice.org on Windows?

If you have downloaded the binary file for Windows from OpenOffice.org, it may be in ZIP, TAR, or EXE format.

  1. If the file is in the ZIP or TAR format, decompress it into a temporary folder (with a program such as WinZip or Pkzip).
    If the file is in the EXE format, simply double-click the file to decompress it.
  2. Double-click the Install icon that you find in the Install Folder; or, double-click the file named setup.exe. A guided installation dialog will begin and will step you through the installation process.
