Where can I find definitions for the error codes?

The error code definitions live in the Help files.

  1. In the OpenOffice.org window, go to the Help drop-down menu.
  2. Select Contents.
  3. Select the Index tab by clicking on it.
  4. In the Search input box, type error codes. An index list will pop up, with "error codes" in it.
  5. Under the error codes section in the index list, double-click on OpenOffice.org Calc. This brings up the page of calc error codes.
