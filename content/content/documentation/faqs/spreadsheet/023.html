What is the maximum number of cells in an OpenOffice.org spreadsheet?

The maximum number of rows per sheet is 32,000. The maximum number of columns per sheet is 256. The maximum number of cells per sheet is 8,192,000. And the maximum number of sheets per workbook is 256.

  • A large OpenOffice.org spreadsheet was converted from another application. Some of my rows are missing! What happened?
