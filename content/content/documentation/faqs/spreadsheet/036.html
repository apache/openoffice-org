Why does Calc refuse to open my text file?

A text file must be imported into Calc using .csv or .txt format. Otherwise, the file will automatically open in Writer.

  1. Rename the file so that it uses the .csv file extension.
  2. In the File > Open dialog, set the File type field to Text CSV. This is quite a way down in the list so it is easy to miss. It is grouped with the spreadsheet file types.

For more information:

  • In the pull-down menus, go to Help > Contents > Index and type csv files;spreadsheets in the Search term input bar.
