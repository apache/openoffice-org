Why do I see the formula text and not the result of the formula?

In OOo Calc all formulas must start with the equal sign (=).*

If your formula does start with an = character, then check that the options are set to not view the formula. Go to Tools > Options; then, in Spreadsheet > View verify that there is no "tick mark" in the box beside the Formulas selection.

*This is a difference between Calc and MS-Excel(R).
