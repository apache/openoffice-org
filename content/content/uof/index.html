UOF Project

Overview

We are the UOF team and our main goal is to provide a set of extensions to allow for interoperability between ODF (OpenDocument) standards (currently ODF 1.1) and UOF(currently UOF 1.1). As a part of this interoperability initiative, a set of extensions are being developed that can be installed on top of OpenOffice suite to allow for opening and saving UOF files that adheres to UOF specifications.

The extensions are based on XSLT transformations between two XML formats, along with some pre-processing to transform them into a friendly structure firstly. We chose to use an Open Source development model that allows developers from all around the world to participate and contribute to the project.

      Unified Office document Format(UOF)

      TheUnified Office document Format(UOF) is a Chinese open standard for office applications document format, including Word processing, Spreadsheet and Presentation,which is being developed by the Chinese Office Software Work Group (COSWG), led by the China Electronics Standard Institute (CESI), the Ministry of Industry and Information Technology(MIIT), major suppliers of Chinese office software suites such as RedOffice, WPSOffice, EIOffice and NeoshineOffice, and other academic institutions like Beijing Information Science & Technology University( BISTU). The UOF is different from other document format standards, for it bases on Chinese XML labels and respects Chinese users custom.

      OASIS OpenDocument/ISO/IEC 26300 file format(ODF)

      The OpenOffice.org application uses the OpenDocument file format (also known as ODF) as native file format. This means, OpenOffice.org stores all documents in OpenDocument file format unless the user requests to store them in a different file format.

      The OpenDocument file format is developed by a technical committee at OASIS. It further is an approved and published ISO standard (ISO/IEC 26300:2006). The OpenDocument specification is available at the OASIS OpenDocument TC's web page. See opendocument.xml.org for more information on OpenDocument.
