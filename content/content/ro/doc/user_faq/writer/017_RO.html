017 - Cum pot proteja (asigura) o porţiune de text astfel încât conţinutul să nu poată fi schimbat?

Urmaţi aceşti paşi pentru a proteja o secţiune de text:

  • Scrieţi sau inseraţi textul dorit

  • Marcaţi textul de protejat

  • Din meniurile derulante alegeţi: Insert -> Section (Inserare -> Secţiune)

  • Bifaţi căsuţa de lângă "Protected" (Protejează)

  • Daţi secţiunii un nume (sau lăsaţi numele dat implicit)

  • Apăsaţi pe căsuţa "insert" box

  • Dacă doriţi puteţi proteja secţiunea cu o paolă. Acest lucru va împiedica pe oricine care nu cunoaşte parola să elimine protecţia pusă pe această porţiune de text.

Pentru a pune o parolă pe o porţiune protejată de text:

  • Din meniurile derulante alegeţi: Format -> Sections (Formatare -> Secţiune)

  • Alegeţi secţiunea dorită după nume (dacă este posibil)

  • Apăsaţi pe căsuţa de lângă "Password protected" (Protejare prin parolă)

  • În dialogul care apare introduceţi o parolă şi confirmaţi-o

  • Apăsaţi pe OK (în dialogul despre parolă)

  • Apăsaţi pe OK (în dialogul despre formatarea secţiunii)

Urmaţi aceşti paşi pentru a modifica o secţiune protejată anterior:

  • Din meniurile derulante alegeţi: Format -> Sections

  • Introduceţi parola în dialogul care apare (apare doar dacă a fost stabilită o parolă); Apăsaţi pe OK

  • Alegeţi secţiunea dorită după nume (dacă este posibil)

  • Apăsaţi pe butonul "Remove" (elimină)

  • Apăsaţi pe butonul OK

  • Faceţi schimbările dorite

  • Urmaţi paşi, de mai sus, de la instrucţiunea "protejaţi o secţiune de text" pentru a aplica din nou protecţia pe porţiunea de text

Notaţi că, odată ce aţi stabilit o parolă pentru o proţiune de text, va fi cerută parola înainte de orice dialog care v-ar da posibilitatea să accesaţi acea porţiune de text.
