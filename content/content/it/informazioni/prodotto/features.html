Caratteristiche OpenOffice.org 2.0

Introduzione
OpenOffice.org 2.0 - elaboratore di testo, foglio elettronico, programma di presentazioni e altro

OpenOffice.org 2.0 è una suite di ufficio multipiattaforma e ricca di funzionalità. L'interfaccia utente e le funzionalità sono molto simile a quelle degli altri prodotti presenti sul mercato come Microsoft Office o Lotus SmartSuite, al contrario di questi prodotti commerciali, OpenOffice.org è completamente gratuita per tutti.

Open Standards XML File Format (OASIS OpenDocument)
A partire con la versione 2.0 OpenOffice.org utilizza il formato standard aperto XML OASIS OpenDocument come formato predefinito. Il formato OASIS è un formato file indipendente sia da qualsiasi produttore che da programmi specifici, e questo ne garantisce la libertà ed indipendenza.
In aggiunta ad OpenOffice.org stesso, la suite per ufficio open source KOffice come anche i derivati da OpenOffice.org come StarOffice supporta il formato file OASIS OpenDocument. Il formato file OASIS OpenDocument è anche uno dei formati file raccomandati dalla Commissione Europea.
Nuove Viste Multi-Panello
La nuova vista multi-panello aumenta la produttività ed usabilità di OpenOffice.org. La vista multi-panello fornisce tutti gli strumenti e le finestre più importanti  attraverso un unica interfaccia utente  coerente, integrata e semplice.  Gli ex utenti di Microsoft PowerPoint troveranno OpenOffice.org 2,0 facile usare.
Nuovi CustomShapes (compatibile con Microsoft AutoShapes)
OpenOffice.org ora supporta CustomShapes, che sono molto simili a AutoShapes di Microsoft.  Di conseguenza, AutoShapes viene importato e visualizzato correttamente.  CustomShapes sono figure che possono cambiare non soltanto la loro dimensione ma anche la loro apparenza.
Altre Transizioni ed Effetti Animati per le Presentazioni
OpenOffice.org ora supporta molti altri effetti di animazioni e transizioni per lucidi.  Ora gli utenti hanno più opzioni per dare alle loro presentazioni un'apparenza professionale.  In più, il supporto potenziato agli effetti migliora la compatibilità con Microsoft PowerPoint.
Esportazione PDF Potenziato
OpenOffice.org 1,1 aveva introdotto la caratteristica dell'esportazione del pdf.  La versione 2,0 ora estende le caratteristiche di base dell'esportazione.  Con la nuova versione è possibile definire i livelli di compressione delle immagini incluse.  In più, il supporto del pdf ora gestisce correttamente le  thumbnail, le immagini ed i hyperlink.
Nuovo Modulo Database Potenziato
OpenOffice.org ha sempre avuto strumenti di interfacciamento per i database, ma nelle precedenti versioni erano molto nascosti.  OpenOffice.org 2,0 comincia gestire le basi di dati come qualunque altra applicazione, una nuova base di dati può essere generata via menu "File - Nuovo".  Per i nuovi utenti, OpenOffice.org fornisce una nuova procedura guidata Tabella che permette generare le tabelle della base di dati senza alcuna conoscenza delle basi di dati o del SQL.  Il nuovo motore integrato della base di dati di HSQLDB, basato su tecnologia Java, consente di generare un "documento database".  Questi semplici file database non richiedono la presenza di un server database come MySQL o Adabas D. Tutte le informazioni  (definizioni di tabella, dati, ricerche, formulari, rapporti) sono memorizzati in un file XML.
Stampa Guidata in Serie
La nuova procedura Stampa Guidata in Serie rende molto facile fare spedizioni a centinaia di destinatari.  La procedure guidata indirizza gli utenti passo dopo passo attraverso il procedimento.  Le varie fonti di dati possono essere usate per accedere alle informazioni sugli indirizzi.
Funzionalità Conteggio Parole Potenziato
In passato era possibile trovare le statistiche sotto "File - Proprietà del documento ...".  Inoltre era impossibile contare il numero di parole in una selezione.  OpenOffice.org 2,0 elimina questa limitazione.  Ora un nuovo dialogo di Conteggio Parole può essere trovato nel menu "Strumenti".
Supporto per Tabelle Nidificate
Con OpenOffice.org 2,0 ora è possibile generare le tabelle all'interno di altre tabelle (tabelle annidate).  Ciò aumenta l'usabilità complessiva generale ed inoltre migliora la compatibilità con Microsoft Word.
Supporto per le Firme Digitali
Un'altra nuova caratteristica è il supporto delle firme digitali.  OpenOffice.org usa i certificati immagazzinati nei repositori comuni.
Supporto XForms
OpenOffice.org 2,0 ora permette di generare i formulari basati su standard aperti W3C XForms.  Con XForms è molto facile implementare una logica semplice senza nessuna abilità di programmazione.
Filtri WordPerfect
OpenOffice.org 2,0 integra un filtro di WordPerfect sviluppato dalla comunità open source.  Gli sviluppatori sono invitati a contribuire a questo progetto.
65,536 Righe in Calc
OpenOffice.org 1,1 supportava soltanto 32.768 righe nei documenti del foglio elettronico, che hanno determinato dei problemi di interoperatibilità con Microsoft Excel.  La nuova versione ora porta OpenOffice.org allo stesso livello di Microsoft Excel.
Supporto Tabelle Pivot Potenziato
Con il DataPilot in OpenOffice.org 2,0 (simile alla funzionalità  PivotTable di Microsoft Excel) ora è possibile generare i nuovi gruppi, utilizzare filtri  di dati basati sui valori, mostrare le differenze e le percentuali anziché i valori assoluti, ecc.  Le nuove caratteristiche di DataPilot trasformano OpenOffice.org Calc in uno strumento molto potente per l'analisi di dati.
Installatori Nativi
OpenOffice.org 2,0 introduce l'utilizzo di programmi di installazione nativi del sistema operativo di riferimento. Per esempio, i file .CAB e .MSI sono forniti per Microsoft Windows;  I file RPM sono disponibili per Linux.  Per gli utenti con esigenze di gestione più avanzate, SUN fornisce strumenti di configurazione ed amministrazione come componenti del suo software StarOffice.  StarOffice 8 inoltre introduce strumenti per facilitare la migrazione delle macro.
Integrazione Desktop Nativo
Le applicazioni di OpenOffice.org 2,0 si integrano piacevolmente nell'ambiente desktop.  Quindi, OpenOffice.org assomiglia e si comporta come qualsiasi altra applicazione su una data piattaforma.  Su Windows XP assomiglia ad un'applicazione di Windows e su Linux usa la stessa interfaccia utente di GIMP o Evolution.
Barre degli Strumenti Fluttuanti
Le barre degli strumenti sono molto più flessibili in OpenOffice.org 2,0.  Ora è molto facile spostare, fissare, staccare  e personalizzarele barre degli strumenti.
Supporto, Servizi e Soluzioni
Un numero crescente di aziende forniscono supporto, servizi e soluzioni per OpenOffice.org. Sempre più e più aziende offrono versioni derivate di OpenOffice.org che includono componenti supplementari come gli strumenti di amministrazione (per esempio StarOffice di Sun Microsystems).
