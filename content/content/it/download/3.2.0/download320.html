Le versioni con JRE comprendono un Java Runtime Environment, necessario per le funzioni più avanzate. Se avete già un Java Runtime Environment e volete mantenerlo, usate le versioni senza JRE; altrimenti usate quelle con JRE.

Chi volesse installare la versione Windows senza JRE può scegliere l'installazione personalizzata e deselezionare la voce relativa a JRE.

Altre Risorse

  • Dimensione file: Windows: 154 MB | Linux: 180 MB | Mac OS X: 180 MB
  • CD-ROM: Per acquistare un CD, visitate l'elenco disponibile presso la pagina CD-ROM internazionale (inglese) e contattate il distributore più vicino per il prezzo e dettagli di spedizione.
  • MD5SUM: il controllo MD5SUM permette di verificare l'integrità del file prelevato. Come effettuare il controllo (in inglese)
  • Dizionari italiani: i dizionari italiani inclusi in OOo 3.1 sono disponibili anche per chi usa OOo 3.1 in altre lingue tramite l'estensione Dizionari Italiani. In OOo 3.1 italiano tale estensione è già inclusa.
  • Java: JRE/JDK può essere prelevato presso http://java.com in base alle vostre esigenze. Verificate i requisiti di sistema per informazioni aggiuntive.
  • Centrale di download: ulteriori download non localizzati, a volte più recenti, sono disponibili presso la Centrale di Download, unitamente alla Documentazione per il codice sorgente, al più recente codice via CVS e ad altri file binari e sorgenti.
