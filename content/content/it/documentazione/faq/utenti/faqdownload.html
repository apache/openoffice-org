Generali

Dove trovo i file da prelevare per il mio sistema operativo ?

Per l'italiano, la versione stabile corrente si trova all'indirizzo: http://it.openoffice.org/varie/downloads.html
Più in generale si va all'indirizzo: http://download.openoffice.org/index.html#latest e si fa clic sul collegamento "Latest build: OpenOffice.org <numero-versione>", per visualizzare la pagina Web in cui si può scegliere la lingua e il sistema operativo che interessano, per la versione più recente del software.

Qual è il modo migliore per fare il download di un file così grande ?

Potete utilizzare un programma di gestione dei download per avviare, sospendere e riprendere il download in momenti e in giorni diversi. Nella pagina download manager sono elencati i programmi di questo tipo più diffusi.

E' troppo grande da prelevare via FTP. Quali alternative ho ?

Potete ordinarlo su CD da vari distributori.

Non riesco più a trovare il file che ho prelevato. Dov'è ?

Se avete prelevato la versione Windows di OpenOffice.org in italiano, il file dovrebbe avere un nome del tipo: OOo_1.1.0_Win32Intel_install_it.zip ("1.1.0" è il numero di versione e "_it" indica la versione in italiano). Se non ricordate in quale cartella avete salvato il file, potete utilizzare il programma di utilità di Windows "Trova" (o "Cerca", in Windows XP).
Le operazioni da eseguire sono: "Start" => "Trova" => "File o cartelle". Nella casella Nome file, digitate "OOo" e lanciate la ricerca.

Dopo aver trovato il file, utilizzate un programma come WinZip per scompattare il file zip in una directory. Successivamente, entrate in questa directory e lanciate in esecuzione il programma "setup.exe", facendo doppio clic sulla sua icona.

Che devo fare dopo aver prelevato il file ?

Dovete scompattare il file zip che avete prelevato, utilizzando un qualsiasi programma di gestione dei file compressi, probabilmente già installato nel vostro computer. In Windows, il più famoso è WinZip, mentre nella maggior parte delle altre piattaforme c'è gzip.
Successivamente, è sufficiente lanciare in esecuzione il programma di installazione (setup.exe) che è nella directory in cui avete scompattato i file.

Note di rilascio

Dove trovo le note di rilascio ?

Per avere le note di rilascio dell'ultima versione, navigate all'indirizzo: http://download.openoffice.org/index.html e fate clic sulla voce "Release notes".
La documentazione viene aggiornata ogni volta che viene rilasciata ufficialmente una nuova versione del programma.

Cos'è cambiato dalla versione precedente ?

Per saperlo, leggete le note di rilascio della versione di OpenOffice che state utilizzando.

Perché le note di rilascio non contengono informazioni più specifiche ?

Gli sviluppatori e le persone che si occupano della gestione dei rilasci non hanno la disponibilità di molto tempo al giorno. Se pensate che manchi qualcosa, inviate un suggerimento o, meglio ancora, partecipate al progetto e contribuite anche voi, nel tempo che avete disponibile.

Varie

Dell'intera applicazione, posso prelevare solo il programma di gestione testi ?

No, in questo momento non è possibile. OpenOffice è un'applicazione integrata (una "suite") e questo significa che è un'unica applicazione con diverse interfacce.
La maggior parte di queste interfacce utilizza le stesse librerie, per cui il file complessivo occupa circa 70 MB. Un file da installare con i componenti di base necessari al funzionamento del solo programma di gestione testi sarebbe già grande più di 50 MB. Rispetto al totale non si avrebbe un grande risparmio, considerando la perdita di tutte le altre applicazioni.

Posso prelevare OpenOffice in parti ?

Non proprio, ma leggete la domanda di queste FAQ della sezione Download/Generali sui programmi di gestione dei download.

Posso prelevare solo le patch per aggiornare la mia versione corrente, senza dover prelevare completamente la nuova versione ?

Al momento di redazione di questa risposta, con OpenOffice non è ancora possibile, ma si spera di renderlo possibile in futuro.

Cosa devo fare dopo aver prelevato il file ?

Dovete scompattare il file compresso e lanciare in esecuzione il programma di installazione (setup.exe).

Unicows.dll

Cos'è ?

E' una libreria che serviva in precedenza perché il programma potesse gestire i caratteri internazionali nei sistemi operativi Windows 95, Windows 98 e Windows ME. Era necessaria con le prime versioni di sviluppo di OpenOffice 1.1 (quelle indicate con: 1.1beta1, 1.1beta2, 1.1rc1 e 1.1rc2 ). Adesso, però, è compresa nel file da prelevare per l'installazione della versione definitiva di OpenOffice 1.1.
Non ce n'è bisogno se avete Windows NT o Windows XP.

A cosa serve ?

Potreste avere bisogno di prelevarla solo se state utilizzando Windows 95, Windows 98 o Windows ME.

Dove la trovo ?

Sembra che Microsoft cambi questo collegamento di continuo. Alla data di redazione di questa risposta, l'indirizzo è quello che segue il collegamento: "Download MSLU".
In alternativa, eseguite l'aggiornamento alla più recente versione di OpenOffice e verrà installata automaticamente con il file di installazione.
