Créer des tables .csv dynamiques Le but de ce document est de présenter une technique permettant d'utiliser de manière dynamique des fichiers de données au format .csv dans un document Base. Cela permet de :
- Utiliser dans Base des données issues d'une autre application qui génère des fichiers .csv.
- Partager des données saisies sous Base avec une autre application capable de traiter des données au format .csv.
