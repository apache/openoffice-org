How-Tos Programmation

Ces 'Comment faire' sont des explications pas à pas et illustrées d'utilisation des fonctions d'OpenOffice.org.  La colonne 'V2' indique que le document a été mis à jour vers la version 2.

Comment diffuser vos macros avec un Add-on Tout est dit dans le nom du document ! Un Add-on est un programme qui vous permet de simplifier notablement l'installation de vos macros par vos utilisateurs. SXW
Outils Xray Ce document, qui contient la macro Xray, vous expliquera comment l'installer et l'utiliser.  Xray est un outil qui permet aux programmeurs de macros de visualiser les propriétés, méthodes et interfaces disponibles sur une variable objet. Nouvelle version publiée le 16/12/2006. SXW V2
Installeur de macros Ce document Calc est en fait un installeur générique qui permet d'installer une bibliothèque en basic OOo et de créer un menu personnalisé pour accéder aux fonctions macros de cette bibliothèque. Traduction à partir du document original de Didier Lachieze que vous trouverez ichttp://www.ooomacros.org/dev.php SXC
Dans la jungle de l'API Un document qui vous apprendra à vous débrouiller avec l'API de OOo. Ecrit de façon assez personnelle, il est destiné aux débutants en Basic et contient, expliqué très simplement, tous les éléments pour se repérer. SXW
Exemples pour bdd Exemples de macros pour manipuler des bases de données et réaliser des formulaire évolués. SXW
Batchmacro Comment accéder à une feuille de classeur par une macro lancée en ligne de commande. SXW

Ces boîtes à outils aident les programmeurs à piloter OpenOffice.org depuis un langage externe, avec la technologie COM Automation. Chaque boîte à outils est un zip contenant :
 - un module avec des routines similaires à celles qu'offre OpenOffice.org Basic pour accèder à l'API
 - un module contenant toutes les constantes et énumérations de l'API
 - un module donnant plusieurs exemples de programmation
 - une routine permettant d'utiliser l'outil Basic Xray depuis le langage; l'outil pour Delphi contient aussi une version native de Xray.
