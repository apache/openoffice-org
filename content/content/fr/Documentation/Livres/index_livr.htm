Programmation OpenOffice.org -
Macros OOoBasic et API		 OpenOffice.org, suite bureautique libre et gratuite, est munie du langage de script OOoBASIC et d’une API permettant de manipuler ses objets. Ainsi automatisable et extensible, elle peut s’intégrer parfaitement au système d’information de l’entreprise.
L'objectif de ce livre est à la fois d'accompagner les utilisateurs découvrant les macros et la programmation dans OpenOffice.org, et de fournir un document de synthèse de l'API pour les utilisateurs avancés.
Ces derniers pourront éventuellement transposer les exemples donnés en langage Basic dans d'autres langages, en Python par exemple. Sur le CD-Rom d'accompagnement, le lecteur retrouvera tous les exemples contenant ces macros, la suite bureautique OpenOffice.org en version 1.1.2fr, le SDK , et enfin de nombreux outils et macros de la communauté fr.OpenOffice.org.
