Cas Test : edt_nav04

Zone de test : Solaris - Linux - Windows

But du test

Vérifier que le Navigateur d'OpenOffice.org peut s'ouvrir et se fermer avec succès

Description:

    1. Ouvrez un nouveau document texte
    2. A partir de la barre des menus, sélectionnez Edition - Navigateur
    3. Le Navigateur doit s'afficher
    4. Pressez F5, le navigateur doit se fermer.
Résultat attendu
Toutes les opérations doivent se dérouler avec succès

Fin du cas test
