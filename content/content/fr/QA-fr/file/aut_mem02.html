Cas Test : aut_mem02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer un mémo à partir de l'Autopilote



Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Autopilote - Mémo
    2. La boite de dialogue du Mémo à partir de l'Autopilote doit se lancer
    3. Assurez-vous que Moderne est sélectionné dans la boite de dialogue
    4. Sélectionnez Créér dans la boite de dialogue
    5. Assurez-vous qu'OpenOffice.org créé un Mémo à partir de l'autopilot
    6. Le document doit montrer :
      . une entête indiquant "Mémo Interne"
      . suivi par un cadre indiquant
         De :
         A : 
    7. Fermez le document Mémo
        
       
Résultat attendu :
Toutes les opérations doivent s'être déroulées avec succès
