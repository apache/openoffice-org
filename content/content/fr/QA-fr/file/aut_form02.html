Cas Test : aut_form02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer un formulaire à partir de l'Autopilote

Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Autopilote - Formulaire
    2. La boite de dialogue de l'Autopilote de formulaire doit s'ouvrir

    3. . assurez vous que Source de Données - Bibliographie est sélectionnée dans la boite de dialogue
      . assurez vous que biblio est sélectionné dans le champs Table ou Requête de la boite de dialogue
      . sélectionnez Author dans la fenêtre Champ Existant de la boite de dialogue
      . placez Author dans la fenêtre Champ de Formulaire en utilisant la touche "->"
    4. Sélectionnez Suivant >> dans la boite de dialogue de l'Autopilote
    5. Sélectionnez Créez dans la boite de dialogue
    6. Sauvegardez le document sous test_form.sxw
    7. Assurez vous qu'OpenOffice.org a bien créé le formulaire
    8. Le document doit montrer une liste de nom au-dessous du champ Author
    9. Fermez le document Formulaire
        
     
Résultat attendu :
Toutes les opérations doivent se compléter avec succès
