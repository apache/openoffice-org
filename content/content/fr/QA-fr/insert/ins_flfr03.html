Cas Test : ins_flfr03

Zone de Test : Solaris  - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut insérer un cadre flottant dans un document texte


Nécéssité
Le cas test sav_RTF doit avoir été complété avec succès avant d'exécuter ce test

Description:

      1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
      2. A partir de la barre des menus, sélectionnez Insertion Frame
      3. La boite de dialogue de propriété du Frame doit s'afficher
      4. Dans le champ Contenu, sélectionner RTFfichtest comme vous l'avez précédemment créez dans le test sav_RTF
      5. Assurez-vous qu'un Frame est inséré dans le document, affichant le contenu de fichier RTFfichtest
      6. Fermez le document
Résultat attendu
Toutes les opérations doivent être complétées avec succès

Fin du cas test
