Cas Test : for_autf05

Zone de Test : Solaris - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut dés/activer l'autoformat


Description:

  1. A partir de la barre des menus, sélectionnez Fichiez - Nouveau - Texte
  2. Un nouveau document texte doit se lancer
  3. Assurez-vous que l'Autoformat - Pendant la saisie peut être dés/activé en sélectionnant Format - Autoformat - Pendant la saisie
  4. Fermez le document texte

Résultat attendu :

Toutes les opérations doivent se compléter avec succès
