Cas Test : for_pg05

Zone de Test : Solaris - Linux - Windows

But du Test 

Vérifier qu'un document texte OpenOffice.org peut contenir une page de couleur grise


Description:

    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
    2. Un nouveau document texte doit se lancer
    3. A partir de la barre des menus, sélectionnez Format - Page
    4. Une nouvelle boite de dialogue de formatage de la page doit s'ouvrir
    5. Assurez-vous qu'elle comporte bien  8 onglets
      Gérer, Page, Arrière Plan, Entête, Pied de page, Bordures, Colonnes, Note de bas de page
      sélectionnez chaque onglet
    6. Sous l'onglet  Arrière Plan  sélectionnez la couleur grise dans couleur d'arrière plan
    7. Assurez vous que l'arrière plan de la page apparaît bien en gris
    8. Fermez le document texte
Résultat attendu
Toutes les opérations doivent se compléter avec succès

Fin du cas test
