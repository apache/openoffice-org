2 Roundtable

2.1 High contrast switching of toolboxes

CJ mentioned that some toolboxes (e.g. Navigator, Stylist) and the target window of the hyperlink dialog have problems with switching into high contrast mode or switching back. He will file tasks for those problems.



The next topic is a problem with the database beamer. It doesn't keep the connection to the document frame it belongs when it is not docked into the frame window.

CJ will discuss the problem (task for this problem is available) with Andreas Schluens if the problems can be solved or if it may be necessary to disable undocking of the beamer window.



GT raised the problem of multi-branched resource files (*.src) that were created by the merge-in process that merges changes in the translation database into these files.

He will start a discussion about this problem in the appropriat inhouse newsgroup.



He also brought up the problem of mnemonics in user interface specifications. CJ will take care of this problem.
