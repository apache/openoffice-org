2 Roundtable

2.1 New toolbox images

We will create new toolbox images (modern look, 256 colors, own color list). So the extra team (Stella and a new student) have to draw about 6.000 bitmaps. While the creation of the images isn't finished, the imagelist creation tool (bmp.exe) accepts bitmaps in different color depths. In the same step, we will remove all useless bitmaps. Also we want to change the look of the dropdown toolbox buttons. We will replace the green arrows with a new control like the backward and forward buttons of Mozilla.



2.2 UI changes

Frank mentioned that it is necessary to change the usability of some dialogs (macro recorder, edit templates, find & replace). Christian told us that he has got already some tasks for it. These changes will be part of the next major version.
