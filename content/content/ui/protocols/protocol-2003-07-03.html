2.1 Move dialog source code to UI project

For OOo we can start to move some dialog code into the UI project. In the first step this will not cause changes at the APIs of the dialogs.

2.2 New tabbed control (aka Schwipp-Schwapp Control) for the Three-Pane-View in OOo Impress

This control was formerly used in the Explorer of StarOffice 5.x. The existing code may be reused in the new Three-Pane-View in Impress. The code for this control should reside in the UI project. It may also be used in some of the tabbed dialogs, where such a functionality makes sense. A new feature of this control may be the possiblity to rearrange the tabs using Drag&Drop.
