Keyboard Shortcuts for OpenOffice / StarOffice Draw and Impress

Author: Armin Weiss , Frank Loehmann

Date: 02/06/2002

This is not a complete list of all shortcuts used in Draw/Impress. This proposal lists only shortcuts, which have to be changed or added to fulfill accessibility criteria.

Selecting a Drawing Object via Keyboard

If the cursor position is within text in your document, do the following:

1. Press F10 to move the focus to the menu bar.

2. Press Ctrl+Tab until the Drawing toolbar is the focus.

3. Press the Right Arrow key to choose Select Objects.

4. Press Ctrl+Enter to select the first drawing object.

4 b. Or Press Enter to Return Focus to Document

5. Press the Tab key to cycle forward (or Shift+Tab to cycle backward) through the objects until handles appear on the object you want selected.

Create Objects

To allow users to create drawing objects on their own, Draw and Impress create a default object when a drawing tool is selected and Ctrl + Return is pressed. After creation this object has the focus in the document.

This default object has the dimension of 5 * 5 cm / 2 * 2 inches.

Select Objects

Tab: Selects first object on page if focus is already in document

Move Objects

Cursor Keys: Move in defined steps

Alt + Cursor Keys: Move object by one screen pixel

Travel

Placeholder on Impress pages: Ctrl + Return:

Ctrl + Return: Starting point Focus in document

  1. Jumps to first placeholder of current page (in edit mode)

  2. Jumps to next placeholder if present (in edit mode)

  3. Inserts a new page with a copy of the current page layout, after the current page. (no other drawing objects will be copied)

  4. Jumps to first placeholder of current page

Ctrl + Return: Starting point: Focus on drawing object

  1. Jumps to next place holder (if present) in hierarchical order of current page

  2. Jumps to next placeholder if present. (in edit mode)

  3. Inserts a new page with a copy of the current page layout, after the current page. (no other drawing objects will be copied)

  4. Jumps to first placeholder of current page (in edit mode)

General Shortcuts

STRG+N: Creates new presentation / drawing

STRG+M: Inserts new page

STRG+D: Duplicates the current page and inserts it after current page

F5: Start presentation

STRG+K: Call hyperlink dialog

F7: Spellchecker

Outliner

Shift + F9: Show/Hide formatting in outline view

Presentation

During a presentation in full screen mode the following short cuts can be used.

N, Return, Page-Up, Cursor right, Cursor Down and Spacebar (or mouse click): Next page or executes next animation

P, Page-Up, Left Arrow, Arrow up or Bachspace: Jumps to previous page or execution of previous animation
