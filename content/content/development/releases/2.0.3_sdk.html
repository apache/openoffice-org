OpenOffice.org 2.0.3 Software Development Kit (SDK)

The OpenOffice.org SDK is an add-on for the OpenOffice.org office suite. It provides the necessary tools and documentation for programming the OpenOffice.org APIs and creating own extensions (UNO components) for OpenOffice.org.

What's new in OpenOffice.org 2.0.3 SDK?

  • New API's
    We have introduced 40 new UNOIDL types. Please see "Overview of new API's" for a roughly overview of new API's.
  • Improved UNOIDL reference
    The UNOIDL reference gets improved again. The pages becomes a breakdown, that means the inheritance tree of mutliple inheritance interfaces are better structured, better positioning of the deprecated|unpublished keywords on the left side of the documented type name, consolidation of mutliple "see also" tags.
