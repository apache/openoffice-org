Feature title TaskId Spec. abstract Component
HSQLDB databases now allow to modify existing table field types i41785 feature-info:
Since m92, OpenOffice.org is able to edit the types of existing table fields HSQLDB databases. That is, if you create a new HSQLDB database, a table therein, and then choose to change the definition of that table, it was previously not possible to change the type/characteristics of existing columns in the table. Technically, this new behaviour is an enhancement, since it introduces new functionality for this particular database type. To the user, it's the fix for a bug (which in fact has been reported several times in the Beta test), since s/he expects from OOo's pet database HSQLDB to offer the most basic "rapid database design" functionality. 		dba
