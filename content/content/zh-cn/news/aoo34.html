Apache OpenOffice 团队发表 Apache OpenOffice™ 3.4

全球超过百万人使用，开源、跨平台、支持多种语言的文字处理软件，数不尽的新功能。 Windows 、 Linux 和 Macintosh 版本免费下载。

www.OpenOffice.org — 2012 年 5 月 8 日— Apache OpenOffice 团队发表今天正式发表 Apache OpenOffice™ 3.4 。这是 OpenOffice 在 Apache 软件基金会下的第一个版本。

Apache OpenOffice 是原版的开源文字处理软件，适合专业及一般使用者。

「 Apache 软件基金会自从受赠 OpenOffice.org 起，基金会及我们的孵化计划团队，就接下了一个严峻的挑战：重新组织社区，并要将授权不明的 OpenOffice ，转变成严格符合Apache授权的软件。」 Apache 软件基金会主席，兼 Apache OpenOffice 项目督导的 Jim Jagielski 表示，「随著 Apache OpenOffice 3.4 发表，项目团队交出漂亮的成绩单：吸引了来自超过 21 个赞助企业的开发者，同时避免企业过度干预伤害社区；建立稳定的系统基础，同时拥有比其他同质产品更多的显著优势；最重要的是：建立了活跃、多元的使用者与开发者社区。」

Apache OpenOffice 是开源文字处理软件的领导品牌，全球超过百万人使用，涵盖家庭、企业、政府部门、研究机构、学术单位，并有十五种语言版本。 Apache OpenOffice 3.4 现已提供免费下载。 OpenOffice 3.4 的特色如下：

  • 包含文字处理、电子表格、演示文稿图形、资料库、绘图、数学公式编辑，并提供 Windows 、 Linux （ 32 及 64 位元）及 Macintosh 版。
  • 提供：英文、阿拉伯文、捷克文、德文、西班牙文、法文、加利西亚文、匈牙利文、义大利文、日文、荷兰文、俄文、巴西葡萄牙文、简体中文及正体中文版本。
  • 加强开放文件格式支持，包含新的 ODF 1.2 加密选项，及新的电子表格函数标准。
  • 增强了电子表格中的数据透视支持表功能。
  • 加强图形功能，包括线段端点样式、裁切变形及支持可缩放向量图档 (SVG)。
  • 性能和质量上改进。

要知道全部有哪些新功能，请参阅新版说明。

Apache OpenOffice 使用者还可享有众多合作厂商团体的周边产品，超过两千三百个模板，和超过八百个扩展组件。

社区开发模式「 The Apache Way 」

OpenOffice 基本系统最初是在 1990 年代由 Star Division 开发，分别在 1999 年和 2010 年被升阳电脑和甲骨文公司购并，并在 2011 年中移交给 Apache 软件基金会。

目前团队由开发人员、测试人员、翻译等各种志愿者组成，通过俗称「 The Apache Way 」的流程运作，并有来自全球上百位的开源爱好者共同支持。

「从十年前的初版发表到现在，在 Apache 下的 OpenOffice 第一个版本 3.4 版，揭开了 OpenOffice 历史上崭新的一页。」 RedMonk 的首席分析师 Stephen O'Grady 指出。「经过几个月的努力， OpenOffice 得以用新的授权发布，更能吸引全球使用者、开发者和独立软件商。」

Apache OpenOffice 目前为 Apache 孵化项目，由预备项目管理委员会 (PPMC) 管理日常的运作、社区发展和新版本发布。 Apache OpenOffice 的原始码、下载、说明文件、通信群组和相关信息，请参阅 http://openoffice.apache.org/ 。

作为第一个 OpenOffice 的 Apache 官方版本，开发团队符合了孵化项目管理委员会订立的政策。为此，开发团队将与 Apache 软件基金会 (ASF) 授权政策不兼容的代码、程序库、功能等，一一找出来、以兼容的功能替代或移除，以符合孵化项目管理委员会的授权政策。

展望

展望未来， OpenOffice 计划支持更多语言，强化无障碍设计，增强使用体验和性能，并采用 IBM Lotus Symphony 所贡献的新功能和新设计。

Apache OpenOffice 项目管理委员会欢迎您的加入。有很多地方需要大家的参与与贡献。欲知详情请参阅如何参与。

「 Apache OpenOffice 投资开发了十五年以上，是开源文字处理软件的领导品牌。我们很高兴能完全以 Apache 2.0 授权发表。这是未来在 Apache 下， OpenOffice 创新开发的基础。」身兼版本管理者和资深核心开发者的 Juergen Schmidt 说。「这个重大里程碑，象征 OpenOffice 已经拥有足够多元、活跃、专业的社区基础，向未来迈进。我们可望能在接下来几周内，脱离孵化阶段，晋升为 Apache 一级项目。」

关於Apache软件基金会 (ASF)

Apache 的技术，广泛使用在超过一半以上的全球网际网路、多达千万亿比特的资料、每秒数兆的运算、上亿的项目上，协助改善了无数使用者和开发者的体验。自 1999 年成立起，以「 The Apache Way 」协助、开发、孵化自由软件的创新发展。全球六大洲有超过 400 个个人会员、 3500 位贡献者共同合作，管理超过 150 个项目。

「 Apache 」、「 Apache OpenOffice 」、「 OpenOffice 」及「 OpenOffice.org」为 Apache 软件基金会在美国及其他国家的（注册）商标。其他品牌及商标为其注册人所有。
