Apache OpenOffice Windows版

Windows版本支持

当前的Apache OpenOffice支持Microsoft Windows XP，Vista ，Windows 7和Windows 8。

具体平台的版本说明：

  • Windows 2000: Open Office可能可以在Windows 2000，但我们没有进行测试，也不建议。
  • Windows XP: 测试 ，稳定 ，没有特定于平台的已知问题。
  • Windows Vista: 没有特定于平台的已知问题。
  • Windows 7: 经测试，稳定。
  • Windows 8: 我们运行的“桌面应用程序”，而不是一个Metro App。需要x86或64位的Windows。我们不支持Windows RT，适用于平板电脑基于ARM版本的Windows。

注：Apache OpenOffice是分发为一个32位应用程序。 可以成功地在64位版本的Windows 7和Windows8以32位模式运行。但是在这种情况下需要一个32位的JVM以实现一些功能。有关详情请点击这里.

硬件要求

硬件的要求是相当低的，甚至较老的机器也能运行OpenOffice：

  • 至少256M的内存 (推荐为512MB)
  • 最少650M（默认安装）. 在安装完成并删除临时文件后，Apache OpenOffice将使用约440M的磁盘空间。
  • 1024 x 768或者更高分辨率，至少256色

针对Windows的特征

Apache OpenOffice在Windows具有以下特殊功能：

  • 邮件客户端集成，通过消息传递应用程序编程接口（MAPI）
