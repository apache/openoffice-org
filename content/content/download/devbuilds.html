Apache OpenOffice daily/nightly builds and snapshots

The following installation packages are automatically generated by buildbots.
For testing purposes only: They are not formally approved/tested and can contain issues!
End users must use only our officially approved and widely tested builds, available from the Download page.

Target public

The following builds are for use by OpenOffice project volunteers who are participating in the QA or Localization activities and are tracking specific bugs in our Bugzilla.

Available builds

  • Windows nightly build: 10 languages, built every night from trunk (AOO450-dev); EXE format.
  • Windows snapshot build: 10 languages, built every Thursday from branch AOO42X (AOO420-dev); EXE format.
  • Windows snapshot build: 10 languages, built every Friday from branch AOO41X (AOO4114-dev); EXE format.

  • Linux 64-bit nightly build: 10 languages, built every night from trunk (AOO450-dev); RPM, DEB format.
  • Linux 64-bit snapshot build: 10 languages, built every Friday from branch AOO42X (AOO420-dev); RPM, DEB format.
  • Linux 64-bit snapshot build: 10 languages, built every Thursday from branch AOO41X (AOO4114-dev); RPM, DEB format.
