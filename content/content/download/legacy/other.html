Java Runtime Environment (JRE)

The JRE is required for the Base (database) component, for wizards as well as several other features. The package increases the download size between 15 - 40 MB depending on the chosen platform. For more information see: Java and OpenOffice.org. When the download link includes the Java Runtime Environment (JRE) and the download does not work then please delete the "-wJRE" part from the link or uncheck the option. Only for Windows and Linux (RPM) the builds are released with and without an included JRE.

Important Notes:

  • A language pack is not a release for its own and has to be installed over a stable release with the same version number and platform. It contains resource files for a specific language and platform to show, e.g., menus, dialogs, error messages in your language. It may contain translated Help content. Language packs are created only when also a full installation file exist. In case it is not fully localized, some elements of the UI or help topics will display in English as default language.

Where to get help when I have a problem?

If you encounter problems with installation or using OpenOffice.org, please contact us via the following ways:

  • User's mailing list *)
  • User forums (different languages)
  • Another user forums

*) Please note that all mails go to a public mailing list, not an individual person. Mails will be archived and can be accessed also by other users. To receive answers, you will need to subscribe to the mailing list before sending mails. For instructions, see: Learn more about how to use mailing lists.
