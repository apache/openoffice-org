OpenOffice.org se une a la Apache Software Foundation

El 1 de junio de 2011 Oracle anunció la donación del código de OpenOffice.org a la Apache Software Foundation.

OpenOffice.org es ahora Apache OpenOffice, un proyecto en proceso de incubación en la Apache Software Foundation.

El sitio del podling de Apache OpenOffice http://openoffice.apache.org/ fue establecido el 13 de junio de 2011.

Puede ayudarnos en esta nueva etapa de OpenOffice.org uniéndose a las listas de correo internacionales (en inglés) o las listas en español.

Declaración de la Apache Software Foundation sobre Apache OpenOffice

14 de octubre de 2011: El 1 de junio de 2011, Oracle Corporation presentó el código base de OpenOffice.org ante la Apache Software Foundation (ASF). Esta presentación fue aceptada, y el proyecto OpenOffice.org está ahora siendo desarrollado como un “podling” en el Apache Incubator bajo el proceso meritocrático de la Apache Software Foundation, informalmente denominado “The Apache Way”.

OpenOffice.org forma ahora parte de la familia Apache. El proyecto se llama Apache OpenOffice.

A través de sus 12 años de historia, la ASF ha recibido contribuciones de individuos y organizaciones por igual, pero, como política, no solicita donaciones de código. El código base de OpenOffice.org no fue buscado por la ASF antes de su aceptación en la Apache Incubator.

Oracle dona OpenOffice.org a la Apache Software Foundation

1 de junio de 2011: En un anuncio efectuado hoy, Oracle ha donado el proyeto OpenOffice.org a la Apache Software Foundation. La propuesta consiste en que OpenOffice.org se establezca como un proyecto en el Incubator de Apache.

Oracle deja de brindar su versión comercial de OpenOffice.org

15 de abril de 2011: Oracle Corporation ha anunciado hoy su intención de convertir a OpenOffice.org en un proyecto de código abierto completamente basado en la comunidad, y no ofrecer más una versión comercial de OpenOffice.org (llamada Oracle Open Office, antes conocida como StarOffice en las épocas de Sun Microsystems).
