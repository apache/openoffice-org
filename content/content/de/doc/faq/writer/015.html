015 - Gibt es in OpenOffice.org eine ähnliche Möglichkeit, die der Befehl 'Format übertragen' in MS Word bietet?

Ja. Er heißt "Gießkannenmodus" und ist im Vorlagenfenster zu finden.

Dazu wählen Sie Format-> Stylist (oder F11).

Im Vorlagenfenster klicken Sie das dritte Symbol von rechts für den "Gießkannenmodus".

Es handelt sich bei diesem Knopf um einen EIN/AUS-Schalter. Klicken Sie einmal, dann ist die Option eingeschaltet. Klicken Sie ein zweites mal auf dieses Symbol, dann wird die Option wieder ausgeschaltet.

Wenn Sie die Option eingeschaltet haben, wählen Sie eine Vorlage aus der Vorlagenliste aus, und führen dann den Gießkannencursor über die Textpassagen, die Sie gerne verändern möchten.

Denken Sie daran, die Option wieder auszuschalten.
