001 - Gibt es eine Korrekturkennung (Änderungskontrolle) in OpenOffice.org?

Aber natürlich - so gehen Sie vor:

  • Öffnen Sie die Datei, die korrigiert werden soll

  • Im Pulldown Menü wählen Sie: Bearbeiten -> Änderungen -> Aufzeichnen

Machen Sie nun Ihre Änderungen. Sie werden feststellen, dass die neuen Textpassagen, die Sie eingeben, rot und unterstrichen sind. Ein Text, den Sie löschen, bleibt trotzdem sichtbar, erscheint aber in rot und ist durchgestrichen.
