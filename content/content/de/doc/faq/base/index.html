Häufig gestellte Fragen zur Datenbank BASE

Fragen

  1. Warum erhalte ich eine Fehlermeldung "JRE ist erforderlich"?
  2. Können mehrere Anwender gleichzeitig auf BASE zugreifen?
  3. Wo erhalte ich weitere Hilfe?

Antworten

1. Warum erhalte ich eine Fehlermeldung "JRE ist erforderlich"?

Die von OpenOffice.org HSQLDB verwendete Datenbank benötigt eine Java Laufzeitumgebung (kurz JRE).
Vergewissern Sie sich, dass Sie eine JRE auf Ihrem System installiert haben. Sollte dies nicht der Fall sein, können Sie entweder OpenOffice.org als Installationspaket mit Installationsdateien für eine JRE von http://de.openoffice.org herunterladen und beides installieren oder falls Sie die aktuelle Version von OpenOffice.org schon installiert haben nur die Installationsdateien für eine JRE von http://java.com/de/ herunterladen und installieren.
Es kann auch vorkommen, dass Sie eine Java-Laufzeitumgebung zwar installiert haben, Sie sie aber noch in OpenOffice.org eintragen müssen. Gehen Sie dazu in das Menü Extras -> Optionen... -> OpenOffice.org -> Java und aktivieren Sie die Option Eine Java-Laufzeitumgebung verwenden. Das Programm sucht nun nach einer JRE und zeigt diese im Listenfeld darunter an. Dort aktivieren Sie den Optionsbutton vor einer der gefundenen JRE (wählen Sie im Zweifel die JRE mit der höchsten Versionsnummer) und schließen Sie den Dialog mit OK.

Siehe auch: Wofür benötigt OpenOffice.org Java?.

2. Können mehrere Anwender gleichzeitig auf BASE zugreifen?

Die mitgelieferte Datenbank von OpenOffice.org ist die HSQLDB. Deutschsprachige Infos dazu finden Sie aufWikipedia, die Homepage zu HSQLDB ist http://www.hsqldb.org/.

Diese eingebaute Datenbank ist eine Single-User-Datenbank. Das bedeutet, ein gleichzeitiger Zugriff auf die Datenbank ist derzeit nicht möglich

Setzen Sie ggf. eine Client-Server Datenbank, wie z. B.MySQL und Postgresql, die über ODBC bzw JDBc an OpenOffice.org angebunden werden können, als Datenbank-Backend ein.

Mehr zu OpenOffice.org BASE finden Sie unter http://de.openoffice.org/doc/howto_2_0/base oder im Abschnitt BASE der Dokumentation.

3. Wo erhalte ich weitere Hilfe?

Sollten Sie Ihr Problem mit den Hilfen auf dieser Seite nicht lösen können und benötigen Sie deshalb persönliche Hilfe, so wenden Sie sich mit Ihrer Frage am einfachsten an die Users-Mailingliste. Informationen zum An- und Abmelden erhalten Sie unter http://de.openoffice.org/probleme.html.

Möglicherweise finden Sie aber die Lösung Ihres Problems auch in der Dokumentation, in den Foren oder unter www.ooowiki.de.
