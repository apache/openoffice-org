Häufig gestellte Fragen zum Präsentationsmodul IMPRESS


Fragen

  1. Wie kann ich Musik auch über einen Folienwechsel hinaus abspielen lassen?
  2. Wie kann ich eine Präsentation auf einem fremden Rechner abspielen?
  3. Wie kann ich Listenelemente nacheinander einblenden?
  4. Wo erhalte ich weitere Hilfe?

Antworten

1. Wie kann ich Musik auch über einen Folienwechsel hinaus abspielen lassen?

Es ist erst seit Version 2.3 von OpenOffice.org möglich, bei einer Präsentation eine Musik auch über einen Folienwechsel hinaus abzuspielen. Mit früheren Versionen kann man Entsprechendes mit einem Makro erreichen, das die Funktion shell benutzt um einen externen Player aufzurufen.
Weiterlesen: Für eine detailliertere Beschreibung empfehlen wir Ihnen das OpenOffice.org-Wiki.

2. Wie kann ich eine Präsentation auf einem fremden Rechner abspielen?

Sie haben in OpenOffice.org (OOo) IMPRESS eine Präsentation erstellt und möchten nicht Ihr eigenes Notebook für die Vorführung verwenden. Auf dem Rechner, auf dem die Präsentation laufen soll, ist kein OpenOffice.org installiert.
Sie haben folgende Möglichkeiten:

  • Sorgen Sie dafür, dass auf dem Zielsystem OpenOffice.org installiert wird.
  • Verwenden Sie OpenOffice.org Portable, das Sie auf einem USB-Stick installieren und direkt davon starten können.
  • Speichern Sie Ihre Präsentation mit OpenOffice.org im PDF- oder Flash-Format (Datei -> Exportieren); auf dem Zielsystem muss ein entsprechender Viewer/Player zur Verfügung stehen (z.B. Acrobat Reader).
  • Ist auf dem Zielsystem Microsoft PowerPoint installiert, speichern Sie Ihre Präsentation mit OpenOffice.org im PPT-Format (Datei -> Speichern unter, Dateityp: Microsoft PowerPoint).

"Tipp:" Wenn Sie sich zum Export in das PPT-Format entschließen, überprüfen Sie das Ergebnis sicherheitshalber mit dem Microsoft PowerPoint-Viewer. Speichern Sie auf jeden Fall für sich selbst eine Kopie im OpenDocument-Format (ODP).

3. Wie kann ich Listenelemente nacheinander einblenden?

Sie möchten die einzelnen Punkte einer Liste jeweils mit einem Mausklick einzeln "aufdecken" und/oder Text "einfliegen" lassen.

  • Erstellen Sie zunächst die Folie mit der kompletten Liste bzw. erfassen Sie den gesamten Text für diese Folie.
  • Wählen Sie den Aufgabenbereich (Ansicht -> Aufgabenbereich) "Benutzerdefinierte Animation".
  • Markieren Sie den ersten Punkt Ihrer Liste und klicken Sie im Aufgabenbereich"Benutzerdefinierte Animation" auf die Schaltfläche Hinzufügen.
  • In der sich öffnenden Palette Benutzerdefinierte Animation wählen Sie einen Effekt aus und klicken auf OK.
  • Anschließend markieren Sie den zweiten Punkt Ihrer Liste, klicken wieder auf Hinzufügen usw. bis die Liste komplett animiert wurde.

Beim Klicken ist unter Starten voreingestellt und bedeutet, dass die Animationen jeweils nach einem Klick (oder dem Betätigen der Leertaste) ausgeführt werden.
"Tipp:" Zu viele und aufwendige Effekte können eine Präsentation leicht unprofessionell wirken lassen.

4. Wo erhalte ich weitere Hilfe?

Sollten Sie Ihr Problem mit den Hilfen auf dieser Seite nicht lösen können und benötigen Sie deshalb persönliche Hilfe, so wenden Sie sich mit Ihrer Frage am einfachsten an die Users-Mailingliste. Informationen zum An- und Abmelden erhalten Sie unter http://de.openoffice.org/probleme.html.
Möglicherweise finden Sie aber die Lösung Ihres Problems auch in der Dokumentation, in den Foren oder unter www.ooowiki.de.
