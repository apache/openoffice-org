Ist Formel kompatibel mit dem Formeleditor von Microsoft Word?

Formel besitzt Import- und Export-Filter für MathType. (Der Formeleditor von Microsoft Word ist eine limitierte Version von MathType.)

Wenn Sie von einem Text-Dokument nach Microsoft Word speichern, konvertiert OpenOffice.org  eingebettete Formel Formeln nach MathType.

Beim Import von Microsoft Word in ein Text-Dokument macht OpenOffice.org die umgekehrte Umwandlung.

Wie bei jeder Dateitypenumwandlung sollten Sie sich bewusst sein, dass Formel und MathType verschiedene Funktionen haben und es daher Unterschiede im Ergebnis geben kann.

Wenn Sie es bevorzugen, dass die Formel im originalen Format bleibt, können Sie die Konvertierungsoptionen ändern:

  • Im Menü wählen Sie Extras-Optionen aus.
  • Dann die Registerkarte Laden/Speichern-Microsoft Office.
  • Entfernen Sie das Häkchen bei MathType nach OpenOffice.org Math / OpenOffice.org Math nach MathType.
