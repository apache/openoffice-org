Wie kann ich eine linke Klammerl ohne eine schließende lrechte Klammer leingeben?

Da Klammern immer eine gruppierende Funktion haben, müssen sie immer paarweise zusammengehören.

Um eine einzige Klammer ohne dazugehörigem Partner schreiben zu können, müssen Sie das Klammernzeichen mit einem vorangestellten "\" Zeichen übergehen. Zum Beispiel bekommt man eine "["-Klammer ohne korrespondierende "]"-Klammer, indem Sie \[ eingeben.

Folglich, wenn Sie das Intervall [3;8) eingeben möchten, müssen Sie \[3 ";" 8\) tippen. Die Anführungszeichen um das ";" sind wichtig, damit es nicht kursiv geschrieben wird. (Hier haben Sie recht, es hat nichts mit Klammern zu tun)

Wenn Sie skalierte Klammern schreiben möchten, ist es notwendig, dass left und right zusammenpasst, weil der Ausdruck, den die Klammern umschreiben sollen, wohldefiniert sein muss.

In diesem Fall müssen Sie mit Hilfe des Schlüsselwortes none eine unsichtbare Klammer erzeugen; somit erhalten Sie nur eine Klammer. Zum Beispiel erhalten Sie eine skalierte "["-Klammer ohne dazugehöriger "]"-Klammer, wenn Sie left [ ... right none eingeben.

Folglich, wenn Sieschreiben möchten, müssen Sie x=left lbrace stack { -5 # 3 } right none eingeben.
