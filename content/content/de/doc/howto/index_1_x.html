Dokumentationen für Version 1.0 und 1.1

Auf dieser Seite finden Sie die für die Version 1.0 und 1.1 von OpenOffice.org erstellten Dokumentationen. Falls Sie Vorschläge für weitere Dokumente zu dieser Version der Office-Suite haben oder ein neues bereit stellen wollen, teilen Sie uns das bitte mit. Nehmen Sie hierzu einfach über unsere Mailing-Liste Kontakt zu uns auf.
Die Dokumente "Erste Schritte" behandeln die grundlegenden Funktionen von OpenOffice.org. Es wird empfohlen, diese Schritt für Schritt durchzugehen, um über den Funktionsumfang der einzelnen Pakete informiert zu werden. Die qualitätsgeprüften Dokumentationen sind umfangreicher und daher nicht für den schnellen Einstieg sondern für die Vertiefung geeignet:

Installation

Hier finden Sie Beschreibungen zur Installation unter verschiedenen Bedingungen: Ein ausführliches Installationshandbuch, das sowohl die Systemvoraussetzungen wie auch die Einzelplatz- und die Netzwerkinstallation, die Einrichtung von Drucker, das Versenden von Faxen aus OpenOffice.org sowie die Installation von Schriften erläutert.
