Autoformat

Über den Menübefehl »Format - AutoFormat« oder das Symbol »AutoFormat« wird der Dialog »AutoFormat« geöffnet. In dem Dialog kann einem zuvor ausgewählten Tabellenbereich eine Formatierung zugewiesen werden.

Es müssen mindestens 3 Zeilen und 3 Spalten markiert werden, damit der Menüpunkt zur Verfügung steht.


Calc 77: Dialog »Autoformat«.

Mit der Schaltfläche »Hinzufügen« kann ein Bereich von mindestens 4 x 4 Zellen als neues »AutoFormat« aufgenommen werden.


Calc 78: Dialog »AutoFormat hinzufügen«.

Der Dialog erlaubt die Vergabe eines Namens für das »AutoFormat«.
