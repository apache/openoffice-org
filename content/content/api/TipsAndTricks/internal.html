OpenOffice Internal Spots

The following is a list of projects or documents which are in a close relation to the API project and provides additionally useful information.

  • UNO Development Kit Project
    This project provides information about UNO as the fundamental base for the OpenOffice API.
  • Scripting Project
    This project provides information about the upcoming scripting framework.
  • OpenOffice Wiki
    The OpenOffice Wiki provides a lot of useful information for developers and can probably be seen as the most important place for developers.
  • Extensions Project
    The OpenOffice Extensions project with focus on extensions development and an extensions repository.
  • OpenOffice for Developers
    This provides an useful entrance point for all who are interested in developing for OpenOffice or with the OpenOffice API.
  • OpenOffice Community Forum
    The forum provides a lot of information for OpenOffice users and developers.

Anything missing here? Write us!
