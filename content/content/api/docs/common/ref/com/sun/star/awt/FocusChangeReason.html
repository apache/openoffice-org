:: com :: sun :: star :: awt ::

constants group FocusChangeReason
Description
A combination of these values can be used to specify the reason for a focus change.
Constants
TAB Focus changed because TAB was pressed.  
CURSOR Focus changed because KeyLeft/Right/Up/Down was pressed.  
MNEMONIC Focus changed because mnemonic key was pressed.  
FORWARD Changed Focus to the next control.  
BACKWARD Changed Focus to the previous control.  
AROUND Changed Focus forward from last to first or backward from first to last.  
UNIQUEMNEMONIC Focus changed because mnemonic key was pressed and this mnemonic is unique.
