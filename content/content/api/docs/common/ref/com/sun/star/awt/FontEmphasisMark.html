:: com :: sun :: star :: awt ::

constants group FontEmphasisMark
Description
These values are used to specify the kind of emphasis mark.

They may be expanded in future versions.
Constants
NONE specifies no emphasis mark.  
DOT specifies emphasis mark dot.  
CIRCLE specifies emphasis mark circle.  
DISC specifies emphasis mark disc.  
ACCENT specifies emphasis mark accent.  
ABOVE specifies that the emphasis mark should be positioned above the characters.  
BELOW specifies that the emphasis mark should be positioned below the characters.
