The following values apply for a status bar item:

  • Alignment
    • ItemStyle::ALIGN_LEFT
    • ItemStyle::ALIGN_CENTER
    • ItemStyle::ALIGN_RIGHT
  • Drawing
    • ItemStyle::DRAW_OUT3D
    • ItemStyle::DRAW_IN3D
    • ItemStyle::DRAW_FLAT
  • ItemStyle::AUTO_SIZE
  • ItemStyle::OWNER_DRAW
