Mac OS X Quicktime Mac OS X natively uses the Quicktime player to play music and sound. The players currently integrated into OpenOffice.org do not function on Mc OS X. This project integrates Quicktime support. Mike Sicotte

 Preliminary - November 5, 2007
Fondu-Integration OpenOffice.org does not support the Apple native fonts. To work around this problem, this feature extracts Apple's native fonts and converts them into truetype fonts (.ttf). This feature will be replaced in future OpenOffice.org version with native Apple truetype font server. Eric Bachard Standard 2.03 (X11 version only) April 27,2006
