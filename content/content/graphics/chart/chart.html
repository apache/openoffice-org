    • Want to help?

    You can help a lot with identifying problems and submitting issues! Download one of the latest developer builds and give it a try. When you find a bug please check whether someone else already did submit an issue for that problem. The following list of open chart issues will help you with the research: Open Chart Issues . If the problem is unknown you are welcome to submit a new issue. Please only describe one problem per issue. Thanks a lot for your help!

    For further involvement visit the Chart Wiki. We are looking forward to your participation!
