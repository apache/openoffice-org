Por que participar do projeto OpenOffice?

Há vários motivos pelos quais voluntários participam de um projeto de código aberto como o OpenOffice. Os mais comuns são:

  • Ajudar a reduzir a “divisão digital” assegurando o acesso contínuo a aplicações de produtividade livres e de alta qualidade por todo o mundo.
  • Ajudar a trazer os benefícios do OpenOffice para seu país ou região.
  • Para “retribuir” ao código aberto.
  • Para aprender ou praticar uma nova habilidade.
  • Para melhorar o currículo ou portifólio, demonstrando o trabalho feito neste projeto.
  • Para ajudar um cliente ou parceiro comercial que trabalhe com o OpenOffice.
  • Para desenvolver credenciais para trabalhos de consultoria relacionados ao OpenOffice.
  • Para trazer algo para o OpenOffice que você sempre quis que ele tivesse.
  • Para se divertir, trabalhando em uma equipe internacional de voluntários unida que desenvolve o mais antigo e mais famoso projeto de código aberto existente.

Um projeto de código aberto multidisciplinar

Não é preciso ser um programador para contribuir para o Apache OpenOffice. Nosso projeto envolve uma grande variedade de diferentes habilidades, níveis de envolvimento e graus de experiência técnica. Portanto, se você deseja participar do Apache OpenOffice, há quase certamente um lugar para você.

Procuramos pessoas para:

  • Escrever ou atualizar a documentação
  • Ajudar no suporte a novos usuários
  • Testar o programa e relatar problemas
  • Consertar defeitos no programa
  • Desenvolver novas funcionalidades
  • Produzir arte e outros materiais de marketing
  • Fazer traduções para os mais diferentes idiomas
  • Manter o portal e desenvolver novas matérias
  • Promover o OpenOffice e ajudar a explicar seus benefícios
  • qualquer coisa que você perceba ser necessária

Todas essas contribuições ajudam a manter o projeto ativo e reforça a comunidade. Nós damos as boas vindas e incentivamos a participação, e tentamos facilitar tanto quanto possível para que as pessoas participem.

Como começar

O OpenOffice é um projeto de código aberto grande e complexo, com uma cultura desenvolvida durante muitos anos como um projeto independente, combinado com nossa experiência na Apache.

Se você conhece o estilo Apache de desenvolvimento de código aberto então você deve entrar nessa. Inscreva-se nas listas de um dos projetos e apresente-se. Nós o ajudaremos a iniciar as atividades.

Se, por outro lado, você é novo na Apache, ou prefere uma abordagem mais metódica para começar, temos um conjunto de Módulos de Orientações para os Novos Voluntários desenvolvido para ajudá-lo a passar por vários assuntos básicos relacionados ao Apache OpenOffice e como fazemos as coisas.

A escolha do caminho que você deseja seguir é sua e é uma questão de preferência pessoal. Você pode, até mesmo, seguir os dois caminhos.

Nós também mantemos uma lista de áreas específicas onde precisamos de voluntários na nossa página Procura-se Ajuda no wiki.
