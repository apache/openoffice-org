Nola alda dezaket azpiformula baten tamaina?

Tamaina aldatzeko komandoaren bost aldaera daude:

  • size 20 a a 20 pt-ko tamainan idazteko.
  • size +5 a a-ren uneko tamaina 5pt gehiago handitzeko.
  • size -5 a a-ren uneko tamaina 5pt txikitzeko.
  • size *1.5 a a-ren uneko tamaina bider 1,5 egiteko.
  • size /1.5 a a-ren uneko tamaina zati 1,5 egiteko.

Azpiformula oso bateko letra-tamaina alda dezakezu, formula { } edo bestelako parentesiak erabiliz taldekatzen baduzu. Adibidea:

  • Sartu a b size *2 {c d} e cde idazteko.

Letra-tamainaren hainbat aldaketa aplikatzean, barru-barruan dagoenak lehentasuna du adibidean ageri den bezala:

  • Sartu size *1.5 {a size 12 b size *1.5 c d} abcd idazteko.

Math-en erabilitako oinarriaren letra-tamaina aldatzeko, ikus FAQ #008.
