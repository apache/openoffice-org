Apache Software Foundation vetää nykyisin www.openoffice.orgia

Sivustomme on siirretty Apache Software Foundation -säätiön palvelimelle. Liity käyttäjien postituslistalle tai käyttäjien keskustelupalstoille, jos sinulla on kysyttävää tai kommentteja.

Pysy Apache OpenOfficen tasalla

Pysy ajan tasalla OpenOffice.orgin kehityksessä ja Apache OpenOffice -projektissa lukemalla Apache OpenOffice.org blogia tai tilaamalla Announce-postituslistan.

OpenOffice.org siirtyi Apachen hautomoon

Kesäkuun 1. 2011, Oracle ilmoitti lahjoittavansa OpenOffice.org-koodin The Apache Software Foundationin Incubator-hautomoon. Apache OpenOfficen "kasvatuksen" sivusto http://openoffice.apache.org avattiin kesäkuun 13. 2011. Voit liittyä tähän uuteen OpenOffice.orgin kehitysvaiheeseen tilaamalla postituslistojamme.
