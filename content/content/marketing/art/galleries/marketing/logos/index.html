OpenOffice.org Official Logos have been moved to this page


This page is outdated since the release of OpenOffice.org 3.0.
You will be redirected to the new gallery page in about 5 seconds. If redirection doesn't work, please follow this link manually.

These are the official OpenOffice.org logos to be used as references to both the product and the open source community behind the product.

Please take care that you download the right file for your purpose! There are several different file formats and three color models. As changing the color model causes modifications to the colors you should use the model you need at the end right form the beginning. Additional information can be found in the OpenOffice.org Artwork Style Guide .

Logos for OpenOffice.org 3.0 have not been added to this page yet, but they can already be found in our files repository.
