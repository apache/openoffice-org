	<strong>How to submit files to the project:</strong>
	</TD>
	</TR>
	<TR>
	<TD class="tableb">

<p><b>You have graphics or files you want to send to us ?</b>
<BR>
Posting the files on the project's mailing list it is <b>not recommended</b> 
because some members still use slow dial-up connections.<br><br>
If you have created artwork you want to show to us please follow the 
steps below in order to send us the files. 

<p><b>This method is the recommended way of submitting files to the Art Project:</b>
<ol>
<li> First you have to be a registered user of the OpenOffice.org website. For registration please visit: 
	<a href="http://www.openoffice.org/servlets/Join">http://www.openoffice.org/servlets/Join</a>;
<li> If you are registered, log-in to the website by visiting: 
	<a href="http://www.openoffice.org/servlets/TLogin">http://www.openoffice.org/servlets/TLogin</a>;
<li> LOCATE THE ISSUE:<br> 
	If there is already an existing issue on the topic for which you wish to submit your work (please 
	ask about it's existence on the project's mailing list <a href="mailto:art@marketing.openoffice.org">
	art@marketing.openoffice.org</a>):
	<ul>
	<li>if you know the issue's address (URL) go to that address;
	<li>if you know only the issue number, go to <a href="http://marketing.openoffice.org/servlets/ProjectIssues">
	http://marketing.openoffice.org/servlets/ProjectIssues</a> and in the "Query database" section enter the 
	issue number and press the Find button;
	</ul>
<li> OR CREATE A NEW ISSUE: <br> 
	If there is no issue created, please use the instructions in our 
	<a href="http://marketing.openoffice.org/art/about-issuezilla.html">IssueZilla Tutorial</a>
	on how to create an issue for the Art Project. Once the issue created proceed to the next step.
<li> ADDING THE FILE: <br>
	On the issue page, go to the <b>Attachments</b> section and click on the 
	"<b>Create a new attachment (proposed patch, testcase, etc.)"</b> link and:
	<ol>
	<li> hit Browse button and select your local file that you wish to submit
	<li> provide one line description for the file
	<li> select the correct file format/mime type
	<li> hit the submit button and a confirmation that your attachment was added will appear
	</ol>
<li> TELLING US ABOUT THE FILE:<br>
	After you have attached the file you can go back to the issue and in the "Attachments" section you will see 
	a link to the file you have added. You can copy that link location (in some browsers: Right-Click 
	and Copy Link Location) and paste the link in your emails to the mailing list. 
	It is also recommended that in your message you also mention the Issue Number and/or the Issue URL 
	in order for us to easy locate the right issue.
<li> ASKING FOR HELP:<br>
	If you have difficulties, please do not hesitate to ask for help on the project's mailing list.
</ol>
