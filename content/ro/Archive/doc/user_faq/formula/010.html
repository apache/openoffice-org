Cum schimb mărimea unei subformule?

Existăa cinci variante ale comenzii de schimbare a dimensiunii:

  • size 20 a afişează un a cu corpul de literă de 20 pt.

  • size +5 a afişează un a de dimensiunea curentă a corpului de literă plus 5pt.

  • size -5 a afişează un a de dimensiunea curentă a corpului de literă minus 5pt.

  • size *1.5 a afişează un a de 1.5 ori dimensiunea curentă a corpului de literă.

  • size /1.5 a afişează un a de 1.5 ori mai mic decât dimensiunea curentă a corpului de literă.

Puteţi schimba marimea corpului de literă a unei subformule întregi dacă o grupaţi cu { } sau alt tip de paranteze. Examplu:

  • Scrieţi a b size *2 {c d} e pentru a obţine abcde.

Când se aplică simulta mai multe schimbări de dimensiune a corpului de literă, cel mai din interior este prioritar după cum se vede în acest examplu:

  • Scrieţi size *1.5 {a size 12 b size *1.5 c d} pentru a obţine abcd.
