การใช้ CVS บน Windows

ติดตั้ง Cygwin

  • พูดโดยคร่าวๆ แล้ว Cygwin เป็นชุดโปรแกรม ที่ทำให้เราใช้คำสั่งของระบบปฏิบัติการ Unix บนระบบปฏิบัติการ Windows ได้
    ในขั้นตอนการพัฒนา OO.o บน Windows นั้น เราจำเป็นต้องใช้ชุดโปรแกรม Cygwin
  • ดาวน์โหลด และดูวิธีติดตั้ง ได้ที่ http://www.cygwin.com/
    ในขั้นตอนการติดตั้ง Cygwin ให้แน่ใจว่า ได้เลือกติดตั้งแพคเกจเหล่านี้*
    • cvs (Category: Devel)
    • openssh (Category: Net)
    • openssl (Category: Libs)
    • patch (Category: Utils)
    * สำหรับผู้ที่ต้องการสร้างโปรแกรม (build) OO.o ด้วยตัวเอง จะต้องติดตั้งแพคเก็จมากกว่านี้ ดูรายละเอียดได้ที่ [วิธีการสร้าง]

โดยทั่วไปในการส่งโค้ดที่แก้ไขแล้วนั้น Contributor จะต้องส่งผ่านมาทาง Developer เพื่อให้ Developer เป็นผู้ส่งเข้า CVS ให้ เนื่องจาก Contributor จะไม่สิทธิเขียน/แก้ไข CVS ด้วยตัวเอง

นักพัฒนาสามารถขอสิทธิ ในการเขียน/แก้ไข CVS ด้วยตัวเองได้ [ดูวิธี]
