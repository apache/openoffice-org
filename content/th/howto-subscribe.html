การเข้าร่วมกลุ่มผู้ใช้ภาษาไทยของโอเพนออฟฟิศอ็อก

ขั้นตอน

  1. ต้องเป็นสมาชิกของ OpenOffice.org เสียก่อน, ถ้าเป็นสมาชิกแล้ว ให้ข้ามไปข้อถัดไป
    • การสมัครทำได้ง่ายมาก, ดูที่เมนูด้านซ้ายมือ เลือก Join
    • กรอกชื่อล็อกอิน (Login name) ที่ต้องการ กับ อีเมล (Email) ที่จะใช้ติดต่อกับ OO.o (คนอื่นๆ จะไม่เห็นอีเมลนี้), แล้วกดปุ่ม Register
    • หลังจากนั้น เราจะได้รับรหัสผ่าน (Password) ที่อีเมลที่ได้แจ้งไป (ข้อแนะนำ: ควรเปลี่ยนรหัสผ่านทันทีในการล็อกอินครั้งแรก)
  2. ตอนนี้เราเป็นสมาชิกแล้ว, ให้ล็อกอินเข้าใช้งาน
    • เมนูด้านซ้ายมือ กดที่ Login
    • กรอกชื่อล็อกอิน หรืออีเมล ที่ช่องแรก, กรอกรหัสผ่านที่ช่องที่สอง, แล้วกดปุ่ม Login
  3. สมัครเข้าร่วมกลุ่มผู้ใช้ภาษาไทย
    • ไปที่หน้าเวบของกลุ่ม, ที่ http://th.openoffice.org
    • เลือกเข้าร่วมกลุ่ม, ดูด้านมุมบนขวาของหน้า เลือก Join This Project
    • จากนั้น จะมีข้อความขึ้นมาว่า 
      You have no roles in the th project. Use the form below to request a role.
Specify the role you want to request in the th project.
      หมายถึง ให้เราเลือกหน้าที่ความรับผิดชอบ (role) ที่ต้องการสำหรับโครงการนี้. ในครั้งแรกจะมีเพียง role เดียวให้เลือก คือ Observer (ผู้สังเกตการณ์ = อนุญาตให้เข้าไปดูโค้ดของโครงการได้ และตั้งประเด็นใหม่ในโครงการได้) ก็ให้เลือก Observer แล้วกดปุ่ม Submit Request
      หมายเหตุุ: หลังจากเข้าร่วมกลุ่มแล้ว สมาชิกสามารถร้องขอความรับผิดชอบใหม่ได้ภายหลัง กับผู้ดูแลโครงการ. รายละเอียดความรับผิดชอบต่างๆ
    • หลังจากส่งคำร้องไปแล้ว ทางผู้ดูแลโครงการ จะได้รับคำร้องและดำเนินการต่อให้ (โดยปกติน่าจะดำเนินการให้ได้ภายในหนึ่งสัปดาห์ทำการ -- หากไม่ได้รับการตอบรับภายในสองสัปดาห์ กรุณาแจ้งเตือนทันที)
  4. ตรวจสถานะของตนเองได้ ในรายชื่อสมาชิกของกลุ่ม, ที่มุมบนขวาของหน้า เลือก Member List

เอกสารที่เกี่ยวข้อง

  • งานในกลุ่ม th.openoffice.org
  • mailing list
  • การขอสิทธิแก้ไข CVS
