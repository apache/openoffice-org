குறுந்தகட்டில் தமிழ் ஓப்பன்ஆபீஸ் மென்பொருளை அளிக்க விரும்புவோர் தகவலுக்கு...

ஓப்பன் ஆபீஸ் நிரலை குறுந்தட்டில் வழங்குவதற்குத் தொடர்பான CD-ROM Project ஐ இங்கே பார்க்கவும் . சொல்திருத்திக் கோப்புகள் (Spell checkers) GPL என்ற திறவூற்று உரிமத்தின் (GPL) கீழ் வெளியிடப்பட்டுள்ளதால் அவற்றையும் வினியோகிக்க கேட்டுக்கொள்ளப்படுகிறது. குறுந்தகட்டில் வழங்கும்போது readme கோப்புகளும் இருக்குமாறு பார்த்துக்கொள்ளவும்

உதாரணத்திற்குக் ஓப்பன் ஆபீஸ் குறுந்தட்டுகள் கீழ்காணும் அம்சங்களைப் பெற்றிருக்கவேண்டும்.

  • விண்டோஸ்(x86) மற்றும் லினக்ஸுக்கான(x86) புதுப்பிக்கப்பட்ட தமிழ்-ஓப்பன்ஆபீஸ் செயலிக்கோப்புகள்.
  • விண்டோஸ்(x86) மற்றும் லினக்ஸுக்கான(x86) ஆங்கிலமொழி ஓப்பன் ஆபீஸ் செயலிக் கோப்புகள்
  • ஓப்பன் ஆபீஸ் மூலநிரலகள்
  • GPL, LGPL மற்றும் SISSL உரிமங்களின் பிரதிகள்.
  • சொல்திருத்திக் கோப்புகள், readme கோப்புகளுடன்

யாரேனும் தமிழ்-ஓப்பன்ஆபீஸ் செயலியை குறுந்தகட்டில் வெளியிட விரும்புகிறீர்கள் என்றால் எங்களை mugunth@openoffice.org அல்லது evolution@openoffice.org ஆகிய மின்னஞ்சல் முகவரிகளில் தொடர்புகொள்ளவும்.
இத்தளத்தில் உங்கள் தகவல் இடம்பெற கண்டிப்பாக, உங்கள் முகவரி, மின்னஞ்சல் போன்ற தகவல்களை கொடுக்கவேண்டும்.
