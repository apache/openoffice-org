OpenOffice.org in Your Language

OpenOffice.org is a world-wide project. As such, it is important that it be available to as many linguistic communities as possible. There are two projects that seek to spread the use of OpenOffice.org over the various world languages. OpenOffice.org now supports almost all major languages, and support for many medium/minor languages has started. But help is always needed

  • Localization (L10n)
    The Localization project's task is to translate the OpenOffice.org interface into other languages. Thanks to this project, OpenOffice.org is now available in several dozen different languages.

  • Native-lang Confederation (NLC)
    The Native Lang Confederation provides a space for support, information, advocacy, and some development of OpenOffice.org in a person's native language. The constituent projects work closely with the l10n Project to ensure that OpenOffice.org is supported in all the languages to which it has been translated ("localized").
