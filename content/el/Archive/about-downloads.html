Σύντομες οδηγίες Εγκατάστασης

Linux

deb:

1. Απεγκαταστήστε την παλιά έκδοση που έχετε στο σύστημα σας
2. Αποσυμπιέστε το OOo_xxxLinuxIntel_install_el_deb.tar.gz
3. cd στο κατάλογο που αποσυμπιέσατε τα αρχεία (DEBS)
4. sudo dpkg --install --force-overwrite *.deb
5. Για να μπουν οι συντομεύσεις στο μενού εγκαταστήστε το .deb που υπάρχει στον κατάλογο /desktop-integration

rpm:

1. Αποσυμπιέστε το OOo_xxxLinuxIntel_install_el.tar.gz
2. cd στο κατάλογο που αποσυμπιέσατε τα αρχεία (RPMS)
3. su & rpm -Uvih *rpm
4. Για να μπουν οι συντομεύσεις στο μενού εγκαταστήστε το αντίστοιχο για την διανομή σας αρχείο που υπάρχει στον κατάλογο /desktop-integration

* Αν οι γραμματοσειρές στο μενού του OpenOffice δεν φαίνονται σωστά- δοκιμάστε να αλλάξετε την κλιμάκωση απο το Εργαλεία > Επιλογές> OpenOffice.org > Προβολή

Οδηγίες εγκατάστασης σε Ubuntu με τη χρήση του Synaptic package manager

Windows

1. Απεγκαταστήστε την παλιά έκδοση που έχετε στο σύστημα σας
2. Εκτελέστε το αρχείο .exe και ακολουθήστε τις οδηγίες του προγράμματος εγκατάστασης

Mac OS X

Ανοίξτε το αρχείο .dmg και αντιγράψτε το αρχείο Openoffice.org στο φάκελο των εφαρμογών.
Αν υπάρχει παλιότερη έκδοση θα ερωτηθείτε αν θέλετε να την αντικαταστήσετε.
