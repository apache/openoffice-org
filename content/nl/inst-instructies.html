Installatiehandleiding

Er is een uitgebreide, gellustreerde Nederlandstalige installatiehandleiding (Windows, Linux, Solaris) beschikbaar; deze is te vinden bij de documenten van nl.openoffice.org. De vertaling is verzorgd door Arthur Buijs.

Ook is de installatiehandleiding voor Windows van de nl.openoffice.org CD-ROM hier te vinden. Deze is ook geschikt voor beginnende computergebruikers, maar behandelt alleen de installatie voor één gebruiker.

Installatie onder Windows

- Lee Corbin (vertaling, bewerking en uitbreiding: Simon Brouwer)

De Windows versie van OpenOffice.org 1.1 is beschikbaar als een .zip bestand. De recentere versies van Windows ondersteunen Gecomprimeerde Mappen - als dit het geval is, wordt het bestand dat je hebt gedownload getoond als een map met een rits (Engels: zipper) er door heen. Als dit niet zo is, moet je eerst een apart uitpakprogramma hebben geïnstalleerd. Een bekende is het (niet gratis) WinZip. Het gratis programma UltimateZip is echter aan te bevelen als uitstekend alternatief.

Hoe je OpenOffice.org het beste kunt installeren, hangt ervan af of je op het systeem de enige bent die het programma gaat gebruiken, of dat je het beschikbaar wil maken voor meerdere gebruikers.

Installeren voor één gebruiker
Je installeert OpenOffice.org in de volgende 4 stappen:

  1. Pak het gedownloade bestand uit, door er met de rechtermuisknop op te klikken. In het pop-up menu klik je vervolgens op (afhankelijk van je uitpakprogramma) Alles Uitpakken..., Extract To Folder..., of iets dergelijks.
  2. In de map waarin het bestand uitgepakt is, vind je een map genaamd "install". Hiering staan de eigenlijke installatiebestanden van OpenOffice.org waaronder het bestand setup.exe. Dubbelklik daarop om de installatie te starten.
  3. Volg daarna eenvoudig de aanwijzingen op het scherm, om OpenOffice.org 1.1 te installeren.

Installeren voor meerdere gebruikers
Als er meerdere gebruikers zijn ingesteld op je systeem, dan moet iedere gebruiker die OpenOffice.org 1.1 wil gebruiken, het pakket apart installeren. Normaal gesproken neemt dit veel schijfruimte (ca. 145 MB per gebruiker) in beslag. De benedenstaande manier van installeren kost één keer 145 MB, en per gebruiker slechts enkele megabytes extra.

  1. Pak het gedownloade bestand uit, door er met de rechtermuisknop op te klikken. In het pop-up menu klik je vervolgens op (afhankelijk van je uitpakprogramma) Alles Uitpakken..., Extract To Folder..., of iets dergelijks. In de map waarin de uitgepakte bestanden geplaatst worden, vind je een map genaamd "install".
  2. Voor deze stap kun je é&eacuten van de volgende twee manieren kiezen.
    • Met de Opdrachtprompt: Gebruik in een Opdrachtprompt het cd commando om naar de map "install" te gaan, en geef vervolgens het commando setup /net.
    • Geen ervaring met de Opdrachtprompt? Je kunt ook op het bestandje setupnet.bat dubbelklikken.
  3. Volg daarna de aanwijzingen op het scherm. Maak een notitie van de map waarin OpenOffice.org 1.1 wordt geïnstalleerd (bijvoorbeeld C:\Program Files\OpenOffice.org 1.1).

    De volgende stappen moeten worden uitgevoerd door iedere gebruiker van het systeem die OpenOffice.org wil gebruiken.

  4. Na je te hebben aangemeld op het systeem, dubbelklikken op OpenOffice.org setup in de map waarvan de locatie in stap 3 hierboven is genoteerd.
  5. Volg de aanwijzingen van het setup programma. Kies voor "Installatie Werkstation". Kies een map voor de setupbestanden in je "eigen" submap. (voorbeeld: C:\Documents and Settings\gebruikersnaam\OpenOffice.org 1.1).

Installatie onder Linux

-Martijn Dekkers (vertaling: Simon Brouwer)

Laatste wijziging: 5 mei 2002


OpenOffice.org installeren op Linux is heel gemakkelijk! Om zeker te zijn van een optimale configuratie, moet je eenvoudigweg deze stappen volgen.

Eerste vereisten: als je integratie met Java wenst, zorg er dan voor dat je de nieuwste JRE (Java Runtime Environment) hebt geïnstalleerd. Het moet tenminste JRE 1.3 zijn. Je kunt de JRE voor Linux vinden op http://www.blackdown.org/java-linux/mirrors.html - of het zou ook op de installatiedisk van je Linuxdistributie kunnen staan.

  1. Zorg ervoor dat je als root bent aangemeld

  2. Download het .tar.gz bestand met de OpenOffice.org software, en pak het uit in een tijdelijke directory. "/tmp" is bijvoorbeeld een goede keus. Als voorbeeld ga ik er vanuit, dat je het .tar.gz bestand in de /tmp directory hebt staan. Open een terminalvenster, zoals xterm of konsole. Om het .tar.gz bestand uit te pakken, ga naar de /tmp directory: cd /tmp en pak het bestand uit: "tar -zxvf [tarball name]" (voor [tarball name] de naam van het .tar.gz bestand invullen).

  3. Dit maakt een directory "install" aan. Ga naar deze directory: "cd install."

  4. Voor setup uit, met het volgende commando: "./setup -net". Dit commando begint de eerste stap van het installeren onder Linux.

  5. Volg de aanwijzingen van het installatieprogramma. Wanneer je wordt gevraagd waar je de OpenOffice program files wilt plaatsen, is een goede keuze "/opt/OpenOffice.org[version number]". Bijvoorbeeld, als je OpenOffice.org 1.0 installeert, "/opt/OpenOffice.org1.0".
    (Opmerking redactie: je kunt in plaats van /opt ook /usr/local invullen; /opt wordt echter aanbevolen in de Filesystem Hierarchy Standard).

  6. Wanneer het installatieprogramma klaar is, moet je de gebruikersinstallatie van OpenOffice.org uitvoeren. Om dit te doen, ga je eerst naar de programmadirectory waar je OpenOffice.org in hebt geïnstalleerd: cd /opt/OpenOffice.org1.0.

  7. Voer vanuit deze directory het volgende commando uit: "./setup (nu dus zonder -net)".

  8. Het gebruikersdeel van de installatie wordt nu uitgevoerd. Geef aan dat het een normale installatie moet doen (dit kopieert ongeveer 1.4 MB aan bestanden naar je home directory) en laat het de bestanden in je gebruikersdirectory binnen de OpenOffice.org1.0 directory plaatsen (bijvoorbeeld: "/home/billg/OpenOffice.org1.0"). Volg de aanwijzingen op het scherm. Wanneer setup vraagt waar de bestanden moeten worden geplaatst, vindt het de JRE meestal zelf al, maar in sommige gevallen moet je de padnaam ervan invoeren (tip: je kunt hier ook achter komen door in een shell het commando whereis java in te typen)

  9. Dat is alles! Als je GNOME of KDE gebruikt, zul je zien dat OpenOffice.org 1.0 volledig in je omgeving is ingepast. Wanneer je een andere Windowmanager gebruikt, kun je OpenOffice.org 1.0 starten door ~/OpenOffice.org1.0/soffice te typen.

  10. Wanneer je klaar bent met installeren, kun je de installatiebestanden in /tmp verwijderen.

Veel plezier!
