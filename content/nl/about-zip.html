OpenOffice installeren onder Windows met een apart uitpakprogramma


(oude pagina. Als je hierop terechtkomt laat het dan even weten: simon.oo.o@xs4all.nl)
pagina laatst gewijzigd op 7 mei 2002

Voor het geval je een Windowsversie hebt die geen Gecomprimeerde Mappen ondersteunt, en je hebt ook nog geen apart programma geïnstalleerd dat een .zip bestand kan uitpakken, volgt hier een beschrijving gebaseerd op het gratis en open-source uitpakprogrammaatje unzip.exe van info-zip.

(Opmerking: als je toch al een uitgebreider in- en uitpakprogramma wilde installeren, kun je kijken naar het bekende, maar niet gratis Winzip. Het gratis, hoewel geen open-source, programma UltimateZip is echter aan te bevelen als uitstekend alternatief. Met dank aan: Wim Borghs, Martijn Weisbeek en Paul Berendsen).

  1. Download het bestand unz550xN.exe naar dezelfde map als het gedownloade bestand met de OpenOffice.org software.
  2. Dubbelklik op het bestand unz550xN.exe. Het pakt zichzelf dan uit. In de map komen naast het gedownloade OpenOffice.org bestand, het programma unzip.exe te staan, plus nog een aantal andere bestanden die je verder niet nodig hebt.
  3. Open een Opdrachtprompt (in Windows 9x/ME heet dit "MS-DOS prompt"). Zoek dit in het Startmenu ergens onder Programma's; bij sommige versies van Windows vind je het in het submenu Bureau-accessoires.
  4. Type in de Opdrachtprompt: cd "mapnaam" waarbij je mapnaam vervangt door de naam van de map waarin het gedownloade OpenOffice.org bestand (en inmiddels ook unzip.exe) staan. Voorbeeld: als de bestanden in C:\Mijn documenten\download staan, type dan cd "C:\Mijn documenten\download" (gevolgd door ENTER).
  5. Type vervolgens unzip *.zip (gevolgd door ENTER). Nu wordt het OpenOffice.org bestand uitgepakt in een nieuwe submap genaamd install.
  6. Dubbelklik op het programma setup.exe in deze submap. Dit start het installatieprogramma.
  7. Volg nu eenvoudig de aanwijzingen op het scherm, en OpenOffice.org 1.0 wordt op je computer geïnstalleerd.
Meerdere gebruikers
Als er meerdere gebruikers zijn ingesteld op je systeem, dan moet iedere gebruiker die OpenOffice.org 1.0 wil gebruiken, het pakket apart installeren. Normaal gesproken, neemt dit veel schijfruimte in beslag. Om dat te vermijden, kun je de "multi-user" optie gebruiken, hoewel de installatie dan iets ingewikkelder wordt:

  1. Pak het gedownloade bestand uit op de boven beschreven manier, tot en met stap 5.
  2. Type in de Opdrachtprompt: setup /net (gevolgd door de Enter toets).
  3. Volg daarna de aanwijzingen op het scherm. Noteer de map waarin OpenOffice.org 1.0 wordt geïnstalleerd.

Het bovenstaande installeert een gedeelde versie van OpenOffice.org 1.0 op je systeem. Nu kan iedere gebruiker die de OpenOffice.org software wil gebruiken, na zich te hebben aangemeld op het systeem, dubbelklikken op het programma setup.exe dat staat in de map waarvan je de locatie in stap 3 hierboven hebt genoteerd. Dit installeert dan de bestanden die voor de individuele gebruiker nodig zijn, wat per gebruiker slechts enkele megabytes extra inneemt op de harde schijf.
