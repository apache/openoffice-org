Instructies voor de installatie van OpenOffice.org 3.0

Beknopte aanwijzingen

  • Windows
  • Linux (.rpm)
  • Linux (.deb)

Installatie onder Windows

De installatie van OpenOffice.org onder Windows is zeer eenvoudig.

Allereerst: als je niet weet of je een Java JRE (minimaal versie 1.4) op je systeem hebt, en je hebt niet het "+JRE"-installatiebestand gedownload, download en installeer dan eerst de Java JRE.

Dubbelklik op het gedownloade OpenOffice.org-installatiebestand om het uit te voeren.

Volg daarna de aanwijzingen van de installatie.

Nadat de installatie is voltooid, kun je OpenOffice.org starten via het menu Start van Windows.

De eerste keer dat je OpenOffice.org opstart, vindt een tweede gedeelte van de installatie plaats. Als OpenOffice.org voor meerdere gebruikers is geïnstalleerd, moet iedere gebruiker deze procedure doorlopen. Er wordt gevraagd om te registreren. Dit helpt de OpenOffice.org community met informatie, maar het is volkomen vrijwillig.

Bij de installatie van 3.0 wordt, als er een 2.x versie is geïnstalleerd, gevraagd of de vorige versie moet worden verwijderd. De programmabestanden worden dan automatisch verwijderd. Persoonlijke gegevens uit de 2.x installatie, kunnen bijna allemaal worden overgenomen. Niet geïmporteerd worden aanpassingen in het OpenOffice.org hoofdmenu en toetsenbordinstellingen, gebruikersgegevens (Extra>Opties) en geregistreerde gegevensbronnen.

Veel plezier bij het gebruiken van OpenOffice.org 3!

 

Installatie onder Linux (rpm)

Van OpenOffice.org 3.x worden voor Linux zowel installatiebestanden in .rpm-formaat als in .deb-formaat beschikbaar gemaakt. Het installatiebestand voor distributies die de rpm-package manager gebruiken, zoals Red Hat, Fedora, SuSE, CentOS en Mandriva, heeft een bestandsnaam zonder toevoeging. Het installatiebestand voor debian-gebaseerde distributies, waaronder Ubuntu, heeft een bestandsnaam met "deb" erin. 

Hieronder wordt installeren met rpm behandeld, zie voor installeren met .deb verderop).

Belangrijk: alvorens de software te installeren, eerst alle distributie-eigen OpenOffice.org-componenten de-installeren!

Eerste vereisten: als je integratie met Java wenst, en Java nog niet is geïnstalleerd, dan kun je een versie met JRE (Java Runtime Environment) downloaden.

  1. Pak het bestand uit naar een directory, bijvoorbeeld met het commando tar xvzf OOo_3.x.x_LinuxIntel_install_nl.tar.gz
  2. Zorg ervoor dat je als root bent aangemeld of su naar root
  3. cd naar de directory met het uitgepakte downloadbestand.
  4. cd naar de directory met bestanden voor desktopintegratie.
  5. Verwijder de rpm-bestanden die niet van toepassing zijn op jouw systeem. Bijvoorbeeld, op een Fedora 3-systeem, verwijder alle RPMs die betrekking hebben op andere distributies, zoals openofficeorg-suse-menus-1.9.79-1.noarch.rpm.
  6. Voer daarna uit: rpm -Uvih *rpm

Bij de installatie van 3.0 wordt, als er een 2.x versie is geïnstalleerd, gevraagd of de vorige versie moet worden verwijderd. De programmabestanden worden dan automatisch verwijderd. Persoonlijke gegevens uit de 2.x installatie, kunnen bijna allemaal worden overgenomen. Niet geïmporteerd worden aanpassingen in het OpenOffice.org hoofdmenu en toetsenbordinstellingen, gebruikersgegevens (Extra>Opties) en geregistreerde gegevensbronnen.
 

Installatie onder Linux (debian-gebaseerde distributies)

Hieronder wordt installeren met het .deb-pakket behandeld. Het installatiebestanden met de aanduiding "deb" erin is bedoeld voor op debian gebaseerde distributies, waaronder Ubuntu. Heb je een distributie die gebruik maakt van de rpm-package manager, zie dan hier.

Verwijder de distributie-eigen OpenOffice.org vanuit een terminalscherm. Voer daarin de volgende opdracht uit:

# aptitude remove openoffice.org-core

Open een terminalscherm en ga naar de locatie met het gedownloade bestand. Pak het bestand uit op je bureaublad met de opdracht:

# tar xvzf OOo_3.0.0_LinuxIntel_install_nl_deb.tar.gz -C ~/Desktop/

vervolgens installeren met:

# dpkg -i ~/Desktop/OOO300_m9_native_packed-1_nl.9358/DEBS/*.deb

Voor GNOME-gebruikers is het mogelijk om de desktop integratie (snelkoppelingen in het menu) te installeren met de volgende opdracht:

# dpkg -i ~/Desktop/OOO300_m9_native_packed-1_nl.9358/DEBS/desktop-integration/*.deb

Uiteraard kan OpenOffice.org ook gestart worden vanuit een terminalscherm met de volgende opdracht:

# soffice

Nog even je bureaublad opruimen en klaar...
