Coordination page for localization of OOo 2.0/SO8

Issues

There already have been filed many issues regarding the translation of the GUI. A list of all issues can be found here.
If you have filed issues and they're not in this list, please check if NL: is in the summary of the issue. If not, they will not be found and put in the list.

Reviews

If you have any questions about the whole tranlating and reviewing process, you can have a look at the project page. This page is available in English and in Dutch. For further questions you can register to the localization mailinglist.

The review of the help files has started. Since there's a large number of files that have to reviewed, the overview of all files that need reviewing is available as an downloadable file.
