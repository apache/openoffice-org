Summary
Prepare a socket for the accept-call. The socket must have been bound before to the local address.
Parameters
MaxPendingConnections 
The maximum number of pending
connections (waiting to be accepted) on this socket. If you use
-1, a system default value is used.
Return
sal_True if call was successful.
