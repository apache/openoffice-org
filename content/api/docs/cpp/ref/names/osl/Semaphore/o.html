virtual abstract const volatile template static inline C-linkage
NO NO NO NO NO NO NO NO

Summary
The underlying oslSemaphore has no reference count.
Description
Since the underlying oslSemaphore is not a reference counted object, copy constructed Semaphore may work on an already destructed oslSemaphore object.
