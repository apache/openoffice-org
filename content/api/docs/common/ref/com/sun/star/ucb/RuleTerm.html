struct RuleTerm
Description
describes a term.

A term is used to select objects to which a rule should apply.

Elements' Summary
Property the name of the property used to match the term.  
Operand the value of the property used to compare with the document property.  
Operator the operator used to compare the property of the document with the given value (e.g. "contains" or "greater equal").  
CaseSensitive this flag indicates whether a string "operand" shall be compared case sensitive.  
RegularExpression this flag indicates whether a string "operand" shall be treated as a regular expression.
