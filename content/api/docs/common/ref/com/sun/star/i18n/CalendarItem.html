struct CalendarItem
Description
One entry in a calendar, for example, a day of week or a month or an era.

A sequence of CalendarItems is contained in Calendar::Days, Calendar::Months, Calendar::Eras

Elements' Summary
ID A unique ID for an entry of this type, usually the lower case abbreviated English name, for example, "sun" for Sunday.  
AbbrevName The abbreviated name, for example, "Sun". 
FullName The full name, for example, "Sunday".
