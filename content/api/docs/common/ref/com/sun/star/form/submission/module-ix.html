:: com :: sun :: star :: form ::

module submission
Description
collects functionality for submitting forms to external data sinks
Interfaces
XSubmission is implemented by components which support submitting data.
XSubmissionSupplier is implemented by a component which allows access to a component which can submit data.
XSubmissionVetoListener is implement by components which want to observe (and probably veto) the submission of data.
