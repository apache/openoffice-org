:: com :: sun :: star :: sheet ::

struct TableFilterField2
Description
describes a single condition in a filter descriptor.

This struct has the FilterOperator2 constants group as member, whereas the TableFilterField struct uses the FilterOperator enum.

Since
OOo 3.2
See also
SheetFilterDescriptor
Elements' Summary
Connection specifies how the condition is connected to the previous condition.  
Field specifies which field (column) is used for the condition.  
Operator specifies the type of the condition as defined in FilterOperator2.  
IsNumeric selects whether the TableFilterField2::NumericValue or the TableFilterField2::StringValue is used.  
NumericValue specifies a numeric value for the condition.  
StringValue specifies a string value for the condition.
