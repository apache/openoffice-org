service BaseStorage
Description
This is a service that allows to get access to a storage hierarchy.
Exported Interfaces - Summary
XStorage

This is a general interface representing storage functionality. It allows to copy storages, get access to child elements and to control lifetime.

::com::sun::star::beans::XPropertySet

allows to get access to storage properties.
