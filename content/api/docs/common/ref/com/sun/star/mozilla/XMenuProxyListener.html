interface XMenuProxyListener
Description
Listens for changes in menu items.
Methods' Summary
menuChangedMultiple is called, if the content or graphical representation/state of the menu changes completely, for one or more menu items  
menuChangedSingle is called, if the content or graphical representation/state of the menu changes, by one property for one or more menu items  
menuItemDeleted is called, if one menu item designated by the ID, is deleted.
