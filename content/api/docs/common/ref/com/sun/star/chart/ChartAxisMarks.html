:: com :: sun :: star :: chart ::

constants group ChartAxisMarks
Description
With these constants you can specify how the tick-marks of an axis are displayed.

You can combine INNER and OUTER with an arithmetical or-operation to get tick-marks that extend in both directions.
Constants
NONE Do not display any marks.  
INNER Display marks that point into the diagram area.  
OUTER Display marks that point out of the diagram area.
