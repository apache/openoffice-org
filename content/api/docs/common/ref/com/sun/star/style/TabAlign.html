enum TabAlign
Description
These enumeration values are used to specify the alignment of the text range delimited by a tabulator.
Values
LEFT The text range is left-aligned between the previous tabulator (or the left border, if none) and this tabulator.  
CENTER The text range is centered between the previous tabulator (or the left border, if none) and this tabulator.  
RIGHT The text range is right-aligned between the previous tabulator (or the left border, if none) and this tabulator.  
DECIMAL The decimal point of the text range to the left of this tabulator is aligned to the position of this tabulator.  
DEFAULT The default alignment for tabulators is applied.
