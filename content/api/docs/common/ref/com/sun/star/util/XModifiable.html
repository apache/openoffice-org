interface XModifiable
Base Interfaces
XModifiable
┗ XModifyBroadcaster
XModifyBroadcaster
(referenced interface's summary:)
broadcasts each modification made on the date data of the object which supports this interface.
Description
makes the modify state of the object accessible.

Additionally, it makes it possible to register listener objects, which get notification whenever the status or content of the object changes.
