:: com :: sun :: star :: ui ::

unpublished service WindowContentFactory
Supported Interface
::com::sun::star::lang::XSingleComponentFactory
Usage Restrictions
not published
Description
specifies a factory which creates a window that is a container for user interface elements. Dependent on the provided arguments different window types can be created. This container window must be capable of being integrated into another window (e.g. docking or floating windows).

The specific type of the created window depends on the provided arguments.

OOo 3.1
