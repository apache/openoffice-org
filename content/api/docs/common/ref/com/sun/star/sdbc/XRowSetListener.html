Developers Guide
Database - Events and Other Notifications
Methods' Summary
cursorMoved is called when a rowset's cursor is moved.  
rowChanged [ DEPRECATED ]
is called when a row is inserted, updated, or deleted.  
rowSetChanged is called when the row set has changed, or in other words, when the row set has been reexecuted.
