struct DataPilotFieldAutoShowInfo
Description
contains the auto show information of a DataPilotField.

If enabled, only a number of items with the highest or lowest result values are shown. The other items are hidden automatically.

See also
DataPilotField
Elements' Summary
IsEnabled specifies whether the AutoShow feature is enabled or not.  
ShowItemsMode specifies the mode which items have to be shown.  
ItemCount specifies the number of the items to show.  
DataField specifies the field where the values to show and select are taken from.
