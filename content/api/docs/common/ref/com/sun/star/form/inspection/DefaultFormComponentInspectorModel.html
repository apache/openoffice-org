unpublished service DefaultFormComponentInspectorModel
Supported Interface
::com::sun::star::inspection::XObjectInspectorModel
Usage Restrictions
not published
Description
implements a ::com::sun::star::inspection::XObjectInspectorModel for inspecting form components, in particular all components implementing the ::FormComponent service.

A DefaultFormComponentInspectorModel provides the following handlers by default:

  • ButtonNavigationHandler
  • CellBindingPropertyHandler
  • EditPropertyHandler
  • EventHandler
  • FormComponentPropertyHandler
  • SubmissionPropertyHandler
  • XMLFormsPropertyHandler
  • XSDValidationPropertyHandler

See also
::com::sun::star::inspection::XObjectInspectorModel::HandlerFactories
