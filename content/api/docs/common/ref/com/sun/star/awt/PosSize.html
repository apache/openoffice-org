constants group PosSize
Description
These constants are used to flag the parameters of a rectangle.
Constants
X flags the x-coordinate.  
Y flags the y-coordinate.  
WIDTH flags the width.  
HEIGHT flags the height.  
POS flags the x- and y-coordinate.  
SIZE flags the width and height.  
POSSIZE flags the x- and y-coordinate, width and height.
