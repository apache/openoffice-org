:: com :: sun :: star :: beans ::

struct PropertyValue
Description
specifies a property value.
Elements' Summary
Name specifies the name of the property.  
Handle contains an implementation-specific handle for the property.  
Value contains the value of the property or void, if no value is available.  
State determines if the value comes from the object itself or from a default and if the value cannot be determined exactly.  
Elements' Details
Name
string Name;
Description
specifies the name of the property.

The name is unique within a sequence of PropertyValues. Upper and lower case are distinguished.
Handle
long Handle;
Description
contains an implementation-specific handle for the property.

It may be -1 if the implementation has no handle. If available it can be used for fast lookups.
Value
any Value;
Description
contains the value of the property or void, if no value is available.
State
PropertyState State;
Description
determines if the value comes from the object itself or from a default and if the value cannot be determined exactly.
