:: com :: sun :: star :: style ::

constants group CaseMap
Description
These constants are used to specify a case-related mapping for formatting and displaying characters.
Constants
NONE The case of the characters is unchanged.  
UPPERCASE All characters are put in upper case.  
LOWERCASE All characters are put in lower case.  
TITLE The first character of each word is put in upper case.  
SMALLCAPS All characters are put in upper case, but with a smaller font height.
