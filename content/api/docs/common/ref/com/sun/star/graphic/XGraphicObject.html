:: com :: sun :: star :: graphic ::

unpublished interface XGraphicObject
Usage Restrictions
not published
Description
XGraphicObject objects represent in-memory image and graphic objects.

Such objects are accessable using GraphicObject scheme urls like

  • vnd.sun.star.GraphicObject:10000000000001940000012FB99807BD
The numeric portion of the url is formed from UniqueID. As long as at least one instance of XGraphicObject with a particular UniqueID exists, the associated image/graphic is available.

See also
XGraphicObject, GraphicProvider, MediaProperties
