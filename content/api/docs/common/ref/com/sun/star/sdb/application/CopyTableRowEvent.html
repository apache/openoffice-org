:: com :: sun :: star :: sdb :: application ::

unpublished struct CopyTableRowEvent
Base Hierarchy
CopyTableRowEvent
 ┗ ::com::sun::star::lang::EventObject
Usage Restrictions
not published
Description
specifies an event happening while copying table data between databases.

Whenever this event is fired to an XCopyTableListener, ::com::sun::star::lang::EventObject::Source contains the wizard instance which actually does the copying.

See also
CopyTableWizard
Elements' Summary
SourceData contains the result set which is being copied by the wizard currently.  
Error denotes the error which happened while copying the data.
