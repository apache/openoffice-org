:: com :: sun :: star :: drawing ::

service AccessibleGraphControl
Description
The service describes the accessible graph controls that are used in the image map of the Draw and Impress applications and the contour dialog of the Writer application.

The children of graph controls are shapes that define contrours.

Since
OOo 1.1.2
