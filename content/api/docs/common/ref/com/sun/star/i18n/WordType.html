:: com :: sun :: star :: i18n ::

constants group WordType
Description
Constants to specify the type of words.

Used with XBreakIterator::nextWord(), XBreakIterator::previousWord(), XBreakIterator::getWordBoundary(), XBreakIterator::getWordType(), XBreakIterator::isBeginWord(), XBreakIterator::isEndWord()

Constants
ANY_WORD Any "words" - words in the meaning of same character types, collection of alphanumeric characters, or collection of non-alphanumeric characters.  
ANYWORD_IGNOREWHITESPACES Any "words" - words in the meaning of same character types, collection of alphanumeric characters, or collection of non-alphanumeric characters except blanks.  
DICTIONARY_WORD "words" - in the meaning of a collection of alphanumeric characters and some punctuations, like dot for abbreviation.  
WORD_COUNT The mode for counting words, it will combine punctuations and spaces as word trail.
