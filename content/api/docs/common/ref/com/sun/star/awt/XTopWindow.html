interface XTopWindow
Description
manages the functionality specific for a top window.

Methods' Summary
addTopWindowListener adds the specified top window listener to receive window events from this window.  
removeTopWindowListener removes the specified top window listener so that it no longer receives window events from this window.  
toFront places this window at the top of the stacking order and shows it in front of any other windows.  
toBack places this window at the bottom of the stacking order and makes the corresponding adjustment to other visible windows.  
setMenuBar sets a menu bar.
