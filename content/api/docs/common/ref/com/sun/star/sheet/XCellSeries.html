interface XCellSeries
Description
provides methods to fill out a cell range automatically with values based on a start value, step count and fill mode.

Methods' Summary

Methods' Details
fillSeries
void
fillSeries( [in] FillDirection  nFillDirection,
[in] FillMode  nFillMode,
[in] FillDateMode  nFillDateMode,
[in] double  fStep,
[in] double  fEndValue );
Description
fills all cells in the range based on the specified settings.
Parameter nFillDirection
specifies the direction to fill the rows/columns of the range.
Parameter nFillMode
specifies the type of the series.
Parameter nFillDateMode
specifies the calculation mode for date values.
Parameter fStep
contains the value used to increase/decrease the series values.
Parameter fEndValue
contains the threshold value on which the calculation of the current series stops.
fillAuto
void
fillAuto( [in] FillDirection  nFillDirection,
[in] long  nSourceCount );
Description
fills all cells in the range in a way that is specified by the first cell(s) in the range.
Parameter nFillDirection
specifies the direction to fill the rows/columns of the range.
Parameter nSourceCount
contains the number of cells in each row/column used to constitute the fill algorithm.
