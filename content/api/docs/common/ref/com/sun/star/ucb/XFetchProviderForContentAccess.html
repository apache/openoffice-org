:: com :: sun :: star :: ucb ::

interface XFetchProviderForContentAccess
Description
provides the possibility to load information offered by a XContentAccess for several rows of a ContentResultSet with a single function call.
Methods' Summary
fetchContentIdentifierStrings returns the content identifier strings of the columns of the indicated rows  
fetchContentIdentifiers returns the XContentIdentifiers of the columns of the indicated rows  
fetchContents returns the XContents of the columns of the indicated rows  
Methods' Details
fetchContentIdentifierStrings
FetchResult
fetchContentIdentifierStrings( [in] long  nRowStartPosition,
[in] long  nRowCount,
[in] boolean  bDirection );
Description
returns the content identifier strings of the columns of the indicated rows
Returns
FetchResult::Rows contains a sequence of anys. Each of these anys contains a string.
Parameter nRowStartPosition
the starting row of the resultset
Parameter nRowCount
the count of rows
Parameter bDirection
true, if you want the rows to be read in the same order, as they are contained in the result set ( true <-> forward step; false <-> backward step )
