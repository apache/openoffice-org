interface XDispatchProviderInterceptor
Base Interfaces
XDispatchProviderInterceptor
┗ XDispatchProvider
XDispatchProvider
(referenced interface's summary:)
provides XDispatch interfaces for certain functions which are useful at the UI.
Description
makes it possible to intercept request of XDispatch.

Can be registered as an interceptor by using interface XDispatchProviderInterception.
