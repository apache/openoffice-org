unpublished enum PieChartOffsetMode

Description
Mode used for a pie chart template to determine the initial state of exploded pies.
Values
NONE Default, no pies are exploded.  
ALL_EXPLODED All pies are exploded by a certain percentage. The default is 10 percent.
