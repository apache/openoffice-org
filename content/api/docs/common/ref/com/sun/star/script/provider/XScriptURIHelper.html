unpublished interface XScriptURIHelper
Usage Restrictions
not published
Description
This interface is used to help transform Scripting Framework storage locations to Scripting Framework script URIs and vice versa.
Methods' Summary
getRootStorageURI Obtain the root storage URI for this ScriptURIHelper. The resulting string can be used to access the storage for this using the Universal Content Broker  
getStorageURI Obtain the storage URI for a specific Scripting Framework script URI.  
getScriptURI Obtain the Scripting Framework script URI for a specific UCB URI
