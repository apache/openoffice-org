unpublished constants group ImageType
Usage Restrictions
not published
Description
Determine the image set of an image manager.

The constants describe bits in a bit field which determine the current image set of an image manager.

Since
OOo 2.0
Constants
SIZE_DEFAULT an image with default size.  
SIZE_LARGE an image with big size;  
COLOR_NORMAL an image with normal colors.  
COLOR_HIGHCONTRAST an image with high contrast colors.
