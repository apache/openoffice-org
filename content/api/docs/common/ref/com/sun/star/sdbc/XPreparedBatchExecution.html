interface XPreparedBatchExecution
Description
is used for batch execution on PreparedStatements.

A PreparedStatement uses one precompiled SQL Statement. In batch execution it is possible to set collection of parameter settings, which are executed in one batch job.

Methods' Summary
addBatch adds a set of parameters to the batch.  
clearBatch makes the set of commands in the current batch empty.  
executeBatch submits a batch of commands to the database for execution.
