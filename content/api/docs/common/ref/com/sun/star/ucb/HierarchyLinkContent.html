:: com :: sun :: star :: ucb ::

service HierarchyLinkContent
Description
A HCP Link is a content which points to another location.

It is always contained in HCP Folder. A HCP Link has no children.

See also
HierarchyContentProvider, HierarchyRootFolderContent, HierarchyFolderContent

Supported Properties

  • boolean IsDocument ( read-only, always true )
  • boolean IsFolder ( read-only, always false )

  • string TargetURL ( the URL of the link target )
