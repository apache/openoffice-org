Description
* Constant definiton of a certificate container status.
Values
OTHER_NAME * Cutomize name/value pair * The value of CertAltNameEntry contains a NamedValue. * *  
RFC822_NAME * The entry contains rfc822 name. * The value of CertAltNameEntry contains a OUString. * *

EDI_PARTY_NAME * Currently unsupported.  
X400_ADDRESS * Currently unsupported.
