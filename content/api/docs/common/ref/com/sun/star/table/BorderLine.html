:: com :: sun :: star :: table ::

struct BorderLine
Description
describes the line type for a single cell edge.
Elements' Summary
Color contains the color value of the line.  
InnerLineWidth contains the width of the inner part of a double line (in 1/100 mm).  
OuterLineWidth contaions the width of a single line or the width of outer part of a double line (in 1/100 mm).  
LineDistance contains the distance between the inner and outer parts of a double line (in 1/100 mm).  
Elements' Details
Color
::com::sun::star::util::Color Color;
Description
contains the color value of the line.
InnerLineWidth
short InnerLineWidth;
Description
contains the width of the inner part of a double line (in 1/100 mm).

If this value is zero, only a single line is drawn.
OuterLineWidth
short OuterLineWidth;
Description
contaions the width of a single line or the width of outer part of a double line (in 1/100 mm).

If this value is zero, no line is drawn.
LineDistance
short LineDistance;
Description
contains the distance between the inner and outer parts of a double line (in 1/100 mm).
