:: com :: sun :: star :: text ::

constants group ChapterFormat
Description
These constants define the display format of the chapter number in a chapter text field.
Constants
NAME The title of the chapter is displayed.  
NUMBER The number including prefix and suffix of the chapter is displayed.  
NAME_NUMBER The title and number including prefix and suffix of the chapter are displayed.  
NO_PREFIX_SUFFIX The name and number of the chapter are displayed.  
DIGIT The number of the chapter is displayed.
