constants group StatusBarFunction
Description
used to specify the function used to calculate a result in the spreadsheet's status bar.
Constants
NONE nothing is calculated.  
AVERAGE average of all numerical values is calculated.  
COUNTNUMS all values, including non-numerical values, are counted.  
COUNT numerical values are counted.  
MAX maximum value of all numerical values is calculated.  
MIN minimum value of all numerical values is calculated.  
SUM sum of all numerical values is calculated.
