:: com :: sun :: star :: sheet ::

unpublished constants group ReferenceFlags
Usage Restrictions
not published
Description
defines flags for references.

The values can be combined.
Constants
COLUMN_RELATIVE selects a relative column reference.  
COLUMN_DELETED marks a deleted column reference.  
ROW_RELATIVE selects a relative row reference.  
ROW_DELETED marks a deleted row reference.  
SHEET_RELATIVE selects a relative sheet reference.  
SHEET_DELETED marks a deleted sheet reference.  
SHEET_3D selects a 3D sheet reference.  
RELATIVE_NAME marks a reference from a relative range name.
