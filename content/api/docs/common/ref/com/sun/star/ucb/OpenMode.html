:: com :: sun :: star :: ucb ::

constants group OpenMode
Description
These are the possible values for OpenCommandArgument::Mode.
Constants
ALL open a folder, include all children in result set (documents and folders).  
FOLDERS open a folder, include only children, that are folders, in result set.  
DOCUMENTS open a folder, include only children, that are documents, in result set.  
DOCUMENT open a document. There are no special requirements for data access sharing.  
DOCUMENT_SHARE_DENY_NONE open a document. Allow shared read and write access.  
DOCUMENT_SHARE_DENY_WRITE open a document. Deny shared write access.
