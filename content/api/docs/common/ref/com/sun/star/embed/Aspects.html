constants group Aspects
Description
The constant set contains possible aspects for an embedded object.

This constant set provides a set of values that can be used to specify the kind of object view. It can be used for example by container to request view representation of a certain kind from XEmbeddedObject.

The first 32 bits are reserved for MS OLE aspects.

See also
XEmbeddedObject
Constants
MSOLE_CONTENT specifies view of the object to be displayed as an embedded object inside a container.  
MSOLE_THUMBNAIL specifies view of the object to be displayed in a browsing tool.  
MSOLE_ICON specifies view of the object when object is represented by Icon.  
MSOLE_DOCPRINT specifies view of the object for print preview.
