:: com :: sun :: star :: system ::

exception SystemShellExecuteException
Base Hierarchy
SystemShellExecuteException
 ┗ ::com::sun::star::uno::Exception
Description
May be thrown in cases of errors executing a command using the SystemShellExecute service.

::com::sun::star::uno::Exception::Message may contain a system error message, but it is not mandatory.

The member PosixError specifies a POSIX conforming error code or -1 for unknown errors.
