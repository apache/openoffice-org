:: com :: sun :: star ::

module table
Description
Table specific interfaces (for text and spreadsheet).
Services
AccessibleCellView The accessible view of a cell in a text document or in the page preview of a spreadsheet document. See ::com::sun::star::sheet::AccessibleCell for cells in the edit view of a spreadsheet.
AccessibleTableView The accessible view of a table in a text document or in the page preview of a spreadsheet document. See ::com::sun::star::sheet::AccessibleSpreadsheet for tables in the edit view of a spreadsheet.
Cell represents a singe cell within a table.
CellCursor represents a cell cursor in a table.
CellProperties contains the properties of a table cell.
CellRange represents a range of of cells within a table.
CellRangeListSource defines the a source of list entries coming from a cell range in a tbale document
CellValueBinding defines the binding to a single cell in a table document
ListPositionCellBinding defines the binding to a single cell in a table document, which can be used to transfer a list position into the cell.
TableChart represents a chart based on data in a table or spreadsheet.
TableCharts represents a collection of all charts based on data in a table.
TableChartsEnumeration represents an enumeration of table charts.
TableColumn represents a special cell range containing all cells of a single specific column in a table or spreadsheet.
TableColumns represents a collection of all columns of a table or spreadsheet.
TableColumnsEnumeration represents an enumeration of table columns.
TableRow represents a special cell range containing all cells of a single specific row in a table or spreadsheet.
TableRows represents a collection of all rows of a table or spreadsheet.
TableRowsEnumeration represents an enumeration of table rows.
TableSortDescriptor [ DEPRECATED ]
contains properties which describe sorting of rows or columns in a table.
TableSortDescriptor2 specifies properties which describe sorting of fields (rows or columns) in a table.
Interfaces
XAutoFormattable provides a method to apply TableAutoFormats to a cell range.
XCell provides methods to access the contents of a cell in a table.
XCellCursor provides methods to control the position of a cell cursor.
XCellRange provides access to the cells or to sub-ranges of a cell range.
XColumnRowRange provides methods to access the collections of columns and rows of a cell range.
XMergeableCell provides methods to access information about a cell that is mergeable with other sells.
XMergeableCellRange represents a range of cells that can possibly be merged or unmerged.
XTable
XTableChart provides access to the settings of a chart object in a table or spreadsheet.
XTableCharts provides methods to access charts via name and to insert and remove charts.
XTableChartsSupplier provides a method to access a collection of charts in a table or spreadsheet.
XTableColumns provides methods to access columns via index and to insert and remove columns.
XTableRows provides methods to access rows via index and to insert and remove rows.
Structs
BorderLine describes the line type for a single cell edge.
CellAddress contains a cell address within a spreadsheet document.
CellRangeAddress contains a cell range address within a spreadsheet document.
ShadowFormat describes the settings of a cell shadow.
TableBorder contains the style settings of the border lines of all cells in a cell range.
TableBorderDistances contains the distance settings of the border lines of all cells in a cell range.
TableSortField describes how to sort a single field (row/column) in a tables sort descriptor.
Enums
CellContentType is used to determine the type of contents in a cell.
CellHoriJustify specifies how cell contents are aligned horizontally.
CellOrientation specifies the orientation of a cell.
CellVertJustify specifies how cell contents are aligned vertically.
ShadowLocation specifies the location of the shadow in a ShadowFormat.
TableOrientation used to select whether operations are carried out on columns or rows.
TableSortFieldType enumeration used to specify the type of contents in a sort field (row/column) of a table.
