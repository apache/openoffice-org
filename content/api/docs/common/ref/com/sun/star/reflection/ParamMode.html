enum ParamMode
Description
The parameter mode denotes the transfer between caller and callee of a method.
Values
IN parameter serves as pure input for a called method  
OUT parameter serves as pure output for the callee (in addition to the return value)  
INOUT parameter serves as input as well as output; data can transferred in both directions
