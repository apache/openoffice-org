constants group ContentInfoAttribute
Description
These are the possible values for ContentInfo::Attributes.
Constants
NONE No extra attributes.  
INSERT_WITH_INPUTSTREAM Supply an input stream with the InsertCommandArgument, when committing the new content by executing the command "insert" at it.  
KIND_DOCUMENT The new content is of kind document.  
KIND_FOLDER The new content is of kind folder.  
KIND_LINK The new content is of kind link.

KIND_DOCUMENT

Description
The new content is of kind document.

It's value for the property "IsDocument" will be true.

KIND_FOLDER
const short KIND_FOLDER = 4;
Description
The new content is of kind folder.

It's value for the property "IsFolder" will be true.

KIND_LINK
const short KIND_LINK = 8;
Description
The new content is of kind link.

It will support the property "TargetURL" ( which identifies it as a link ).
