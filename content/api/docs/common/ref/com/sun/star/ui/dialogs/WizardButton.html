Description
denotes the buttons found in a Wizard
Constants
NONE denotes none of the buttons in the wizard 
NEXT denotes the button used to travel forward through the wizard 
PREVIOUS denotes the button used to travel backward through the wizard 
FINISH denotes the button used to finish the wizard 
CANCEL denotes the button used to cancel the wizard 
HELP denotes the button used to request help
