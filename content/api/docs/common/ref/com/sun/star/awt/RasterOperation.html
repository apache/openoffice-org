enum RasterOperation
Description
These values are used to specify the binary pixel-operation applied when pixels are written to the device.
Values
OVERPAINT sets all pixel as written in the output operation.  
XOR uses the pixel written as one and the current pixel as the other operator of an exclusive or-operation.  
ZEROBITS All bits which are affected by this operation are set to 0.  
ALLBITS All bits which are affected by this operation are set to 1.  
INVERT All bits which are affected by this operation are inverted.
