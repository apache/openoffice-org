module oooimprovement
Services
Core specifies a service which allows interfacing the core components of the Core
CoreController specifies a service controlling the core components of the Openoffice.org improvement program. The OOo improvement program extension has to implement this service.
Interfaces
XCore specifies the interface used to query the Core status
XCoreController specifies the interface used to control the OOo Improvement core components
