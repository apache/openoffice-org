:: com :: sun :: star :: rendering ::

unpublished constants group ColorComponentTag
Usage Restrictions
not published
Description
A collection of tags for the individual components of a color.

Color components usually corrspond to physical attributes like the amount of red colorant contained in an output color. This constant field enumerates the most common ones.

Since
OOo 2.3
