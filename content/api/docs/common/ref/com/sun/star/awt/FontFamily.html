constants group FontFamily
Description
These values are used to specify the general kind of font.

They may be expanded in future versions.
Constants
DONTKNOW specifies an unknown font family.  
DECORATIVE specifies the family of decorative fonts.  
MODERN specifies the family of modern fonts.  
ROMAN specifies the family roman fonts (with serifes).  
SCRIPT specifies the family of script fonts.  
SWISS specifies the family roman fonts (without serifes).  
SYSTEM specifies the family system fonts.
