getComponentSchema
XSchema
getComponentSchema( [in] string  aComponent )
raises( BackendAccessException,
::com::sun::star::lang::IllegalArgumentException );
Description
Returns the schema information (component + templates) for a particular component.
Parameter aComponent
component whose schema will be accessed
Returns
an object allowing access to the various parts of the schema, NULL if the component doesn't exist.
Throws
com::sun::star::lang::IllegalArgumentException if the component identifier is invalid.
Throws
com::sun::star::configuration::backend::BackendAccessException if an error occurs while accessing the data.
