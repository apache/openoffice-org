struct ChangesEvent
Base Hierarchy
ChangesEvent
 ┗ ::com::sun::star::lang::EventObject
Description

XChangesSet, XChangesBatch, XChangesListener, XChangesNotifier, ::com::sun::star::container::ContainerEvent, ElementChange
