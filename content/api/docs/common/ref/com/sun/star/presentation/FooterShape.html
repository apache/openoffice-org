:: com :: sun :: star :: presentation ::

unpublished service FooterShape
Usage Restrictions
not published
Description
This service is implemented by the footer presentation shape.

Presentation shapes can be used in a presentation page layouts and theire position and size is by default set by the presentation shapes on the ::com::sun::star::drawing::MasterPage.
