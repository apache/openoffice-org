Exported Interfaces - Summary
::com::sun::star::container::XIndexContainer

The control serves as an indexed container typically for RoadmapItems as specified in com::sun::star::awt:RoadmapItem. The RoadmapItems are held in a sequence. When inserting such items their ID is set equal to the Index of their insertion by default. After removing items the CurrentItem property is - when beyound the upper sequence boundaries - set equal to last index of the RoadmapItem Array.

Description
specifies a graphic to be displayed on the control

If this property is present, it interacts with the ImageURLin the following way:

  • If ImageURL is set, Graphic will be reset to an object as loaded from the given image URL, or NULL if ImageURL does not point to a valid image file.
  • If Graphic is set, ImageURL will be reset to an empty string.
