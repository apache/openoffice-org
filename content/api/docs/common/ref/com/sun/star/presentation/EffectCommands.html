Constants
CUSTOM the command is user defined  
VERB the command is an ole verb. Required parameters are "Verb" of type long that specifies the verb to execute.  
PLAY the command starts playing on a media object. Optional parameters are "MediaTime" of type double that specifies the start time in mili seconds. If not given, play continues at last position known.  
TOGGLEPAUSE the command toggles the pause status on a media object.  
STOP the command stops the animation on a media object  
STOPAUDIO the command stops all currently running sound effects.
