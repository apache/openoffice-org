:: com :: sun :: star :: awt ::

constants group InvalidateStyle
Description
specifies how to invalidate windows.
Constants
CHILDREN The child windows are invalidated, too.  
NOCHILDREN The child windows are not invalidated.  
NOERASE The invalidated area is painted with the background color/pattern.  
UPDATE The invalidated area is updated immediately.  
TRANSPARENT The parent window is invalidated, too.  
NOTRANSPARENT The parent window is not invalidated.  
NOCLIPCHILDREN The area is invalidated regardless of overlapping child windows.
