Cas Test : gra_wmf01

Zone de test : Solaris - Linux - Windows

But du test :
Vérifier qu'OpenOffice.org ouvre un fichier graphique de type .wmf correctement

Nécéssités :

Un fichier *.wmf est nécessaire pour effectuer ce test

Description :
  1. A partir de la barre des menus, sélectionnez Fichier - Ouvrir
  2. la boite de dialogue d'ouverture de fichier doit se lancer
  3. entrez le nom du fichier .wmf, sélectionnez Ouvrir dans la fenêtre de dialogue d'ouverture
  4. fermez le fichier graphique

Résultat attendu :

OpenOffice.org doit ouvrir une nouvelle fenêtre affichant le fichier graphique
