Cas Test : gen_prt07

Test Area : Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut imprimer un document en texte plein avec succès.


Nécessités :
Qu'une imprimante par défaut soit définie pour l'utilisateur

Description:

    1. A partir de la barre des menus, sélectionnez Fichier- Nouveau - Document
    2. Un nouveau document texte doit s'ouvrir
    3. Entrez un court message ex: "ceci est un court message"
    4. A partir de la barre des menus, sélectionnez Fichier - Imprimer
    5. La boite de dialogue d'Impression doit se lancer
    6. Sélectionnez l'imprimante devant imprimer le document dans la liste déroulante dous le Nom
    7. Assurez-vous que le document s'imprime correctement
    8. Fermez le document
Résultat attendu
Toutes les opérations doivent se dérouler avec succès


Fin du cas test
