Cas Test : macro_06

Zone de Test : Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut enregistrer une macro simple, l'assigner et la faire tourner

Description:

      1. Ouvrez un nouveau document texte
      2. A partir de la barre des menus, sélectionnez Outils - Macros - Enregistrer la macro
      3. Entrez un texte court ex :"Court message"
      4. Sélectionnez tout le texte en appuyant sur ctrl+a
      5. Sélectionnez Format - Caractère, sélectionnez l'onglet Effet de Police, changez la couleur pour rouge, appuyez sur OK
      6. Sélectionnez G pour Gras dans la barre d'objets
      7. Sélectionnez Arrêter l'enregistrement
      8. Dans la boite de dialogue de Macro, sélectionnez soffice, Standard, Module 1 et enregistez comme Main
      9. Sélectionnez Outils - Adaptation, sélectionnez l'onglet Clavier
      10. Dans la fenêtre de raccourci clavier, sélectionnez F10
        dans la fenêtre Catégorie, sélectionnez OpenOffice Basic, Standard, Module 1
        dans la fenêtre Fonction, sélectionnez Main
      11. Sélectionnez le bouton Modifier, et cliquez sur OK
      12. Sortez et redémarrez OpenOffice.org avec un nouveau document texte
      13. Pressez F10, le court message de texte, coloré en rouge et formaté en gras doit apparaître dans le document
Résultat attendu
Toutes les opérations doivent se dérouler avec succès

Fin du cas test
