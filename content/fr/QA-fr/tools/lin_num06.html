Cas Test : lin_num06

Zone de Test : Solaris  - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut insérer des numéros de lignes dans un document texte

Description:

      1. A partir de la barre des menus, sélectionnez Outils - Numérotation des lignes
      2. La boite de dialogue de numérotation des lignes doit se lancer
      3. Cochez l'option Activer la numérotation dans la boite de dialogue
      4. Cliquez sur OK
      5. Appuyez plusieurs fois sur la touche Entrée, la marge de la page doit afficher des numéros de comptage des lignes du style 5, 10, 15, 20...
      6. A partir de la barre des menus, sélectionnez Outils - Numérotation des lignes
      7. La boite de dialogue de numérotation doit se lancer
      8. Décochez l'option Activer la numérotation dans la boite de dialogue
      9. Cliquez sur OK dans la boite de dialogue

     
Résultat attendu :
Toutes les opérations doivent se compléter avec succès

Fin du cas test
