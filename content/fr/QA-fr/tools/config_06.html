Cas Test : config_06

Zone de Test : Solaris  - Linux - Windows

But du test 

Vérifier que la boite de dialogue de configuration d'OpenOffice.org fonctionne correctement

Description:

      1. A partir de la barre des menus, sélectionnez  Outils - Adaptations
      2. La boite de dialogue d'adaptation doit se lancer
      3. Assurez-vous qu'elle comporte bien 5 onglets
        Menu, Clavier, Barre d'état, Barre d'outils, Evénements
        sélectionnez chaque onblet 
      4. Cliquez sur Annuler dans la boite de dialogue d'Adaptation
Résultat attendu :
Toutes les opérations doivent se dérouler avec succès

Fin du cas test
