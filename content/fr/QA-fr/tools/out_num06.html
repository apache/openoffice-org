Cas Test : out_num06

Zone de Test : Solaris  - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org lance la boite de dialogue de numérotation des chapitres correctement


Description:

     
    1. A partir de la barre des menus, sélectionnez Outils - Numérotation des chapitres
    2. La boite de dialogue de numérotation des chapitres doit se lancer
    3. Assurez-vous qu'elle comporte bien deux onglets
      Numérotation et Position
      sélectionnez chacun des onglets
    4. Cliquez sur Annuler dans la boite de dialogue de numérotation des chapitres
Résultat attendu :
Toutes les opérations doivent se compléter avec succès
