Cas Test : for_col05

Zone de Test : Solaris - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut insérer 4 colonnes dans un document texte

Description:

    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
    2. Un nouveau document texte doit se lancer
    3. A partir de la barre des menus, sélectionnez Format - Colonnes
    4. Une boite de dialogue de formatage des colonnes doit s'afficher
    5. Régler le nombre de colonnes sur 4 et appuyez sur OK
    6. Assurez-vous que 4 colonnes s'affichent bien dans le document
    7. Fermez le document texte
Résultat attendu :

Totues les opérations doivent se compléter avec succès

Fin du cas test
