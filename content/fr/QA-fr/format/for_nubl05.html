Cas Test : for_nubl05

Zone de Test : Solaris - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut ouvrir la boite de dialogue de Numérotation/puces avec succès

Description:

    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
    2. Un nouveau document texte doit se lancer
    3. A partir de la barre des menus, sélectionnez Format - Numérotation/puces 
    4. La boite de dialogue de Numérotation/puces doit se lancer
    5. Assurez-vous que cette boite de dialogue comporte bien 6 onglets
      Puces, Type de Numérotation, Plan, Images, Position, Options,
      sélectionnez chaque onglet
    6. Fermez le document texte
Résultat attendu :
Toutes les opérations doivent se dérouler avec succès
