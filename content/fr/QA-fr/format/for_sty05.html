Cas Test : for_sty05

Zone de Test : Solaris - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut insérer un style de numérotation 'Numérotation 1' dans un document texte


Description:

    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
    2. Un nouveau document texte doit se lancer
    3. Entrez un court message de texte ex:" ceci est un court message"
    4. A partir de la barre des menus, sélectionnez Format - Styles - Catalogue 
    5. Une nouvelle boite de dialogue de formatage des styles doit apparaître
    6. En utilisant la liste déroulante, vérifiez qu'il y a bien 5 options de style qui apparaissent
      Styles de Paragraphe, Styles de Caractères, Styles de Cadre, Style de Page, Style de Numérotation
      Sélectionnez chaque option de style
    7. Sous Style de numérotation, sélectionnez Numérotation 1 dans la fenêtre, cliquez OK dans la boite de dialogue
    8. Le document doit maintenant s'afficher
      "1. ceci est un court message de texte"
    9. Fermez le document texte
Résultat attendu

Toutes les opérations doivent se compléter avec succès

Fin du cas test
