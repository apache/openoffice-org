Cas Test : cmp_doc04

Zone de Test : Solaris - Linux - Windows


But du test :

Vérifier qu'OpenOffice.org peut comparer deux documents texte

Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
    2. Un nouveau document texte doit se lancer
    3. Entrez un court message ex:"ceci est un court message messagessss"
    4. A partir de la barre des menus, sélectionnez Fichier - Enregistrer sous...
    5. Une boite de dialogue d'enregistrement doit s'ouvrir
    6. Entrez OO10-comp1 dans le champ nom de la boite de dialogue
    7. Sélectionnez OpenOffice.org 1.0 Document Texte comme type de fichier
    8. Sélectionnez Enregistrer dans la boite de dialogue
    9. Fermez le document
       
    10. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
    11. Un nouveau document texte doit se lancer
    12. Entrez un court message ex: "ceci est un court message"
    13. A partir de la barre des menus, sélectionnez Fichier - Enregistrer sous...
    14. Une boite de dialogue d'enregistrement doit s'ouvrir
    15. Entrez OO10-comp2 dans le champ nom de la boite de dialogue
    16. Sélectionnez OpenOffice.org 1.0 Document Texte comme type de fichier
    17. Sélectionnez Enregistrer dans la boite de dialogue
       
    18. A partir de la barre des menus, sélectionnez Edition - Comparer le document
    19. Sélectionnez OO1.0-cmp1 dans la boite de dialogue et cliquez sur Insérer
    20. Une boite de dialogue d'acceptation ou de rejet des modifications doit s'ouvrir et le mot messagesss doit apparaître en surbrillance
    21. Sélectionnez Accepter dans la boite de dialogue. Le mot 'messagesss' doit rester dans le document
    22. Fermez la boite de dialogue et fermez le document
 
Réultat attendu
Toutes les opérations doivent se compléter avec succès


Fin du cas test
