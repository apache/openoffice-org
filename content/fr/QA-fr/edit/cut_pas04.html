Cas Test : cut_pas04

Zone de Test : Solaris - Linux - Windows

But du test 

Vérifier que les fonctions couper/coller d'OpenOffice.org fonctionnent correctement

Description:


    1. Ouvrez un nouveau document texte, entrez un court message ex : "ceci est un court message"
    2. A partir de la barre des menus, sélectionnez Edition - Sélectionner tout
    3. Le texte entré doit être en surbrillance
    4. A partir de la barre des menus, sélectionnez Edition - Couper, le texte en surbrillance doit disparaître du document texte
    5. A partir de la barre des menus, sélectionnez Edition - Coller, "ceci est un court message" doit maintenant être réinséré.
Résultat attendu
Toutes les opérations doivent être complétées avec succès
