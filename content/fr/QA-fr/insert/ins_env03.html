Cas Test : ins_env03

Zone de Test: Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut insérer une enveloppe


Description:

      1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
      2. Un nouveau document texte doit être lancé
      3. A partri de la barre des menus, sélectionnez Insertion - Enveloppe
      4. La boite de dialogue d'insertion doit être lancée
      5. Assurez-vous qu'elle comporte trois onglets
        Enveloppe, Format, Imprimante
        Sélectionnez chacun des onglets
      6. Sélectionnez Insérer dans la boite de dialogue d'insertion, assurez-vous qu'une enveloppe est insérée
      7. Fermez le document texte
Résultat attendu :
Toutes les opérations doivent se compléter avec succès

Fin du cas test
