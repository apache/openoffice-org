Cas Test : ins_ftn03

Zone de test : Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut insérer une note de bas page dans un document texte

Description:

      1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
      2. Un nouveau document texte doit être lancé
      3. A partir de la barre des menus, sélectionnez Insertion - Notes de bas de page
      4. La boite de dialogue d'insertion de bas de page doit se lancer
      5. Sélectionnez l' icône de recherche (bouton avec 3 points dessus) de la boite de dialogue d'insertion
      6. La boite de dialogue de caractères spéciaux doit se lancer
      7. Sélectionnez Annuler dans la boite de dialogue de caractères spéciaux
      8. Cliquez OK dans la boite de dialogue d'insertion de notes de bas de page, un cadre de bas de page doit s'afficher
      9. Fermez le document texte
Résultat attendu
Toutes les opérations doivent se compléter avec succès

Fin du cas test
