Cas Test : ins_mbk03

Zone de Test : Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut insérer un saut de page manuel dans un document texte


Description:

  1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
  2. Un nouveau document texte doit se lancer
  3. A partir de la barre des menus, sélectionnez Insertion - Saut manuel
  4. La boite de dialogue d'insertion de saut doit se lancer
  5. Dans la boite de dialogue d'insertion, sélectionnez saut de page et cliquez sur OK
  6. Dans le document, le curseur doit maintenant se positionner sur la seconde page
  7. Fermez le document texte
Résultat attendu :
Toutes les opérations doivent se compléter avec succès
