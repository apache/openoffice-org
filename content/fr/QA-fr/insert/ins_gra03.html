Cas Test : ins_gra03

Zone de Test : Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut insérer un fichier  image (type gif) dans un document texte


 
Nécessités 
Un fichier *.gif est nécessaire pour réaliser ce test

Description:

      1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
      2. A partir de la barre des mensu, sélectionnez Insertion - Image - A partir d'un fichier
      3. La boite de dialogue d'insertin des images doit s'ouvrir,
      4. Sélection Insérer dans la boite de dialogue d'insertion, assurez-vous que l'image gif s'insère correctement
      5. Fermez le document
Résultat attendu
Toutes les opérations doivent se compléter avec succès

Fin du cas test
