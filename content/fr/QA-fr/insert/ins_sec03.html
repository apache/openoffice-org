Cas Test : ins_sec03

Zone de Test : Solaris  - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut lancer la boite de dialogue d'insertion de section correctement


Description:

  1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
  2. Un nouveau document texte doit se lancer
  3. A partir de la barre des menus, sélectionnez Insertion - Section 
  4. La boite de dialogue d'insertion de section  doit se lancer
  5. Assurez-vous qu'elle comporte bien 4 onglets :
    Section, Colonnes, Arrière-plan, Entête/Pied de page
    Sélectionnez chaque onglet
  6. Sous l'onglet colonnes, cliquez sur 2 colonnes, et sélectionnez Insérer dans la boite de dialogue
  7. Une ligne avec deux colonnes doit être insérée dans le document
  8. Fermez le document texte
Résultat attendu :
Toutes les opérations doivent se compléter avec succès
