Cas Test : ins_htm03

Zone de Test : Solaris  - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut lancer la boite de dialogue  hyperlien et de repère dans un document html

Description:
 
  1. A partir de la barre des menus, sélectionnez Fichiez - Nouveau - Document HTML
  2. A partir de la barre des menus, sélectionnez Insertion - Hyperlien
  3. La boite de dialogue d'hyperlien doit se lancer
    Dans la boire de dialogue sélectionnez chaque icône dans la fenêtre de gauche
    Internet, Mail & News, Document, Nouveau Document
    Après avoir sélectionner chaque icône, assurez vous que le contenu de droite correspond à chaque icône.
  4. Fermez la boite de dialogue d'hyperlien
  5. A partir de la barre des menus, sélectionnez Insertion - Repère
  6. Une boite de dialogue d'insertion de repère doit se lancer
  7. Sélectionnez Annuler dans la boite de dialogue
  8. Fermez le document html

   

Résultat attendu

Toutes les opération doivent se compléter avec succès
