Cas Test : ins_frm03

Zone de Test : Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.ort peut insérer un cadre dans un document texte


Description:
      1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
      2. A partir de la barre des menus, sélectionnez Insertion - Cadre
      3. La boite de dialogue d'insertion de cadre doit être lancée
      4. Assurez-vous qu'elle comporte bien 8 onglets
        Type, Options, Adaptation du texte, Hyperlien, Bordure, Arrière-plan, Colonnes, Macro
        sélectionnez chaque onglet
      5. Appuyez sur OK dans la boite de dialogue
      6. Assurez-vous qu'un cadre est inséré dans le document, recadrez le en sélection un des coins avec la souris et en le glissant
      7. Fermez le document
Résultat attendu
Toutes les opérations doivent se compléter avec succès

Fin du cas test
