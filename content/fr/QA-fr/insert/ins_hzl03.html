Cas Test : ins_hzl03

Zone de Test : Solaris  - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut insérer une règle horizontale dans un document texte


Description:

      1. From the menu bar,  select File - New - Text Document.
      2. From the menu bar,  select Insert - Horizontal Line.
      3. The Insert Horizontal Ruler dialog should be launched.
      4. Select cancel on the horizontal line dialog box.
      5. Close the document.
      1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
      2. A partir de la barre des menus, sélectionnez Insertion - Ligne horizontale
      3. Une boite de dialogue d'insertion de ligne doit s'afficher
      4. Sélectionnez Annuler dans la boite de dialogue
      5. Fermez le document
Résultat attendu
Toutes les opérations doivent se compléter avec succès

 

Fin du cas test
