Cas Test : ins_fld03

Zone de Test : Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut insérer un champ de date dans un document texte


Description:

      1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
      2. A partir de la barre des menus, sélectionnez Insertion - Champ - Date
      3. Assurez-vous que la date du jour est insérée dans le document
      4. Fermez le document
Résultat attendu :
Toutes les opérations doivent se dérouler avec succès

Fin du cas test
