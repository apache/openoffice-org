Répertoires et noms de fichiers

Répertoires 

Les répertoires sont créés sur les Miroirs dans les Structure de répertoire
Les noms de ces répertoires suivent le shémas de numérotation des vesions stables ou des versions développeurs respectivement.

Fichiers

Les noms de fichiers suivent le shéma suivant
OOo_version_OSCPU_quoi_locale.ext

où
OOo =  chaîne statique "OOo"
version = 1.0.1|643|643C|...
OS = Win32 |Linux |Solaris |MacOS X|...
CPU =  Intel | Sparc | PPC|...
quoi = install | solver | source | sdk | ...
locale = optionnel (defaut = en-us) = de | fr |...
ext = tar.gz | zip |... (plus de .bz2 pour économiser de la place sur les mirroirs :)

Pour les localisations, nous utilisons les deux lettres du code ISO

Ce shéma a été repris et étendu à partir d'une discussion sur les mailing listes
