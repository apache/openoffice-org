Cas Test : view_data04

Zone de Test : Solaris  - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut afficher la source de données bibliographique dans l'affichage de sources de données


Description:

    1. A partir de la barre des menus, sélectionnez Affichage - Source de données
    2. Deux nouveaux cadres doivent apparaître dans la fenêtre
    3. Dans le cadre de gauche, doit apparaître la source de données Bibliographique
    4. A partir de l'arborisation, sélectionnez Tables - Biblio
    5. Une liste d'identifiants, Type, Adresses, etc. doit apparaître dans le cadre de droite
    6. A partir de la barre des menus, sélectionnez Affichage - Source de données
    7. Les deux nouveaux cadre ne doivent plus s'afficher maintenant
Résultat attendu :
Toutes les opérations doivent se compléter avec succès


Fin du cas test
