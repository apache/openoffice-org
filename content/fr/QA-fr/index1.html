 Tests OpenOffice.org 

Ensemble des Tests de la Suite


 
 
 
 
 
 
Groupes de Tests :-
Tests d'Installation
 Installation d'OpenOffice.org dans un environnement simple ou multi-utilisateur
Tests des Types de Fichiers
 Teste qu'OpenOffice.org peut ouvrir et afficher correctement des types de fichiers sépcifiques. Peut ouvrir/exécuter un applet Java
Actions des Fichiers
 Teste qu'OpenOffice.org peut créer et sauvegarder différent types de fichiers. Peut envoyer un mail en utilisant un programme externe.
Actions d'Insertion
 Teste qu'OpenOffice.org peut insérer des actions spécifiques dans un document texte.
Actions d'Edition
 Teste qu'OpenOffice.org active les fonctions d'édition comme spécifié dans un document texte. Teste que le copier/coller est intégré avec un éditeur externe et OpenOffice.org
Actions d'Affichage
 Teste que les fonctions d'affichage d'OpenOffice.org fonctionnes comme spécifiées
Actions de Formatage
 Test que les fonctions de formatage d'OpenOffice.org agissent comme spécifiées
Outils/Option
 Teste que la fonction Outils/options d'OpenOffice.org fonctionne correctement
Impression
 Test qu'OpenOffice.org imprime un document texte sur une imprimante par défant .
Aide
 Test les fonctions et le contenu de l'aide.
