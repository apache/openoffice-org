Cas Test : help_04

Zone de Test : Solaris  - Linux - Windows

But du test

Vérifier qu'OpenOffice.org affiche l'aide pour Draw correctement

Description:

    1. A partir de la barre des menus, sélectionnez Aide - Sommaire
    2. La page de sommaire de l'Aide doit être lancée
    3. Dans la page de sommaire de l'aide, sélectionnez l'onglet Contenu
    4. Double cliquez sur Aide pour Draw,
    5. Double cliquez sur Aperçu des fonctionnalités pour Draw
    6. Assurez vous que le contenu de la fenêtre de droite affiche
      "Aide de OpenOffice.org Draw - Bienvenue !"
Résultat attendu :
Toutes les opérations doivent se compléter avec succès

Fin du cas test
