Solaris

But du Test

Vérifier qu'OpenOffice.org s'installe correctement et peut être lancé en mode réseau

Zone de Test : Solaris

Description : installation réseau

  • loguez vous sur la machine test en root

  • téléchargez les fichiers tar-ball openoffice.org qui doivent être testés ici :
    http://www.openoffice.org/dev_docs/source/download.html
  • extrayez les tar-ball et installez-les en exécutant ./setup /net à partir du répertoire d'installation
  • suivez les instructions des fenêtre d'installation
    • acceptez tous les agréments de licence
    • lorsque l'on vous demande Sélectionnez le Type d'Installation, choisissez 'Installation Standard'
    • lorsque l'on vous demande Sélectionnez le Répertoire d'Installation, choisissez 'Répertoire par défaut'
      ou entrez un chemin d'installation alternatif

  • lorsque l'installation est complète, déconnectez-vous de root
  • loguez vous comme nouvel utilisateur sur la machine cible
  • faites un cd vers le répertoire <network-installation-set>
  • faites une installation poste de travail en tapant ./setup à partir du répertoire <network-installation-set>
  • suivez les fenêtres d'installation
    • acceptez tous les agréments de licence
    • lorsque l'on demande Sélectionnez le Type d'Installation, choisissez 'Installation Standard'
    • lorsque l'on demande Sélectionnez le Répertoire d'Installation, choisissez 'Répertoire par Défaut'
    • lorsque l'on vous demande le Java Setup, assurez vous que Java 1.3.1 est sélectionné, si nécessaire, installez Java 1.3.1 sur la machine cible

  • une fois que l'installation est complète, lancez OpenOffice.org en exécutant ./soffice à partir du répertoire d'installation
  • assurez-vous qu'OpenOffice.org se lance correctement.
