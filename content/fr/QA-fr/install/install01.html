But du Test

Vérifier qu'OpenOffice.org s'installe avec succès et peut être lancé sur une install simple utilisateur

Zone de test : Solaris

Description  : installation standard
  • loguez vous comme nouvel utilisateur sur votre machine Solaris,
  • téléchargez les tar-ball OpenOffice.org qui doivent être testés à l'adresse suivante :
    http://www.openoffice.org/dev_docs/source/download.html

  • extrayez le tar-ball et faites une install  en exécutant ./setup à partir du répertoire d'installation

  • suivez les fenêtres d'installation
    • acceptez tous les agréments de licence
    • lorsque l'on demande Sélectionnez le Type d'Installation, choisissez 'Installation Standard'
    • lorsque l'on demande Sélectionnez le Répertoire d'Installation, choisissez 'Répertoire par Défaut'
    • lorsque l'on vous demande le Java Setup, assurez vous que Java 1.3.1 est sélectionné, si nécessaire, installez Java 1.3.1 sur la machine cible

  • une fois que l'installation est complète, lancez OpenOffice.org en exécutant ./soffice à partir du répertoire d'installation
  • assurez-vous qu'OpenOffice.org se lance correctement.
