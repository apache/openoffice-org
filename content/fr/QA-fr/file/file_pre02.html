Cas Test : file_pre02

Zone de Test: Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer une nouvelle  présentation.

Description:
    1. A partir de la barre de menu , sélectionnez Fichier - Nouveau - Présentation
    2. La boite de dialogue de présentation doit s'ouvrir
    3. Assurez-vous qu'une présentation vide est sélectionnée dans la boite de dialogue
    4. Sélectionnez Créer dans la boite de dialogue
    5. Assurez-vous qu'OpenOffice.org créé une nouvelle présentation
    6. Le document doit être vite
      Une boite de dialogue de modification de la diapo doit s'ouvrir
    7. Fermer la boite de dialogue
    8. Fermer le document de présentation
Résultat attendu
Toutes les opérations doivent être complétées avec succès
