Cas Test : aut_agn02


Aire de Test : Solaris - Linux - Windows

But du Test :

Vérifier que OpenOffice.org peut créer un agenda à partir de l'Autopilot

Description
  1. A partir de la barre de menu, sélectionner : Fichier - Autopilote - Agenda
  2. La boite de dialogue d'Autopilote d'Agenda doit se lancer,
  3. Vérifiez qu'il n'y a pas d'apparence préférée de sélectionnée dans la boite de dialogue de l'Agenda
  4. Sélectionnez Créer dans la boite de dialogue d'Autopilote d'Agenda
  5. Assurez-vous qu'OpenOffice crée un Agenda à partir de l'Autopilote
  6. Le document doit indiquer la date du jour,
    une entête indiquant "Les points de l'Agenda"
  7. Fermez le document
Résultat attendu
Toutes les opérations doivent se compléter avec succès.
