Cas Test: sen_mail02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut envoyer un mail en utilisant un logiciel de mail externe



Nécessités :

Solaris/Linux         Netscape 6.1 ou supérieur
Windows                 Outlook 5.0 / Netscape 6.1  ou supérieur

Assurez-vous qu'OpenOffice.org utilise bien un programme de mail externe valide, vérifiez le dans Outils - Options - OpenOffice.org - Programme auxiliaire

Si aucun programme auxiliaire n'a été séléctionné par OpenOffice.org, sélectionnez en un ici
Netscape ou Outlook doivent être configurés pour envoyer des mails avant que ce test puisse être exécuté.
Description:
    1. A partir de la barre des menus, sélectionnés Fichier - Nouveau - Texte
    2. Un nouveau document texte doit être lancé
    3. Entrez un court message ex : "ceci est un court message"
    4. A partir de la barre des menus, sélectionnez Fichier - Envoyer - Envoyer par e-mail
    5. Une fenêtre comportant Nouveau Message (Outlook) ou Composer (Netscape) doit se lancer
    6. Envoyer le mail normalement au destinataire
    7. Assurez-vous que le mail est arrivé au destinataire.
Résultat attendu :

OpenOffice.org doit envoyer un message attaché à un e-mail
