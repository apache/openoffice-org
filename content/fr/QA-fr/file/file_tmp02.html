Cas Test : file_tmp02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer un nouveau modèle de document texte


Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Modèles et Documents
    2. La boite de dialogue de Modèles et  Documents doit être lancée
    3. Assurez-vous qu'il y a bien les 4 icônes présentes dans la partie gauche de la boite, affichant Nouveau document, Modèles, Mes Documents, Exemples
    4. Sélectionnez Nouveau Document dans la partie gauche de la boite de dialogue
    5. Sélectionnez Texte dans la partie centrale de la boite de dialogue,
    6. Sélectionnez Ouvrir dans la boite de dialogue Modèles et Documents
    7. Un nouveau document texte doit être lancé
    8. Fermez ce document texte
Résultat attendu :
Toutes les opérations doivent se dérouler avec succès
