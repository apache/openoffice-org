Cas Test : aut_fax02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer un Fax à partir de l'Autopilote


Description:
    1. A partir de la barre de menu, sélectionnez Fichiez - Autopilote - Fax
    2. La boite de dialogue de l'Autopilote de Fax doit être lancée
    3. Assurez vous que Modern est sélectionné dans boite de dialogue de l'Autopilote de Fax
    4. Sélectionnez Créer dans la boite de dialogue
    5. Assurez vous qu'OpenOffice.org crée la page de Fax
    6. Le document doit montrer
      . une entête portant la mention "Fax"
      . une entête indiquant "Nombre de pages incl. celle-ci"
      . le début de la lettre commence par "Monsieur ou Madame"
    7. Fermez le document Fax
        
      
Résultat attendu :
Toutes les opérations doivent se dérouler avec succès
