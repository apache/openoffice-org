Cas Test : file_lab02

Zone de Test : Solaris - Linux - Windows

But du test 

Vérifier qu'OpenOffice.org peut créer un document d'étiquettes

Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Etiquettes
    2. La boite de dialogue Etiquettes doit se lancer
    3. Assurez vous qu'elle comporte trois onglets intitulés Etiquettes, Format et Options
    4. Sous Options, assurez-vous que Page entière est sélectionné dans la boite de dialogue
    5. Sélectionnez Nouveau Document dans la boite de dialogue
    6. Assurez-vous qu'un nouveau document d'étiquettes est créé
    7. Fermez le document d'étiquettes
Résultat attendu
Toutes les opérations doivent se dérouler avec succès
