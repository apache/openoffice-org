Cas Test : exp_pdf

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer et sauvegarder un document .pdf


Description:
    1. Ouvrez un nouveau document texte, entrez un peu de texte, ex : "un court message de texte"
    2. Sélectionnez Fichier - Exporter comme PDF
    3. Dans la boite de dialogue d'export PDF, entrez un nom pour le nouveau fichier ex : premier
    4. En utilisant Ghostview ou Acroread, ouvrez le nouveau fichier ex : premier.pdf
    5. L'utilisateur doit voir son court message de texte dans le fichier ouvert
 

Résultat attendu:

Toutes les opérations doivent se dérouler avec succès
