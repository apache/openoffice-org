Cas Test : aut_let02

Zone de Test : Solaris - Linux - Windows

But du Test

Vérifier qu'OpenOffice.org peut créer une lettre à partir de l'Autopilote



Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Autopilote - Lettre
    2. La boite de dialogue de l'Autopilote de lettre doit se lancer
    3. Assurez-vous que "Lettre Commerciale" est sélectionné dans la boite de dialogue
    4. Sélectionnez Créer dans la boite de dialogue de l'Autopilote de lettre
    5. Assurez-vous qu'OpenOffice.org crée une lettre à partir de l'Autopilote
    6. Le document doit montrer
      . la date du jour,
      . le début de la lettre doit commencer par Monsieur ou Madame
    7. Fermez le document lettre
       
     
     
Résultat attendu :
Toutes les opérations doivent se dérouler avec succès
