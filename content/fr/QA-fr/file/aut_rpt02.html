Cas Test : aut_mem02

Zone de test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer un rapport à partir de l'Autopilote


Description:
    1. Ouvrez un nouveau document texte et à partir de la barre de menu, sélectionnez Fichier - Autopilote - Rapport
      Dans la boite de dialogue de choix de la base de données, choisissez Bibliographie dans la source de données
    2. Sélectionnez biblio dans la fenêtre de champ Table ou Requête
    3. Dans les champs existants sélectionnez  les suivants et placez-les dans la fenêtre champ en utilisant la touche ->
      Adresse, Annote, Author, Booktitle
    4.  Sélectionnez Suivant
    5. Dans la boite de dialogue de Rapport  - Grouper, sélectionnez Suivant
    6. Dans la boite de dialogue de Rapport -  Trier les options, sélectionnez Suivant
      Un rapport doit maintenant exister dans le document texte
    7. Dans la boite de dialogue de Rapport - Choisir l'apparence, sélectionnez Suivant
    8. Dans la boite de dialogue de Rapport  - Sauvegarder, entrez le nom que vous souhaitez donner à ce rapport et sélectionnez Créer.

     
Résultat attendu :
Toutes les opérations doivent se dérouler avec succès
