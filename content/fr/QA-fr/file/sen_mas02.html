Cas Test : sen_mas02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut envoyer vers un document maître


Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Texte
    2. Un nouveau document texte doit se lancer
    3. Entrez un court message ex "ceci est un court message"
    4. A partir de la barre des menus, sélectionnez Fichier - Envoyer - Créer un document Maître
    5. Une boite de dialogue demandant le nom et le chemin du document maître doit être lancée
    6. Entrez OOMaître dans le champ nom  de la boite de dialogue
    7. Sélectionnez Enregistrer dans la boite de dialogue
    8. Fermez le document

Résultat attendu

Toutes les opérations doivent se compléter avec succès

Fin du cas test
