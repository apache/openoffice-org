Cas Test : file_bus02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer des cartes de visites professionnelles

Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Cartes de visite
    2. Une boite de dialogue de carte de visite doit s'ouvrir
    3. Assurez-vous qu'il y a bien qu'il y a bien 6 onglets indiquant : Média, Cartes de visite, Privé, Bureau, Format et Options
    4. Sous Options, assurez-vous que 'page entière' est sélectionnée dans la boite de dialogue
    5. Sélectionnez Nouveau Docuement dans la boite de dialogue
    6. Assurez-vous qu'un document de cartes de visite est bien créé
    7. Fermez le document de cartes de visite

Expected Result:
Toutes les opérations doivent s'être déroulées avec succès
