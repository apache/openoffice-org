Cas Test : sav_Enco

Zone de test : Solaris - Linux - Windows

But du test :

Vérifier qu'OpenOffice.org peut créer et sauvegarder un document en texte encodé

Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Document Texte
    2. Un nouveau document texte doit être lancé
    3. Entrez un court message de texte ex : "court message de texte"
    4. A partir de la barre des menus, sélectionnez Fichier - Enregistrer sous...
    5. La boite de dialogue Enregistrer sous doit se lancer
    6. Entrez Encofichtest comme nom de fichier dans la boite de dialogue Enregistrer sous
    7. Sélectionnez Texte codé dans le type de fichier de la boite de dialogue
    8. Sélectionnez Enregistrer dans la boite de dialogue
         8.1 Une boite de dialogue d'Options de filtres ASCII  doit être lancée
         8.2 La valeur par défaut doit être Système et les retour de paragraphes sur CR & LF
         8.3 Sélectionnez OK dans la boite de dialogue de Filtre ASCII
    9. Fermez le document

    10. A partir de la barre des menus, sélectionnez Fichier - Ouvrir 
    11. La boite de dialogue d'ouverture de fichier doit se lancer
    12. Vérifier dans le détail de la liste des fichiers le type de fichier, pour Encofichtest il doit être
     
        Windows:  Document texte ou Fichier Texte
        Linux:-       Fichier Texte
        Solaris:-    Fichier Texte

    13. Sélectionnez Encofichtest et assurez vous qu'il s'ouvre correctement
    14. Fermez le document

 

Résultat attendu :

Toutes les opérations doivent se compléter avec succès


Fin du cas test
