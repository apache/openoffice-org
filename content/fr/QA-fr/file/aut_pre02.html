Cas Test : aut_pre02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut créer une présentation à partir de l'Autopilote


Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Autopilote - Présentation
    2. La boite de dialogue de l'Autopilote de présentation doit se lancer
    3. Assurez-vous qu'une présentation vide est sélectionnée dans la boite de dialogue de présentation
    4. Sélectionnez Créer dans la boite de dialogue
    5. Assurez-vous qu'OpenOffice.org crée une présentation
    6. Le document doit être vide
      . une boite de dialogue de modification de la diapo doit se lancer
    7. Fermez la boite de dialogue de modification en cliquant sur Annuler
    8. Fermez  le document de présentation

     
Résultat attendu :
Toutes les opérations doivent se compléter avec succès
