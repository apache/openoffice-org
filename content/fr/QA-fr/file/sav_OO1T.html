Cas Test: sav_OO1T

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier qu'OpenOffice.org peut sauver au format OpenOffice.org Modèle de documents



Description:
    1. A partir de la barre des menus, sélectionnez Fichier - Nouveau - Document texte
    2. Un nouveau document doit être lancé
    3. Entrez un court message ex : "court message de texte"
    4. A partir de la barre des menus, sélectionnez Fichier - Enregistrez sous
    5. La boite de dialogue de Sauvegarder sous doit être lancée
    6. Entres OO1Tfichtext dans le champ nom de fichier de la boite de dialogue
    7. Sélectionnez OpenOffice.org 1.0 modèle de texte dans le champ type de fichier de la boite de dialogue
    8. Sélectionnez Enregistrer dans la boite de dialogue
    9. Fermez le document

    10. A partir de la barre des menus, sélectionnez Fichier - Ouvrir
    11. La boite de dialogue d'Ouverture de fichier doit se lancer
    12. La boite de dialogue affiche la liste des fichiers, assurez vous que OO1Tfichtest  a bien le type Document OpenOffice.org 1.0 modèle
    13. Sélectionnez le document OO1Tfichtext et assurez-vous qu'il s'ouvre correctement
    14. Fermez le document

Résultat attendu :

Toutes les opérations doivent se dérouler correctement
