Cas Test : aut_con02

Zone de Test : Solaris - Linux - Windows

But du test

Vérifier que la fonction convertisseur de document fonctionne correctement


Description:
    1. A partir de la barre de menu, sélectionnez Fichier - Autopilote - Convertisseur de Documents
    2. La boite de dialogue du convertisseur de documents doit se lancer
    3. Assurez-vous que StarOffice - Documents Texte est séléctionné dans la boite de dialogue du convertisseur de documents
    4. Sélectionnez Suivant dans la boite de dialogue du convertisseur de document
    5. Déselectionnez Modèles de documents dans la boite de dialogue de l'Autopilote
    6. Sélectionnez Suivant dans la boite de dialogue du convertisseur de document
    7. Un message de la boite de dialogue du convertisseur de document vous indique que maintenant tous les documents  textes contenus dans le répertoire suivant vont être importés et exportés dans le répertoire par défaut
    8. Sélectionnez Convertir dans le convertisseur de document
    9. La boite de dialogue du convertisseur de document va indiquer combien de fichiers on été convertis dans le répertoire choisi.
    10. Sélectionnez Terminer dans la boite de dialogue du convertisseur de document
       

     
Résultat attendu :
Toutes les opérations doivent se compléter avec succès
