Indications pour Test d'Edition

Instructions:
  1. PLATFORME : Précisez le système d'exploitation sur lequel vous avez fait le test.


  2. OS_VER : Précisez la version de ce système d'exploitation sur lequel vous avez fait le test.


  3. BUILD_ID : est l'identifiant unique de la version d'OpenOffice.org.


  4. Entrez votre identité de Testeur.


  5. Le texte entré dans Autres annulera les options choisies dans le menu déroulant Options.


  6. Lisez les instructions pour chaque test et faites-le.

  7. Sélectionnez Passé, Echec ou Non testé pour chaque test.


  8. Pressez le bouton Enregistrer lorsque vous avez terminé les tests.
