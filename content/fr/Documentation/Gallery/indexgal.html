Gallery

Ces images sont à intégrer directement dans le fichier Gallery d'OpenOffice.org.

Installation des images

Pour installer ces images dans la Gallery, une fois le répertoire téléchargé et dézippé :

- Ouvrez celui-ci soit en cliquant sur l'icône dans la barre de Fonctions, soit en choisissant Gallery dans le menu Outils,

- Cliquez sur le thème de votre choix pour le sélectionner et faites un clic droit sur votre souris pour accéder au menu contextuel. Sélectionnez l'option Propriété. Dans la fenêtre qui s'ouvre, sous l'onglet Fichier, cliquez sur le bouton 'Recherche de fichiers' et naviguez jusqu'au répertoire dans lequel vous avez décompressé les images. Cliquez sur Sélectionner pour fermer la boite de dialogue et valider votre choix.

- Dans la boite de dialogue, sous Type de fichiers, vous avez maintenant le contenu du répertoire qui s'affiche, vous n'avez plus qu'à sélectionner l'image que vous voulez ajouter au thème choisi et à cliquer sur Ajouter.

- Répétez cette opération pour les thèmes de votre choix. Si vous voulez créer un nouveau thème, cliquez sur le bouton Nouveau Thème en haut de la liste des thèmes, la même fenêtre de dialogue que celle décrite plus haut s'ouvre, il ne vous reste qu'à nommer votre thème sous l'onglet Général.

Vous disposez également d'un how-to très détaillé dans la section How-to de la rubrique documentation
