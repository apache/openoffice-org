Documentation - Table des matières

Vous trouverez ici une partie de la documentation nécessaire à l'utilisation d'OpenOffice.org. Celle-ci est réalisée par les membres bénévoles de la communauté francophone. Elle est en constante évolution, n'hésitez donc pas à revenir souvent consulter ces pages.

Une autre partie de la documentation est également disponible sur le Wiki. Cliquez ici pour y accéder.

Une documentation vous manque ou vous avez à proposer quelque chose : rejoignez-nous afin d'enrichir cette section.

Writer
le traitement de textes, table des matières, publipostage, étiquettes, les styles...
Calc
le tableur, formules, formatage conditionnel, plage de données, tableau croisé...
Impress
le module de présentation pour vos diaporamas
Draw
dessiner, cartographie, PAO, flyer, affichette...
Math
l'éditeur de formule mathématique, équation, chimie...
Base
le gestionnaire de base de données, rapport, requête, SQL, formulaire...
Basic
le langage de programmation pour les macros afin d'automatiser vos documents...

Des travaux de création ou de traduction sont en cours, vous pouvez les rejoindre ou créer vos propres documents et les proposer. Nous avons souvent également besoin de traducteurs, si vous parlez ou écrivez plusieurs langues, venez participer à ce projet. Certaines documentations sont sous licence PDL). Merci à tous les membres de cette communauté francophone pour leur générosité, leur disponibilité et la qualité de leur travail.
