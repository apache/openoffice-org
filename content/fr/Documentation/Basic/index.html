Documentation - Table des matières

Vous trouverez ici une partie de la documentation nécessaire à l'utilisation d'OpenOffice.org. Celle-ci est réalisée par les membres bénévoles de la communauté francophone. Elle est en constante évolution, n'hésitez donc pas à revenir souvent consulter ces pages. 

Une autre partie de la documentation est également disponible sur le Wiki. Cliquez ici pour y accéder.

Une documentation vous manque ou vous avez à proposer quelque chose : rejoignez-nous afin d'enrichir cette section.

Ces tutoriels sont des explications pas à pas et illustrées pour programmer avec OpenOffice en Basic ou dans un langage tiers. Ils sont mis à disposition de manière gratuite dans différents formats.

Ce manuel vous donne des informations de base concernant les macros et son langage de programmation le Basic sous OpenOffice.org. Vous apprendrez ce qu'est une macro et comment les créer et les modifier.

Outil XRay

Nouvelle version 6.0 du 01/06/2012
Ce document, qui contient la macro XRay, vous expliquera comment l'installer et l'utiliser.  Xray est un outil qui permet aux programmeurs de macros de visualiser les propriétés, méthodes et interfaces disponibles sur une variable objet.

SXW

Dans la jungle de l'API

Un document qui vous apprendra à vous débrouiller avec l'API de OOo. Ecrit de façon assez personnelle, il est destiné aux débutants en Basic et contient, expliqué très simplement, tous les éléments pour se repérer.

Ces boîtes à outils aident les programmeurs à piloter OpenOffice.org depuis un langage externe, avec la technologie COM Automation. Chaque boîte à outils est un zip contenant :
 - un module avec des routines similaires à celles qu'offre OpenOffice.org Basic pour accèder à l'API
 - un module contenant toutes les constantes et énumérations de l'API
 - un module donnant plusieurs exemples de programmation
 - une routine permettant d'utiliser l'outil Basic Xray depuis le langage; l'outil pour Delphi contient aussi une version native de Xray.
 - un HowTo donnant toutes les explications utiles
