Documentation - Table des matières

Vous trouverez ici une partie de la documentation nécessaire à l'utilisation d'OpenOffice.org. Celle-ci est réalisée par les membres bénévoles de la communauté francophone. Elle est en constante évolution, n'hésitez donc pas à revenir souvent consulter ces pages. 

Une autre partie de la documentation est également disponible sur le Wiki. Cliquez ici pour y accéder.

Une documentation vous manque ou vous avez à proposer quelque chose : rejoignez-nous afin d'enrichir cette section.

Ces tutoriels sont des explications pas à pas et illustrées pour l'utilisation du module Base de OpenOffice. Ils sont mis à disposition de manière gratuite dans différents formats.

Créer des tables .csv dynamiques

Le but de ce document est de présenter une technique permettant d'utiliser de manière dynamique des fichiers de données au format .csv dans un document Base. Cela permet de :
- Utiliser dans Base des données issues d'une autre application qui génère des fichiers .csv.
- Partager des données saisies sous Base avec une autre application capable de traiter des données au format .csv.
