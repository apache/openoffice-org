Outils

Ces outils sous forme de scripts sont des aides dans votre utilisationquotidienne d'OpenOffice.org. N'hésitez pas à nous apporter un retour sur ceux-ci, cela nous permet de les améliorer, d'en réaliser d'autres en fonction de vos demandes. Vous pouvez également proposer les votres sur la liste dev@fr.openoffice.org, nous nous ferons un plaisir de les ajouter ici. Merci à tous ceux qui ont donné ces outils à notre communauté. La colonne 'V2' indique que le document a été mis à jour vers la version 2.
Titre Description Formats V2
Fitoo Outil de corrélation non linéaire. Décompressez le fichier .zip et lisez le fichier Lisez-moi pour savoir comment procéder à l'installation. ZIP
Facturier Outil permettant de créer le suivi des devis et factures. Décompressez le fichier zip, un fichier .exe vous guide dans l'installation. ZIP
Amortissement Tableau d'ammortissement linéaire des immobilisations. SXC
Amortissement Prêt Cette outil vous aidera à calculer l'amortissement d'un prêt. ODS V2
Vocabulary Traducteur mot à mot pour OpenOffice.org. Décompressez le fichier zip, un fichier sxw vous guidera dans l'installation. ZIP
Pinceau Cet outil vous permet de copier les styles à la manière du pinceau de Word. La feuille sxw contient toutes les indications pour l'installation de l'outil sous forme de macro. SXW
Add-ons Sous forme de how-to un exemple pour réaliser un add-on, programme permettant à vos utilisateurs d'installer de façon très simple des macros que vous réalisez. SXW
Multipages Cet add-on vous sera très utile pour créer des documents Calc, Draw ou Impress comportant un grand nombre de pages (ou feuilles). Il vous permet en effet d'ajouter, dupliquer, supprimer, renommer des dizaines de pages à la fois. Le fichier zip contient un fichier Writer pour l'installation et le mode d'emploi ainsi que l'add-on proprement dit, un autre fichier zip. ZIP
TailleFormules Cet add-on vous permet de redimensionner la taille des équations inclues dans un document. Le fichier Zip contient un lisez-moi expliquant l'installation et l'add-on, un autre fichier zip. ZIP
BatchConv BatchConv est un utilitaire de conversion Batch de tout fichier supporté en Import vers les formats supportés en export par OPenOffice.org. La version 1.1 est désormais internationalisée. La version 1.6 permet de selectionner plusieurs fichiers dans le même répertoire. Merci de me faire parvenir toute remarque ou suggestion. SXW
Faire-Part de naissance Un faire-part dont la macro vous facilitera l'impression et le pliage. Le modèle vierge est disponible dans la section modèle de notre site. SXD
Multisave Cet add-on vous permettra d'enregistrer simultanément un document au format .doc, .pdf, .sxw au choix. Pour l'installer, vous devez d'abord dézipper le fichier téléchargé, un document sxw vous exeplique alors la mise en place du second fichier zip. ZIP
SHOOo SHOOo, une classe PHP qui permet de créer une table HTML et un fichier au format Calc (sxc) à partir du résultat d'une requête SQL ou du contenu d'un tableau PHP, illustrant ainsi l'ouverture du format XML d'OOo et son traitement possible en dehors de la suite. ZIP
GenealOOogie GenealOOogie est un ensemble de macros sous forme d'add-on  pour lire et analyser un fichier ged et afficher sous forme d'arbre les descendants ou ascendants d'un individu. Le fichie .zip contient toutes les informations pour l'installation. ZIP
ImagerieOOo Cet outil permet de réduire le poid des images au format Jpeg, gif, png. En effet, dans un sxw, OOo réduit la taille mais pas le poid des images, ce qui fait qu'un document en contenant beaucoup, arrive à peser très lourd assez vite. Nécessite Java. ZIP
OOoWikipedia Un add-on  qui vous permettra de vous connecter à Wikipedia, célèbre encyclopédie libre en ligne. Vous pourrez également choisir la langue dans laquelle vous souhaitez vous connecter à Wikipedia. SXW
Milifred Une macro traceur de papier semi-log ou log-log, bref à vous le papier millimétré SXD
ChiffresEnLettres Vous permet de transformer des chiffres en lettres en utilisant seulement des formules de Calc. SXC V2
MacroCeL Vous permet de transformer des chiffres en lettres en installant un add-on (macro). Le zip contient la documentation d'installation et l'add-on pour OOo version 1.x et  2.x. ZIP V2
ODevisClicFacture Un outil très complet qui vous permettra de réaliser devis et facture. ZIP
CadOOo CadOOo est un add-on qui ajoute des fonctionnalités de dessin à OOo comme un accrochage aux objets plus étendu, une saisie des coordonnées rectangulaires ou polaires absolues ou relatives, une saisie des rayons, des diamtères ou des longueurs. Nouvelle version pour OOo 2.x du 24 juin 2007. ZIP V2
RibanSir Cet outil vous permet de contrôler les  RIB, IBAN, SIREN, SIRET, CB. SXW
Tableau Tableau d'ammortissement linéaire des immobilisations SXC
O_Clic_CP Ce zip contient deux outils, un qui est une table de code postaux permettant de faire une recherche sur toute la France, un second qui comporte un formulaire de saisie automatique des codes postaux. ZIP
AutOOofiltre Macro permettant de créer facilement un autofiltre pour chaque onglet d'un classeur Calc. Il fonctionne pour les  versions 1.1.x et 2.0. Cliquez sur le lien PDF pour télécharger la documentation et sur le lien ZIP pour le package d'installation. PDF ZIP V2
Gestion COOotes Cet outil très complet destiné aux enseignants permet de gérer les notes des élèves. ZIP V2
Sylk4OOo Sylk4OOo est un convertisseur (executable Windows) de fichiers SYLK qui les rend lisibles par Calc v.1 et v.2 en insérant le préfixe attendu pour qu'il ne s'ouvrent pas avec Writer. ZIP V2
OOo2PDF Ligne de commande permettant d'exporter des fichiers au format .pdf ZIP V2
ImpressRunner Cette extension vous offre un support pour les fichiers .pps générés par PowerPoint et permet le lancement automatique des présentations Impress. OXT V2
BC_BPEM Ce fichier Calc est destiné à établir des feuilles de paie pour employés de maison. Nouveau document du 29 décembre 2007. ODS V2

Certains des membres d'OpenOffice.org mettent à votre dispositions sur leur site des outils complémentaires pour OOo :
  • http://www.jdisoftware.co.uk/pages/epdf-home.php une macro qui génère un document PDF à partir d'un document OOo avec des signets PDF, les informations méta du document (auteur, mots clés etc) et des hyperliens.
  • http://search.cpan.org/dist/OpenOffice-OODoc The Perl Open OpenDocument Connector est un composant Perl destiné à faciliter le développement rapide de scripts pouvant consulter et modifier directement tout élément d'un fichier OOo sans passer par les APIs. Merci à Jean Marie Gouarné pour cet outil (LGPL) donné à la communauté. Une page personnel est maintenue par Jean Marie, qui contient tous les liens utiles et d'où l'on peut télécharger un guide de démarrage en français  http://jean.marie.gouarne.online.fr/oodoc.html
  • http://www.opengroupware.org un serveur de goupware opensource que vous pouvez utiliser avec OOo (uniquement pour Linux pour le moment)
  • http://www.via.ecp.fr/~remi/win/ooovirg/ooovirg.php3 Ce logiciel développé par Rémy Peyronnet vous permet de transformer le point en virgule sous Calc.
  • http://ooo2txt.fr.st/ Un filtre développé par Frédéric Labbé, sous forme de script Perl, il permet d'extraire le contenu d'un document OOo au format texte.
  • http://oootools.free.fr Frédéric Labbé place ici tous les outils qu'il développe pour la migration d'OpenOffice.org (script d'install silencieuse, script de volumétrie des fichiers à convertir, dépistage des macros...)
  • http://www.dmaths.com Dmaths est un logiciel écrit par Didier Dorange-Pattoret, complémentaire à l'éditeur d'équation d'OpenOffice.org. Des versions sont disponibles pour OOo1.1/SO7.0 (et SO5.2) sous Linux comme sous Windows. Il facilite l'écriture des formules (en écrivant AM=1/3(AB+AC) on obtient par clics l'écriture vectorielle correspondante). Il permet maintenant le tracé des courbes,  y compris paramétriques et également celui des diagrammes en boite
  • http://ooextras.sourceforge.net Un site regroupant des modèles, des macros, des images... développés par la communauté des utilisateurs d'OpenOffice.org (en anglais)
  • http://evolvedoo.sourceforge.net/ Un package pour Linux qui apporte des outils d'intégration d'OpenOffice.org sous Gnome et KDE (quick-start, installeur de dictionnaire...)
  • http://www.chez.com/ebellot/ooo2sdbk/ Un convertisseur OpenOffice.org vers DocBook créé par Eric Bellot
