Introduction sur la version Mac de OOo :

OpenOffice.org fonctionne depuis janvier 2003 sous Mac OS X. Cette version est la troisième version officielle pour Darwin PPC et MacOS X avec X11. Cette application est suffisamment stable pour être utilisée dans un environnement de production.

Cette version est la version DarwinPPC qui nécessite X11 pour fonctionner (X11 est un gestionnaire de fenêtre qui va s'ajouter à celui (Aqua) de Apple. Il permet d'y faire tourner des applications Linux). Ainsi, elle n'a pas actuellement le look et feel standard de MacOS. Si l'utilisation d'un produit basé sur X11 ne vous dérange pas, cette version est maintenant suffisamment stable pour une utilisation quotidienne et comme première suite bureautique.


 

Configuration système requise

OpenOffice.org 1.1.2 MacOS X (X11) a besoin pour fonctionner de :

  • MacOS X 10.3 obligatoire : cette version de OO ne fonctionnera pas sous 10.2 ou inférieur

  • 256 Mo de mémoire pour avoir des performances correctes, 512 Mo recommandé

  • 300 Mo d'espace sur le disque dur pour OpenOffice.org

  • Xfree86/XDarwin ou Apple X11, dlcompat, ESP-Ghostscript 7.05, fondu et libfreetype 6.2+. L'installeur va détecter s'il vous manque l'un de ces composants et les installera pour vous

  • G4/400 ou supérieur (recommandé)

Particularités concernant Mac : vous ne pouvez pas utiliser directement OOo sur votre Macintosh si X11 n'est pas installé.

OpenOffice.org 1.1.2 peut être installé parallèlement à une ancienne version de OpenOffice.org. Si vous décidez plus tard de désinstaller l'ancienne version de OpenOffice.org, vous devrez lancer le programme Setup de la nouvelle version et exécuter une réparation afin de garantir son inscription correcte dans le système.


Installation X11 d'Apple

X11 est disponible sur le CD4 de la distribution Panther (10.3) : à l'installation de Panther apparaît une option demandant si on souhaite l'installer, sinon, on met le CD4 et on installe le package appelé : /Packages/X11SDK.pkg
Enfin, on peut le télécharger à (http://www.apple.com/macosx/x11/). Ensuite vous avez juste besoin de cliquer sur le fichier d'installation.


Installation de OpenOffice 1.1.2

1- télécharger le fichier OOo1.1.2_MacOSX_Panther_fr_install.pkg.zip (76 Mo) à http://macosxrc.services.openoffice.org/pub/OpenOffice.org/ooomisc/MacOSXrc/ puis décompacter l'archive par un double click.

2- cliquer sur le fichier se terminant par .pkg et suivre les instructions à l'écran (le mot de passe administrateur est demandé). Ne pas changer le répertoire d'installation (/Applications/), et cliquer sur continuer (sinon, le risque de ne pas fonctionner correctement est très grand)

3- à l'aide du Finder, aller dans le répertoire dans lequel OO est installé : /Applications/OpenOffice.org1.1.2 et double cliquer sur l'icône Démarrer_OOorg : le programme démarre (le premier lancement prend un peu de temps pour terminer l'installation)

5- OO est maintenant opérationnel. On peut ensuite installer l'icône de lancement dans le Dock pour un accès plus rapide.

Remarque 1 : si X11 n'est pas déjà lancé au moment de démarrer OO, il va d'abord lancer X11, puis OO s'ouvre

Remarque 2 : que faire si OO affiche un message d'erreur ou ne s'ouvre pas ?

- il faut le lancer à travers un terminal afin de voir le message d'erreur. Pour cela, lancer un Terminal (/Utilities/Terminal)

- taper cd /Applications/OpenOffice.org1.1.2 suivi de Enter

(cette commande permet d'aller "dans" le répertoire où est installé OO)

- taper ./setup suivi de Enter (attention le point devant /setup est important)

- un message d'erreur apparaît alors dans le Terminal et ce message peut être copié/collé pour demander de l'aide sur le forum OO


Installation des dictionnaires

Les dictionnaires anglais et italiens sont automatiquement installés avec la version française de OO. Il faut donc installer le dictionnaire français et l'activer (cette démarche est valable pour installer n'importe quelle autre langue)

1- faire Fichier -> Autopilote -> Installer de nouveaux dictionnaires pour ouvrir un fichier appelé DicOOo.sxw (fourni maintenant par défaut et placé dans /Applications/OpenOffice.org1.1.2/share/dict/ooo).

2- suivre les instructions (il faut être connecté sur Internet)

3- pour pouvoir utiliser ces dictionnaire dans votre langue, vous devez l'activer. Choisissez Outils > Options > Paramètres linguistiques > Linguistique

4- Sélectionnez « OpenOffice MySpell SpellChecker » et cliquez sur le bouton Editer à côté

Dans la fenêtre de dialogue qui s'ouvre, choisissez votre langue dans la liste déroulante et vérifiez que la case en face de OpenOffice MySpell SpellChecker est cochée. Cela active le dictionnaire pour cette langue. Vous pouvez également activer la césure et le dictionnaire des synonymes s'ils sont présents dans cette langue

Cliquez sur OK pour quitter la boite de dialogue

5- Sélectionnez Outils > Options > Paramètres linguistiques > Langue

Dans la fenêtre de dialogue qui s'ouvre, choisissez la langue utilisée pour l'édition. Si vous avez installé les dictionnaires correctement vous devez voir une coche en face de votre langue dans la liste déroulante


Installation des fontes natives Mac

Par défaut, OO s'installe avec ses propres fontes. On peut y ajouter les fontes natives de MacOSX à l'aide d'un utilitaire appelé « fondu »

1- télécharger le fichier « Fondu-040324.pkg.tar.gz » (127 Ko) à http://macosxrc.services.openoffice.org/pub/OpenOffice.org/ooomisc/MacOSXrc/

2- décompresser l'archive : il apparaît un fichier .pkg

3- double cliquer sur ce fichier pour l'installer et suivre les instructions

4- lancer le programme « FontBook » dans /Applications/Font Book : il permet de voir toutes les fontes natives MacOSX et de faire son choix

5- il suffit ensuite de sélectionner la (ou les) fonte(s) désirés(s) depuis le répertoire /Library/Fonts/ et de glisser-déposer sur l'icône OOo_Admin qui est située dans /Applications/OpenOffice.org1.1.2/ : le transfert est immédiat

6- il faut ensuite relancer OO pour que les nouvelles fontes soient disponibles dans le menu
