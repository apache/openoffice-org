Le vote Issuezilla

Les votes vous permettent de prendre part de façon active dans l'ordonnacement et les priorités de l'activité de votre projet. Selon la configuration de votre projet, chaque utilisateur peut avoir un certain nombre de votes. Même si le vote est autorisé dans votre domaine, il peut être restreint en fonction du paramétrage du projet.

Les utilisateurs peuvent répartir leurs votes comme bon leur semble, mais ne peuvent pas dépasser le quota qui leur a été affecté au départ. Le nombre de votes autorisés pour chacun dépend de l'administrateur du domaine (10 votes autorisés par défaut). Avec vos votes, vous pouvez favoriser les bogues à corriger et améliorations à réaliser qui vous sembles les plus importantes.

Utilisation de vos votes

Le nombre de votes que vous utilisez ne dépend que de vos propres choix. Vous pouvez voter plusieurs fois pour la même bogue ou amélioration. Vous pouvez utiliser plusieurs votes pour la même référence ou les répartir sur plusieurs. Vous êtes entièrement libre de vos choix mais vous êtes limité par le nombre total de votes qui vous sont affectés (en général deux).

Combien j'ai de votes ?

Vous pouvez visualiser vos votes à tous moment en allant sur Issue Tracking/Recherche de bogues (NDT: par le menu "My Issues"/"Mes bogues" ou "Find an Issue"/"Rechercher un bogues") et en cliquant sur My Votes/Mes votes la barre de navigation des bogues.

Comment voter pour un bogue ?

Pour savoir si vous pouvez voter pour un bogue ou connaître le nombre de votes possible, ouvrez le formulaire de bogue qui vous intérresse. Si le projet vous autorise à voter, vous trouverez une ligne Votes/Votes juste au dessus du champ Additional Comments/Commentaires supplémentaires. Sur cette ligne vous trouverez un nombre indiquant le nombre de votes que vous avez utilisé pour cette référence ainsi qu'un lien pour voter pour celle-ci.. Cliquez sur Vote for this issue/Voter pour ce bogue pour voir le nombre de bogue dont vous disposez, le nombre de votes utilisés ainsi que le nombre de votes pour cette référence. Ensuite, pour chaque modification de cette fiche de bogue, vous recevrez un courriel vous informant des changements qui ont eu lieu.

Si vous avez l'autorisation de voter pour ce projet, vous pouvez cliquer sur Vote for this issue/Voter pour ce bogue sous le champ Additional Comments/Commentaires additionnel et répartir votre nombre de votes dans chaque champ votes de la page. Après avoir remplit tout cela, cliquer sur le bouton Submit/Envoyer.

Comment voir tous les votes effectués ?

Vous pouvez lancer une requète pour tous les bogues qui ont au moins 1 vote, en entrant "1" dans le champ At least ____ votes/Au moins ____ votes en cliquant sur le bouton Submit query/Envoyer requête. Vous verrez allors un rapport contenant toutes les formulaires de votre projet qui ont au moins un vote. Grace à ce rapport, vous pouvez voir l'ensemble des votes pour chaque bogue et, en cliquant sur le lien d'une fiche vous pouvez ajuster vos votes comme vous le préferez.
