Pré-requis systeme pour Apache OpenOffice 4


Systeme Microsoft Windows

  • Windows XP, Windows 2003, Windows Vista, Windows 7, Windows 8, Windows 10
  • 256 Mo RAM (512 Mo RAM recommandé)
  • Au minimum 650 Mo disponible d'espace disque pour une installation par défaut (incluant le JRE).
    Apres installation et effacement du répertoire temporaire, OpenOffice utilisera approximativement 440 Mo d'espace disque.
  • Résolution 1024 x 768 ou plus en mode 256 couleurs

Systeme Apple MacOS X

  • Mac OS X 10.7 (Lion) ou plus
  • Intel Processor
  • 512 Mo RAM
  • 400 Mo d'espace disque disponible
  • Résolution 1024 x 768 en mode 16,7 millions de couleurs

Distribution Linux (mode 32 ou 64 bits)

  • Noyau Linux version 2.6 ou plus, glibc2 version 2.5 ou plus
  • 256 Mo RAM (512 Mo RAM recommandé)
  • 400 Mo d'espace disque disponible
  • X-Server en résolution 1024 x 768 et en mode 256 couleurs (16,7 millions de couleurs recommandé)


Accessibilité

Toutes plateformes

  • JRE (Environnemnet d'Exécution Java) 1.5.x ou supérieur : voir aussi Java et OpenOffice.org
  • Pont d'acces Java (voir Java Access Bridge)

Linux

  • Installation de GNOME 2.6
