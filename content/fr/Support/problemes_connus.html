Cette page recense de tels défauts connus dans OpenOffice.org. Elle est mise à jour au fur et à mesure de leur découverte.

#108978

Les tableaux Calc insérés dans un document texte disparaissent quand on enregistre le document en format Word97/2000/XP (.doc).

Perte de données si une version OpenDocument du document texte n'a pas été enregistrée auparavant.

Toujours utiliser le format natif OpenDocument pour la version de référence de vos documents. Exporter en doc par Fichier > Enregistrer sous en cas de besoin mais toujours conserver la version de référence en ODF comme version de travail.

Corrigé dans la version 3.2.1
