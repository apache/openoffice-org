Contributor Sponsoring

Can't afford to come?

Unfortunately not all key contributors of the OpenOffice.org project can afford to travel to Hamburg. Therefore, we want to help these community members to find a sponsor for their trip.

If you would like to attend the event either as a speaker or as an attendee, but can't afford to go to Hamburg, please send an email with the subject "Contributor Sponsoring" to
contribsponsor@openoffice.org

The email with your application for sponsorship should include the following information:

  • Full Name
  • Email Address
  • URL of Home Page (if it should be listed on this web page)
  • Company/Affiliation (if any)
  • Postal Address
  • Contact Telephone Number
  • Estimated Travel Costs in EUR
  • Description of OpenOffice.org Contributions
    (no more than 200 words)
  • Are you willing to help at conference?
    (a company might pay for the trip if you would be willing to help at the conference)

All applications will be verified by key community members (e.g. project leads) and then get published on this web page.

Want to sponsor the trip for a contributor?

If you would like to sponsor the trip for a contributor who can't afford to go to Hamburg, please contact one of the listed contributors and send an email with the subject "Contributor Sponsoring" to
contribsponsor@openoffice.org

The email with your sponsoring offer should include the following information:

  • Company Name
  • Contact Name
  • Email Address
  • URL of Home Page (if it should be listed on this web page)
  • Name of Person you will sponsor

All contributor sponsors will be listed by name in the conference program and on this web page. If the supported person is a speaker your company name will also appear on the conference agenda.

Team OpenOffice.org would like to thank for the generous help which allowed several contributors from all continents of world to visit OOoCon 2003.
