constants group CharType
Description
Constants to specify the character type that starts a character block. Character type is Unicode type defined in UnicodeType

Used with XBreakIterator::beginOfCharBlock(), XBreakIterator::endOfCharBlock(), XBreakIterator::nextCharBlock(), XBreakIterator::previousCharBlock()
