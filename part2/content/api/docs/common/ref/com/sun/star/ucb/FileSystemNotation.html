constants group FileSystemNotation
Description
The notational conventions used to denote file system paths on different file systems or operating systems.
Constants
UNKNOWN_NOTATION No information regarding any conventions is available.  
UNIX_NOTATION The conventions of Unix like file systems (e.g., /dir1/dir2/file).  
DOS_NOTATION The conventions of DOS like file systems (e.g., a:\dir1\dir2\file or UNC notation like \\host\dir1\dir2\file).  
MAC_NOTATION The conventions of Mac like file systems (e.g., volume:dir1:dir2:file).
