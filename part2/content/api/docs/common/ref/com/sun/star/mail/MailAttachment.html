:: com :: sun :: star :: mail ::

unpublished struct MailAttachment
Usage Restrictions
not published
Description
A MailAttachment specifies a mail message attachment.
Since
OOo 2.0
See also
XMailMessage
Elements' Summary
Data The actual data which should be attached to a mail message. It is expected that the transferable delivers the data as sequence of bytes. Although a transferable may support multiple data flavors only the first data flavor supplied will be used to retrive the data and it is expected that the type of the data is a sequence of bytes.  
ReadableName The name of the attachment as seen by the recipient of the mail message. ReadableName must not be empty.
