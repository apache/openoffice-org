constants group FootnoteNumbering
Description
These constants are used to specify the footnote numbering.
Constants
PER_PAGE The counter of the automatic footnote numbering restarts each page.  
PER_CHAPTER The counter of the automatic footnote numbering restarts each chapter.  
PER_DOCUMENT The counter of the automatic footnote numbering does not restart.
