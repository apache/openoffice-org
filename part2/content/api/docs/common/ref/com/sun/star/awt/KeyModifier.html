constants group KeyModifier
Description
These values are used to specify which modifier keys are pressed.
Constants
SHIFT refers to both shift keys.  
MOD1 refers in the most cases to the 'Ctrl' key (Cmd on Mac OS X).  
MOD2 refers in the most cases to the 'Alt' key.  
MOD3 refers in the most cases to the 'Ctrl' key (Mac OS X)
