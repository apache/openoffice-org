enum PaperFormat
Description
specifies the format (size) of the paper on a text document.
Values
A3 specifies the paper format as A3.  
A4 specifies the paper format as A4.  
A5 specifies the paper format as A5.  
B4 specifies the paper format as B4.  
B5 specifies the paper format as B5.  
LETTER specifies the paper format as Letter.  
LEGAL specifies the paper format as Legal.  
TABLOID specifies the paper format as Tabloid.  
USER The real paper size is user defined in 100th mm.
