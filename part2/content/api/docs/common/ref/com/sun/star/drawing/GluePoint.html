:: com :: sun :: star :: drawing ::

deprecated struct GluePoint
Usage Restrictions
deprecated
Description
A GluePoint could be attached to a shape or to a page.

If a GluePoint is attached to a shape, it is moved when the shape moves. The ends of connectors can be attached to GluePoint.

See also
GluePoint2
Elements' Summary
Position This is the position of this GluePoint.  
EscapeDirection This is the direction in which the connector line leaves the GluePoint.  
PositionAbsolute If this is true, then the position of this GluePoint is absolute on a page and is not relative to a shape.  
Alignment The alignment of a GluePoint defines how the position of the point is affected by resizing the parent Shape.
