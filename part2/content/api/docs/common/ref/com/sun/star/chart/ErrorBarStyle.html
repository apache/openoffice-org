Constants Constants' Details

:: com :: sun :: star :: chart ::

constants group ErrorBarStyle
Description
specifies the style of error indicators.
Constants
NONE error indicators are not displayed.  
VARIANCE displays error indicators for the variance of the data.  
STANDARD_DEVIATION displays error indicators for the standard deviation (square root of variance) of the data.  
ABSOLUTE the error indicators for all data points have the same absolute value as length for either direction.  
RELATIVE The length of the error indicators is calculated for each data point by taking the percentage given as ChartStatistics::PercentageError of its value.  
ERROR_MARGIN The length of the error indicators for all data points is calculated by taking the percentage given as ChartStatistics::ErrorMargin of the largest data point value.  
STANDARD_ERROR displays error indicators for the standard error, also known as the standard deviation of the mean (SDOM).  
FROM_DATA Uses values given by cell ranges of the container document.
