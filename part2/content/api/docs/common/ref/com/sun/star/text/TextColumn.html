:: com :: sun :: star :: text ::

struct TextColumn
Description
defines a single text column.
Elements' Summary
Width contains the relative width of the column, including both margins. Width isn't a metric value, its a relative value to the sum of the width of all columns.  
LeftMargin contains the left margin of the column.  
RightMargin contains the right margin of the column.
