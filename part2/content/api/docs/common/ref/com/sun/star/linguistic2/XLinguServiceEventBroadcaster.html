interface XLinguServiceEventBroadcaster
Description
is used to register a listener for LinguServiceEvents.

This interface may be used by spellchecker or hyphenator implementations to allow clients to be registered and informed about LinguServiceEvents.

Note: The LinguServiceManager forwards the LinguServiceEvents it receives (from spellcheckers or hyphenators) to its own listeners. Thus, there should be no need to register as a listener for a specific implementation./P>
