:: com :: sun :: star :: sdbc ::

constants group ResultSetType
Description
describes the different scroll capabilities of a result set.
Constants
FORWARD_ONLY is the type for a ::com::sun::star::sdb::ResultSet object whose cursor may move only forward.  
SCROLL_INSENSITIVE is the type for a ::com::sun::star::sdb::ResultSet object that is scrollable but generally not sensitive to changes made by others.  
SCROLL_SENSITIVE is the type for a ::com::sun::star::sdb::ResultSet object that is scrollable and generally sensitive to changes made by others.
