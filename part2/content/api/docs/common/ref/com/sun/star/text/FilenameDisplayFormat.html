constants group FilenameDisplayFormat
Description
These constants are used to specify which parts of an URL are displayed in a field.
Constants
FULL The content of the URL is completely displayed.  
PATH Only the path of the file is displayed.  
NAME Only the name of the file without the file extension is displayed.  
NAME_AND_EXT The file name including the file extension is displayed.
